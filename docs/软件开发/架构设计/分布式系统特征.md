# 分布式系统特征

> 来源：《分布式系统概念与设计》| 标签：架构 / 分布式

---

## 三大特征

1. **组件的并发** — 资源竞争与协作
2. **缺乏全局时钟** — 网络延迟导致时序不确定
3. **组件故障的独立性** — 单点故障隔离

---

## 8个挑战

| 挑战 | 说明 |
|------|------|
| **异构性** | 不同硬件、操作系统、网络 |
| **开放性** | 开放系统可扩展、互操作 |
| **安全性** | 防止未授权访问 |
| **可伸缩性** | 用户增长时系统可扩展 |
| **故障处理** | 故障检测、恢复 |
| **并发性** | 多个用户同时访问 |
| **透明性** | 用户感知为单一系统 |
| **服务质量** | 性能、可靠性保证 |
