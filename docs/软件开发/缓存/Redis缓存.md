# Redis缓存

## 简介

Redis是一个开源的、基于内存的数据结构存储系统，常用作数据库、缓存和消息队列。

## 快速启动

```bash
# Windows启动脚本
init_redis.bat
```

## 主要特性

| 特性 | 说明 |
|------|------|
| **高性能** | 基于内存，读写速度极快 |
| **丰富数据类型** | String、Hash、List、Set、ZSet |
| **持久化** | 支持RDB和AOF两种持久化方式 |
| **高可用** | 支持主从复制、哨兵、集群 |

## 常见应用场景

- 缓存热点数据
- 会话存储
- 实时排行榜
- 消息队列
- 分布式锁
