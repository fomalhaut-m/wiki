## 学习路径

```
Java基础 → 集合框架 → IO编程 → 多线程 → 并发编程
```

### 集合框架学习

```
Collection
├── List（ArrayList、LinkedList、Vector）
├── Set（HashSet、TreeSet、LinkedHashSet）
└── Queue（LinkedList、PriorityQueue、Deque）

Map
├── HashMap、LinkedHashMap
├── TreeMap
└── Hashtable
```

### 多线程学习

```
Thread创建 → 线程状态 → 线程同步 → 线程通信 → 并发工具
```
