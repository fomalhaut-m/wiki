# MySQL 逻辑架构

> 来源：MySQL高级笔记 | 标签：数据库 / MySQL

---

## 架构图

```
┌─────────────────────────────────────────┐
│              连接层                     │
│  客户端 → 连接处理 → 认证 → 线程池     │
└──────────────┬──────────────────────────┘
               │
┌──────────────▼──────────────────────────┐
│              服务层                     │
│  SQL接口 → 解析器 → 优化器 → 缓存       │
└──────────────┬──────────────────────────┘
               │
┌──────────────▼──────────────────────────┐
│              引擎层                     │
│  插件式存储引擎（InnoDB/MyISAM）        │
└────────────────────────────────────────┘
```

---

## 三层结构

### 1. 连接层

- 本地socket通信或TCP/IP通道
- 线程池管理连接
- 认证安全验证
- 权限验证

### 2. 服务层

- **SQL接口**：接收SQL语句
- **查询缓存**：8.0移除
- **解析器**：SQL解析成解析树
- **优化器**：确定执行计划（选择索引等）
- **内置函数**：跨引擎功能（存储过程、函数）

### 3. 引擎层

- 插件式存储引擎架构
- 负责数据存储和提取
- 服务器通过API与引擎通讯
- 可根据需求选择引擎

---

## 存储引擎对比

| 对比项 | MyISAM | InnoDB |
|--------|--------|--------|
| 主外键 | ❌ 不支持 | ✅ 支持 |
| 事务 | ❌ 不支持 | ✅ 支持 |
| 行锁 | ❌ 表锁 | ✅ 行锁 |
| 缓存 | 只缓存索引 | 缓存索引+真实数据 |
| 表空间 | 小 | 大 |
| 关注点 | 性能 | 事务 |
| 并发 | 低（表锁） | 高（行锁） |

## 查看引擎

```sql
SHOW ENGINES;  -- 查看所有存储引擎
SHOW VARIABLES LIKE '%storage_engine%';  -- 查看默认引擎
```

## 引擎选择建议

| 场景 | 推荐引擎 |
|------|----------|
| **事务** | InnoDB |
| **只读/低并发** | MyISAM |
| **高并发** | InnoDB |
| **全文索引** | MyISAM |
| **数据仓库** | MyISAM |
