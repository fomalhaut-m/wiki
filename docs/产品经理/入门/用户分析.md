# 用户分析

---

## 用户分类

### 按使用阶段分类

| 类型 | 说明 |
|------|------|
| **目标用户** | 产品最终用户 |
| **潜在用户** | 对产品有兴趣和需求，但尚未使用 |
| **种子用户** | 产品初期，通过尝试和反馈帮助产品完善 |
| **核心用户** | 产品发布后，长期使用并提供反馈 |

### 按人口统计学分类

- 年龄
- 性别
- 职业
- 收入
- 教育程度
- 地域

### 按熟悉程度分类

| 类型 | 说明 |
|------|------|
| **新用户** | 初次接触产品 |
| **老用户** | 长期使用产品 |
| **专家用户** | 深度使用，了解产品细节 |

---

## 抓住核心用户

### 被动吸引（靠产品）

| 方法 | 说明 |
|------|------|
| **人性** | 满足用户贪、懒、装等心理需求 |
| **产品价值** | 解决核心痛点，提供实际帮助 |
| **用户体验** | 操作便捷、界面美观、反馈及时 |

### 主动运营（靠运营）

| 方法 | 说明 |
|------|------|
| **优惠补贴** | 物质激励，拉新促活 |
| **公利活动** | 荣誉、排名、等级等社交激励 |
| **私利特权** | 会员、专属服务、VIP权益 |
| **裂变机制** | 邀请好友、分享得奖励 |

---

## 核心用户类型

| 类型 | 特征 | 案例 |
|------|------|------|
| **创造型** | 贡献内容/功能/ideas，参与产品迭代 | 社区内容创作者、B站UP主 |
| **消费型** | 高频使用，贡献活跃度和留存 | 重度玩家、付费会员 |
| **传播型** | 主动分享，带来新用户 | 分销员、KOL推荐 |
| **享受型** | 享受产品功能，提高客户粘性 | VIP会员、高端服务用户 |

---

## 用户画像构建

### 构建步骤

1. **数据收集** - 用户调研、数据分析
2. **标签提取** - 提取用户特征标签
3. **画像建模** - 构建典型用户画像
4. **应用验证** - 在产品设计中应用验证

### 应用场景

- 供需关系分析
- 功能优先级判断
- 运营策略制定
- 个性化推荐
