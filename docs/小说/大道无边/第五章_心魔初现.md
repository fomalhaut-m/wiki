# 第五章：心魔初现

三天。

墨渊给了阿邪三天时间。

这三天里，阿邪几乎没有合眼。他把自己关在小屋里，反复思考着对策。逃跑？墨渊的势力遍布整个深渊，他逃不掉。硬拼？他现在的修为，根本不是墨渊的对手。正面对抗只有一个字：死。

可是把孩子们交出去？

他做不到。

他宁愿死，也做不到。

第三天晚上，阿邪的伤势已经稳定了一些。脸上的伤口在深渊之力的修复下开始愈合，不再流血，只是还留着淡淡的红痕。他坐在窗边，看着外面永夜的天空，思绪飘得很远。

**「哥哥。」**

念念的声音从身后传来。

阿邪转过头，看见念念和安安站在他身后，两个小家伙不知道什么时候醒了，正眨巴着眼睛看他。

**「怎么还不睡？」** 阿邪问。

**「在想……以后怎么办。」**

**「哥哥，」** 安安拽了拽他的袖子，**「你不要把我们交给那些坏人，好不好？」**

阿邪的心猛地揪紧了。

**「不会的，」** 他把安安抱起来，放在腿上，**「哥哥答应过你们，就一定会做到。」**

**「可是那些坏人很厉害……」** 念念的眼眶又红了，**「我不想哥哥受伤……」**

**「念念，」** 阿邪认真地看着她，**「如果……哥哥是说如果，如果有一天，哥哥真的保护不了你们了……你们会怪哥哥吗？」**

念念愣了一下。

然后她摇了摇头，小小的手握住阿邪的手指。

**「不会。」** 她说，声音很轻，却很坚定，**「就算哥哥保护不了我们，我也不会怪哥哥。因为……哥哥已经尽力了。」**

**「而且……哥哥不是坏人。哥哥是安安和我见过的，最好的人。」**

阿邪的喉咙一下子堵住了。

他低下头，把念念和安安一起搂进怀里，紧紧地抱着。

**「谢谢你们。」** 他的声音有些哑，**「谢谢你们……愿意相信我。」**

那一刻，他做出了一个决定。

他不能再这样被动地等下去了。

墨渊要的是那两个孩子的灵根。如果能让孩子的灵根消失——或者隐藏得更深——墨渊也许就会失去兴趣。

可怎么做到？

阿邪陷入了沉思。

他的修为不高，炼气期，在深渊里属于最底层的存在。可他有一种特殊的能力——他身上沾染的深渊之力，可以在一定程度上遮蔽气息。

如果把深渊之力覆盖在孩子们身上，遮蔽他们的灵根气息……

但这有风险。深渊之力是魔气，凡人沾染了会生病，修士沾染了会影响修行。两个孩子还小，万一承受不住……

阿邪紧紧攥着拳头。

**「值得一试。」** 他对自己说。

就在这时——

门外再次传来沉重的脚步声。

这一次，比三天前更重，更压迫。

**「阿邪。」**

墨渊的声音穿透门板，带着压抑的怒火。

**「三天到了。」**

阿邪的心沉了下去。

他知道，最后的时刻来了。
