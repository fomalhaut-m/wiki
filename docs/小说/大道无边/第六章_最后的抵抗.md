# 第六章：最后的抵抗

门被撞开的瞬间，黑雾像潮水一样涌入。

阿邪站在小屋中央，身后就是衣柜——里面藏着念念和安安。他知道今天自己可能活不过去，但他不后悔。

**「主人。」** 他的声音出奇地平静。

墨渊踏入小屋，黑雾在他周身翻涌，冰蓝色的竖瞳死死盯着阿邪。

**「本座给过你机会了。」**

**「我知道。」** 阿邪说。

**「那两个孩子，就在衣柜里。」** 墨渊的语气是陈述，不是疑问。

阿邪没有否认。

他抬起头，直视墨渊的眼睛。

**「主人，你要杀我，我无话可说。但在我死之前，我想问你一个问题。」**

**「你有什么资格和本座谈条件？」** 墨渊冷笑。

**「我不是在谈条件，」** 阿邪的声音很平静，**「我只是想知道……当年你救我，真的只是因为我身上有利用价值吗？」**

墨渊愣了一下。

他显然没想到阿邪会在这种时候问出这种问题。

**「你什么意思？」**

**「三年了，」** 阿邪的嘴角扯出一丝苦笑，**「三年来，你教会了我很多东西。你从不掩饰你的残忍和贪婪，可有时候……我又觉得，你不仅仅是在利用我。」**

**「比如上一次你划伤我的脸，」** 他摸了摸自己的脸颊，**「以你的能力，大可以直接杀了我，可你没有。你只是……在惩罚我。」**

**「你为什么要惩罚我？」**

墨渊沉默了很久。

然后墨渊笑了。

那笑声在黑暗里回荡，像是深渊底部传来的回响，让人浑身发冷。

**「因为你的愧疚，」** 墨渊说，**「是本座吃过的最美味的东西。」**

**「你活着，你的愧疚就在。你越挣扎，你的愧疚就越浓郁。本座为什么要杀你？养着你，不是更好吗？」**

阿邪的瞳孔骤然收缩。

原来如此。

原来从头到尾，他都只是墨渊养的一头猪。一头用来产生愧疚这种"情绪"的家畜。

**「你……」**

**「所以，」** 墨渊打断他，**「别再做什么无谓的抵抗了。把孩子交出来，本座可以让你继续活下去。继续做本座的……宠物。」**

阿邪的身体在发抖。

不是因为恐惧，而是因为愤怒。

三年了。三年来他一直活在愧疚里，一直觉得自己是罪人，是废物，是只会带来灾难的存在。可到头来——

**「呵。」**

他笑了。

那是一个释然的笑。

**「原来我所有的愧疚，都是你喂给我的养料。」** 他说，**「原来你划伤我的脸，不是为了惩罚，是为了让我更加愧疚。」**

**「谢谢你，墨渊。」**

**「你让我想明白了一件事。」**

他抬起头，眼神里的恐惧已经消失了，取而代之的是一种前所未有的清明。

**「我的愧疚，是我自己的。不是你给的，不是你塑造的，是我三年前那一步没迈出去，一步一步走出来的。」**

**「它是真实的。它是我欠那对姐弟的债。」**

**「但从今天开始——」**

他的声音骤然变强，带着一种从未有过的坚定。

**「我不会再被你喂养了。我的愧疚，我自己来还。用我自己的方式。」**

**「不是逃跑，不是躲藏，不是继续做你的爪牙。」**

**「是面对。」**

墨渊的眼神变了。

**「有意思，」** 他低声说，**「你想怎么面对？」**

**「很简单，」** 阿邪说，**「你要那两个孩子的灵根，我可以给你。」**

**「但你要放过他们。」**

**「我来代替他们。」**
