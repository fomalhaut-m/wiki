# 第八章：启程

第二天一早，阿邪带着念念和安安离开了深渊。

他在深渊边缘的凡人村落里找到了一个孤寡老人，用身上最后一点银两把孩子托付给她。老人答应收留两个孩子，照顾他们的生活起居，直到阿邪回来。

临别的时候，念念抱着阿邪的腿不肯松手，眼泪把阿邪的衣袍都浸湿了。

**「哥哥，你一定要回来……」**

**「会的。」** 阿邪蹲下身，摸了摸她的头，**「念念，你是姐姐，要照顾好安安。」**

**「我会的……」** 念念抽噎着，**「哥哥，你也要照顾好自己……」**

**「嗯。」**

他摸了摸安安的头，把一个布包塞进安安手里。

**「安安，这是哥哥的护身符。遇到坏人就拿出来，他们会保护你。」**

那是他身上唯一一件与"正道"沾边的东西——一枚青云宗外门弟子的腰牌，上面的"青云"二字虽然已经有些褪色，但仍然带着一丝正气。这是他离开青云宗时唯一带走的东西，三年来从未离身。

可现在，他把他的过去，交给了这两个孩子。

**「哥哥……」** 安安的眼眶也红了。

**「好了，」** 阿邪站起身，深吸一口气，**「哥哥要走了。」**

他转身，头也不回地走进了深渊的黑暗里。

他怕一回头，就舍不得走了。

---

离开深渊之后，阿邪踏上了寻找那对姐弟魂魄的路。

三年前那场火之后，那对姐弟的魂魄去了哪里，他不知道。他只知道，青云宗有一门秘术，可以追溯亡魂的踪迹。

他需要回一趟青云宗。

虽然他已经不是青云宗的弟子，但宗门里还有他认识的人——他的师兄，周长生。当年他离开青云宗的时候，是周长生送他到山门口的。

只是，不知道周长生还愿不愿意见他。

青云宗在东海之滨，距离深渊有万里之遥。阿邪没有灵石买传送阵的票，只能靠双腿走。这一路，他风餐露宿，饿了吃干粮，渴了喝泉水，晚上找个避风的地方修炼一会儿，补充被深渊之力侵蚀的修为。

他的修为在离开深渊之后就开始退步了。

深渊之力是借来的，不是他自己的。离开深渊之后，这股力量就在一点点消散，他的修为也从炼气九层跌到了炼气七层。

可他不在乎。

---

一个月后。

东海之滨，青云山。

阿邪站在山脚下，仰头望着那座巍峨的山峰。

三年了。

三年前他离开的时候，也是这样一个清晨，雾气很重，师兄周长生站在山门口，拍着他的肩膀说：**「小邪，不管你走到哪里，青云宗永远是你的家。什么时候想回来了，师兄在山门口等你。」**

他没有回来。

他不敢回来。

因为他怕。他怕面对宗门里的流言蜚语，怕面对长老们的失望，怕面对自己内心的愧疚。

可现在，他不得不回来。

**「站住。」**

山门口的守卫拦住了他。

**「来者何人？」**

阿邪沉默了一下，从怀里掏出一块腰牌。

那是一块已经有些褪色的腰牌，上面刻着"青云"二字，背面刻着一个"邪"字。

**「外门弟子，阿邪。求见周长生师兄。」**

守卫接过腰牌，仔细端详了一番，眼神里闪过一丝诧异。

**「你就是阿邪？三年前那个……」**

**「是我。」**

守卫把腰牌还给他，意味深长地看了他一眼。

**「周师兄在内门峰，你自己去吧。」**

阿邪点点头，收好腰牌，一步一步走上山门。

内门峰。

阿邪站在周长生的洞府门前，犹豫了很久，才抬起手，敲了敲门。

**「谁啊？」**

熟悉的声音从里面传来。

阿邪的喉咙一下子堵住了。

**「师兄……是我。」**

门开了。

周长生站在门口，看见阿邪的一瞬间，整个人愣住了。

三年不见，周师兄的鬓角多了几丝白发，眼角也多了几道皱纹。但那双眼睛，还是和三年前一样温和、一样包容。

**「小邪？」** 周长生的声音有些颤抖，**「真的是你？」**

**「师兄……」** 阿邪低下头，声音有些涩，**「我回来了。」**

周长生沉默了一下。

然后他一把把阿邪拉进怀里，紧紧地抱住了他。

**「臭小子，」** 周长生的声音有些哽咽，**「三年了，你总算知道回来了。」**
