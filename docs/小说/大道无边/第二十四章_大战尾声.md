# 第二十四章：大战尾声

血煞魔君被制服了。

联军众人在短暂的震惊之后，爆发出震天的欢呼。

**「赢了！我们赢了！」**

**「血煞魔君被灭了！」**

**「阿邪！是阿邪！三年前他斩了墨渊，三年后他又斩了血煞魔君！」**

**「心魔道……这就是心魔道的力量吗？！」**

阿邪站在海面上，看着欢呼的人群，嘴角却没有露出笑容。

他走到血煞魔君身边，蹲下身。

血煞魔君躺在地上，眼神空洞，像是失去了灵魂的木偶。

**「为什么……」** 他的声音沙哑，**「为什么要救我……」**

**「因为我理解你。」** 阿邪说，**「你的仇，我懂。你的痛，我懂。」**

**「可你有没有想过——你杀了那么多人，那些人的父母妻儿，也和你一样痛苦？」**

**「你用仇恨喂养自己一千年，到头来，你得到了什么？」**

血煞魔君没有说话。

他只是躺在那里，眼角有一滴泪滑落。

一千年的仇恨，一千年的执念，被阿邪一句话，斩得干干净净。

**「我……」** 他的声音像是从很远的地方传来，**「我做了什么……」**

**「太多了……」**

**「可是……」** 他艰难地转过头，看着阿邪，**「还能……挽回吗……」**

阿邪沉默了一下。

**「不能了。」** 他说，**「你杀的那些人，活不过来了。」**

血煞魔君的眼神彻底黯淡下去。

**「但是——」** 阿邪的声音突然响起，**「你还有未来。」**

血煞魔君愣了一下。

**「你有一千年的修为，有化神后期的境界。」** 阿邪说，**「如果你愿意，你可以用你的余生，去做一些……弥补的事。」**

**「保护那些像你当年一样失去父母的孩子。」**

**「帮助那些像你当年一样无依无靠的人。」**

**「用你的力量，守护一方平安。」**

**「这——才是真正的道。」**

血煞魔君看着阿邪，眼神复杂。

**「你……为什么要和我说这些……」**

**「因为我曾经和你一样。」** 阿邪说，**「我曾经也活在仇恨和愧疚里，也觉得全世界都对不起我。」**

**「可后来我明白了——恨是一把双刃剑，伤敌一千，自损八百。」**

**「放下执念，不是认输，而是——解脱。」**

血煞魔君沉默了很久。

最后，他艰难地开口。

**「我……愿意。」**

阿邪伸出手，把他从地上拉了起来。

**「那就——重新开始吧。」**
