# 第十二章：破境筑基

洗心崖上，三年光阴，转瞬即逝。

阿邪在禁地里闭关修炼，一刻不曾懈怠。三年来，他的修为在稳步恢复，从炼气七层一步步爬升，终于在第三年年末，重回炼气九层。

不仅如此——

他还在心魔道上有了一丝领悟。

不是刻意的修炼，不是按部就班的功法，而是一种自然而然的水到渠成。他每天都在和内心对话，每天都在反思，每天都在追问"我要成为什么样的人"。三年下来，他感觉自己的心境发生了翻天覆地的变化。

三年前那个懦弱的自己，好像已经是很遥远的事了。

**「是时候了。」**

出关的前一天晚上，阿邪坐在石屋里，闭上眼睛，开始冲击筑基。

筑基是修仙路上的第一道大关。无数修士卡在炼气九层，终生无法突破。阿邪三年前就是炼气九层，三年后还是炼气九层——不是因为他的天赋不够，而是因为他心里有结，一直解不开。

现在，心结解开了。

灵气在他体内汇聚，越来越浓，越来越重。他引导着这股力量，一遍遍冲刷着自己的经脉，拓宽它，稳固它，让它从一条小溪变成一条大河。

这个过程很痛苦。

但他咬着牙，一声不吭地扛了下来。

三年前他怕疼。可现在他不怕了。经历过比这更痛的事情，还有什么好怕的？

不知过了多久。

轰——

一道金光从他体内冲出，直冲云霄。

筑基，成了。

阿邪睁开眼睛，感觉整个世界都变得不一样了。他能清晰地感受到天地灵气的流动，能听到远处溪水的声音，甚至能闻到风吹过来的花香。

**「筑基初期。」** 他握了握拳头，感受着体内澎湃的力量，**「终于回来了。」**
