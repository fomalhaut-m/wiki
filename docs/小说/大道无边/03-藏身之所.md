# 第三章：藏身之所

阿邪的梦境小屋，在深渊巷道最深处的一道裂缝后面。

这是他唯一能躲开墨渊视线的地方——屋子不大，十来平方，破旧的铁架床、掉漆的衣柜、墙角的纸箱堆，空气里带着淡淡的灰尘味和他身上熟悉的皂角味。他在这里住了三年，三年来每一次从墨渊的任务中逃回来，都躲在这里舔伤口。

他把两个孩子带进屋，反手关上房门，按下反锁。

锁扣发出"咔哒"一声轻响，他的心也跟着落下来一半。

**「这里很安全，」** 他蹲下身，看着两个孩子，**「那个坏人暂时找不到这里。」**

女孩打量着四周，小声问：**「哥哥，你是谁？为什么会救我们？」**

阿邪沉默了一会儿。

**「我……」** 他的声音有些涩，**「我是和你们一样的人。被那些坏人追着跑的人。」**

**「可是你看起来很厉害……」**

**「厉害有什么用。」** 阿邪苦笑一声，没有再解释。

他走到衣柜前，打开柜门，把最厚的一件黑色羽绒服铺在柜底，铺得平平整整。然后他把两个孩子抱进去，让他们靠在柔软的衣料上，又拿了几件卫衣堆在他们身边，挡住他们的身子。

**「不管听到什么，都不要出声，不要出来。」** 他的语气严肃又温柔，**「坏人可能会来找你们，只要你们不出声，他就找不到你们。」**

**「那哥哥你呢？」** 男孩仰着头问。

阿邪愣了一下，随即笑了笑。这个笑容很生涩，他已经很久没有笑过了。

**「哥哥没事。哥哥会守在外面，保护你们。」**

他关上衣柜门，又把门口的几个纸箱挪过来，挡住衣柜的门缝，不留一点痕迹。

做完这一切，他靠在衣柜旁的墙上，缓缓滑坐在地上。

暂时安全了。

可他知道，这只是暂时的。墨渊不会善罢甘休，他一定已经察觉到了自己的动摇。也许明天，也许今晚，那团黑雾就会找上门来。

他的手指不自觉地摸向自己的脸颊。光滑的，没有伤疤。可那种被利爪划开的痛感，仿佛还在骨头里隐隐作痛。

**「没关系。」** 他低声自语，**「就算是死，我也不会把孩子们交出去。」**

他欠那对姐弟的。

他欠自己的。

这一次，他要做一个能保护别人的人。
