# 第十九章：新的旅程

平静的日子过了三年。

三年后——

阿邪在院子里教安安打坐。

**「安安，丹田在哪里？」**

**「在这里！」** 安安拍着小肚子，**「奶奶说这里是仓库，吃饱了饭饭就存在这里！」**

阿邪笑了：**「差不多是这个意思。不过丹田不是用来装饭的，是用来装灵气的。」**

**「灵气是什么？」**

**「灵气就是……空气里的一种能量。你看不见，摸不着，但你能感觉到它。比如现在，风吹过来的时候，你是不是感觉凉凉的？那就是灵气在流动。」**

**「哦——」** 安安似懂非懂，**「那我要怎么把它装进丹田里？」**

**「闭上眼睛，感受它，然后用意念引导它进入你的身体……」**

念念从屋里走出来，手里拿着一封信。

**「哥哥，青云宗来信了。」**

阿邪接过信，拆开一看。

信是周长生写的。

信上说——青云宗最近遇到了麻烦。一个新的魔道势力在东海作乱，杀害了不少修士和凡人。掌门希望阿邪能回宗门一趟，帮忙调查此事。

**「哥哥，你要去吗？」** 念念问。

阿邪沉默了一下。

三年来他一直在回避修仙界的事，把自己封闭在这个小村子里。可现在看来，平静的日子不会一直持续下去。

**「要去。」** 他说，**「这件事和我有关。」**

**「和你有关？」**

**「三年前我离开万魔渊，和墨渊的关系断了。但魔道不会因此放过我。」** 阿邪说，**「这封信，与其说是请我帮忙，不如说是——警告。」**

**「警告？」**

**「意思是，我已经不可能置身事外了。」** 阿邪叹了口气，**「当年我欠下的债，终究还是要还的。」**

念念的眼眶红了：**「哥哥……」**

**「别担心，」** 阿邪摸了摸她的头，**「我只是去处理一些事情，处理完就回来。」**

**「你还记得哥哥说过的话吗？」**

**「记得……」** 念念点点头，**「哥哥答应过我们的，从来没有食言过。」**

**「对。」** 阿邪笑了笑，**「所以别担心，等我回来。」**
