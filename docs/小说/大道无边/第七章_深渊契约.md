# 第七章：深渊契约

墨渊停下了脚步。

他盯着阿邪看了很久，眼神里闪过各种复杂的情绪——惊讶、玩味、思索，最后定格在一种说不清道不明的光芒上。

**「你要代替他们？」** 他的声音带着一丝难以置信，**「你知道这意味着什么吗？」**

**「知道。」** 阿邪说，**「被你吞噬。或者成为你的傀儡，永生永世不得超生。」**

**「既然知道，你还敢说这种话？」**

**「因为我还有利用价值。」** 阿邪说，**「你说过，我的愧疚是你最美味的养料。如果我死了，你的养料就没了。可如果我活着——一直活着，一直愧疚着——你能吃一辈子。」**

**「而且，」** 他顿了顿，**「我还能帮你做更多的事。」**

墨渊沉默了很久。

黑雾在他周身翻涌，他在思考，在权衡，在计算。

最后他笑了。

**「阿邪，你让本座刮目相看了。」**

**「你想要什么？」**

**「第一，」** 阿邪说，**「放过这两个孩子，让他们离开深渊，去人间生活。」**

**「第二，给我三年时间。三年之内，我会替你做三件事。做完三件事，我再来还这笔债。」**

**「第三……」** 他犹豫了一下，**「三年后，不管发生什么，你都不能伤害这两个孩子。他们和你要的孩子不一样——他们的父母，已经被你吞噬了。我不想让他们……再失去更多。」**

墨渊静静地听着。

**「你凭什么觉得本座会答应？」**

**「因为你已经没有更好的选择了。」** 阿邪说，**「杀了我，你就失去了一个永久的养料。放过两个孩子，你就得到一个永远的愧疚源。划算。」**

墨渊盯着他看了很久。

**「契约可以，」** 他终于开口，**「但本座要加一条。」**

**「说。」**

**「三年后，不管你完成没完成三件事，你都要回来。你要是敢逃——」**

他的声音骤然变冷。

**「本座会亲自去人间，找到那两个崽子，把他们的灵魂抽出来，炼成灯笼，点一万年。」**

阿邪的身体僵住了。

他知道墨渊说到做到。

**「成交。」** 他说。

墨渊笑了。

他抬起手，一道黑色的光芒从他指尖射出，没入阿邪的眉心。

刹那间，阿邪感觉有什么东西绑住了自己的灵魂。那是一道无形的锁链，连接着他和墨渊，从今以后，他的生死荣辱，都在这道锁链的一念之间。

**「契约已成。」** 墨渊说，**「三年之期，本座等着你。」**

他转身往门口走去，走到一半，又停了下来。

**「对了，阿邪。」**

**「嗯？」**

**「那三件事，本座已经想好了。」**

他回过头，冰蓝色的竖瞳里闪着幽光。

**「第一件——去青云宗，取回你当年欠他们的东西。」**

**「第二件——找到你三年前没能救下的那对姐弟的魂魄，送他们入轮回。」**

**「第三件——」**

他笑了，笑声在黑暗里回荡。

**「来深渊，见本座。」**

黑雾散去，小屋里重新陷入黑暗。

阿邪站在原地，感受着眉心那道无形的锁链。

他知道自己已经没有退路了。

三年。

他只有三年时间。

他缓缓转过身，打开衣柜。念念和安安蜷缩在里面，两个小家伙不知道什么时候醒了，正睁着圆圆的眼睛看着他。

**「哥哥……」** 念念小声问，**「我们……不用死了吗？」**

阿邪蹲下身，把他们搂进怀里。

**「不用了，」** 他说，声音有些哑，**「坏人答应放过你们了。」**

**「真的吗？」**

**「真的。」**

他抱紧两个孩子，眼眶有些湿润。

**「但哥哥要出远门一趟。一年之后，哥哥就回来接你们。」**

**「哥哥会回来吗？」** 安安问。

阿邪沉默了一会儿。

**「会的，」** 他说，**「哥哥答应过你们，就一定会做到。」**

他不知道这个承诺能不能兑现。但他知道，他必须试一试。

为了孩子们。

为了三年前那对姐弟。

为了他自己。
