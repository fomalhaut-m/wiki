# 第二十一章：渡劫

青云宗后山，天雷峰。

这里是宗门专门给弟子渡劫的地方。天雷峰顶有一片开阔的平台，终年被雷云笼罩，闪电如蛇般在云层中穿梭。

阿邪站在峰顶，仰头看着那片漆黑的雷云。

三年前他在这里禁闭三年，每天看着雷电交加，却从未想过，有一天自己会站在这里，迎接属于自己的雷劫。

**「小邪。」** 周长生站在峰下，**「你确定要在这里渡劫？天雷峰的雷劫比普通地方强三成。」**

**「正因为强三成，才能更好地淬炼心魔。」** 阿邪说，**「放心，我有分寸。」**

**「……」** 周长生欲言又止，最后只是叹了口气，**「小心点。有什么不对，立刻放弃。命比什么都重要。」**

**「我知道。」**

阿邪闭上眼睛，开始调整呼吸。

他的心境和三年前不一样了。三年前他来这里的时候，满心都是愧疚和逃避。可现在——

他没有执念了。

三年前那一战，他把愧疚斩断了。之后的三年，他把遗憾弥补了。念念和安安健康长大，那对姐弟入了轮回，青云宗接纳了他，墨渊的势力瓦解了。

他该做的都做了，该还的都还了。

现在的他，是一身轻松的。

**「来吧。」** 他睁开眼睛，轻声说。

轰——

第一道雷劫劈了下来。

金色的雷电从天而降，正中阿邪的头顶。

那痛感比三年前强了十倍不止。阿邪感觉自己的每一根神经都在燃烧，每一寸骨骼都在碎裂。可他没有躲，也没有退。

他张开双臂，拥抱那道雷电。

**「心魔道——」** 他的声音在雷鸣中响起，**「斩的不是劫，是执念。」**

**「我没有执念了。」**

**「那这雷劫——就是我的力量！」**

金光从他体内爆发，和雷电交织在一起。

第二道。第三道。第四道。

雷劫一道接一道劈下，一道比一道更强。阿邪的身体在颤抖，鲜血从七窍中流出，可他的眼神始终清明。

他不是在对抗雷劫，而是在——融合。

把雷电的力量融入自己的道。

把天劫的淬炼，变成自己的养分。

这是心魔道独有的修炼方式——不是斩断，而是接纳。

接纳恐惧，接纳痛苦，接纳天地之威。

把它们变成自己的力量。

---

第七七四十九道雷劫。

最后一道雷劫劈下的时候，阿邪感觉整个世界都安静了。

然后——

**「轰——！！」**

一道前所未有的金光从他体内冲出，直冲云霄。

那金光驱散了所有的雷云，照亮了整个青云山。

金丹，成了。

阿邪跪在地上，大口大口地喘气。浑身是血，衣衫破烂，看起来狼狈至极。

可他的眼睛，亮得惊人。

**「金丹初期……」** 他感受着体内澎湃的力量，嘴角露出一个笑容，**「终于……回来了。」**
