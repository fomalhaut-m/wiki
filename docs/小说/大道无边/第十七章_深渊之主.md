# 第十七章：深渊之主

墨渊倒在地上，气息微弱。

他没有死。化神巅峰的大能，不可能被一招秒杀。

可他的灵魂被斩伤了。

**「你……」** 墨渊挣扎着想爬起来，可身体不听使唤，**「你对我做了什么……」**

**「斩了你一念。」** 阿邪喘着气说，**「你吸食愧疚的那一念，被我斩断了。」**

**「从今以后，你再也无法用愧疚来喂养自己了。」**

墨渊的眼神里闪过一丝恐惧。

这丝恐惧不是因为阿邪，而是因为——他突然发现，自己的力量源泉，真的枯竭了。

他试着去感应那些被他吞噬的生灵的愧疚，可那里空空如也，什么都没有。

**「不……」** 他的声音发颤，**「这不可能……你只是筑基……怎么可能……」**

**「心魔道的可怕之处，」** 阿邪艰难地从地上爬起来，**「不是在于力量有多强大。」**

**「而是在于——它能斩断一切执念。」**

**「包括你自己的执念。」**

他走到墨渊面前，居高临下地看着他。

**「墨渊，你输了。」**

**「不是因为你打不过我，而是因为——你的道，断了。」**

墨渊躺在地上，眼神空洞。

他想不明白。

他修炼了千万年，吞噬了无数生灵，以为自己已经站在了修仙界的巅峰。可到头来，他输给了一个筑基初期的小娃娃。

**「为什么……」** 他喃喃自语，**「为什么会这样……」**

**「因为你从来不肯面对自己。」** 阿邪说，**「你用吞噬来逃避空虚，用吸食愧疚来填补自己。可这些，都是治标不治本。」**

**「真正能让你解脱的，不是更多的养料，而是——直面你自己的内心。」**

**「可惜，你永远不会懂这个道理。」**

阿邪转身，往深渊出口走去。

**「等等！」** 墨渊的声音从身后传来，**「你要去哪里？」**

**「回去。」** 阿邪说，**「回到人间。回到那些孩子身边。」**

**「然后呢？」**

**「然后好好活下去。」** 阿邪说，**「用我剩下的生命，去做更多有意义的事。」**

**「我不会再回来了。」**

墨渊愣住了。

他躺在冰冷的地上，看着阿邪的背影渐渐远去，消失在永夜的尽头。

他赢了？

不，他没有赢。

他输得彻彻底底。

可更可怕的是——他发现，被斩断执念之后，他竟然感觉到了一丝……轻松。

**「这就是……没有执念的感觉吗……」**

他喃喃自语，声音里带着一丝迷茫，一丝释然。
