# 第四章：深渊降临

深夜。

小屋里的灯泡闪了两下，终于彻底灭了。

门外传来沉重的脚步声，每一步都像踩在心脏上，震得地面微微发颤。阿邪浑身一僵，立刻从地上弹起来，挡在衣柜前。

门缝里开始渗进墨色的雾气，像蛇一样在地板上蠕动，一点点往衣柜的方向爬。

**「阿邪。」**

冰冷的声音穿透门板，清晰地传进屋里。

**「本座知道你藏在里面。」**

阿邪的瞳孔骤然收缩。他下意识地往后退了一步，后背紧紧贴着衣柜，双手攥成拳头，指节因为用力而发白。

他深吸一口气，努力让自己冷静下来。

**「主人。」** 他打开门，站在门口，微微低着头，语气恭敬却不卑微，**「您怎么来了？」**

墨渊从黑雾中走出，三年不见，他看起来没什么变化，依然是那副高高在上的模样，冰蓝色的竖瞳在黑暗里闪着幽光。他没有走进屋，只是站在门外，审视着阿邪。

**「本座很好奇，」** 墨渊的声音带着玩味，**「你为何迟迟未归？那两个孩子，在哪里？」**

阿邪的心狂跳，可他强迫自己镇定下来。

**「主人，属下去找了，」** 他尽量让自己的声音听起来自然，**「但那两个孩子的踪迹，属下确实没有找到。他们似乎逃往了更深处的巷道，属下追了很久……」**

**「是吗？」**

墨渊的声音骤然变冷。

下一秒，黑雾猛地涌向阿邪，把他整个人撞进屋里，后背摔在墙上，一口鲜血喷了出来。

**「你以为本座会信？」** 墨渊踏入小屋，每走一步，地板就结上一层薄冰，**「本座已经感应到了……那两个孩子的气息，就在这间屋子里。」**

阿邪的血液瞬间冰凉。

他挣扎着想站起来，可黑雾已经缠上了他的脚踝，把他死死钉在原地。

**「阿邪，阿邪，」** 墨渊的声音像是在叹息，又像是在嘲讽，**「本座待你不薄。三年前，若不是本座救你，你早就死在宗门追杀之下。如今，你却为了两个素不相识的稚童，一而再，再而三地违抗本座？」**

阿邪没有说话。

他抬起头，直视墨渊那双冰蓝色的眼睛。

**「主人，」** 他的声音沙哑，带着压抑的情绪，**「我只是……不想再眼睁睁看着孩子去死了。」**

**「不想？」** 墨渊笑了，笑声像是从深渊底部传来的回响，**「你有什么资格说这种话？你以为你是谁？救世主？大善人？」**

**「三年前那场火，」** 墨渊的声音突然变得森冷，**「你站在人群里，看着一对姐弟被烧死，连一步都不敢迈出去。事后你辞别宗门，四处流浪，愧疚了三年。是你告诉本座，你愿意做任何事来赎罪。本座给了你力量，给了你活下去的机会——你以为这是免费的？」**

阿邪的身体剧烈颤抖。

那些他最不愿面对的记忆，被墨渊一句话撕开了口子，血淋淋地摆在他面前。

**「我……」**

**「你想做善人？」** 墨渊的利爪抬起，对准阿邪的脸，**「本座倒要看看，你这张脸，能撑多久。」**

锋利的刃口划破空气。

阿邪只感觉脸颊一凉，紧接着是火辣辣的剧痛，像是有什么东西从皮肤上撕裂下来。他咬紧牙关，硬是没让自己发出声音，只是闷哼了一声。

鲜血顺着脸颊滴落，在地上晕开一朵朵暗红的花。

**「说，孩子在哪里？」**

**「我……不知道……」**

**「啪！」**

又是一爪，横着划下，伤口更深，血流得更急。

**「再给你一次机会。」**

**「我真的……不知道……」**

墨渊停下了手。他歪着头，看着满脸是血的阿邪，眼神里闪过一丝异样的光芒。

**「有意思，」** 他低声说，**「你的愧疚……真是本座吃过的最美味的东西。」**

他收回利爪，转身往门口走去。

**「阿邪，本座再给你三天时间。三天之后，若你还不把那两个崽子交出来……」**

他回过头，冰蓝色的竖瞳里满是残忍。

**「本座就让你尝尝，什么叫生不如死。」**

黑雾散去，小屋里重新陷入黑暗。

阿邪靠着衣柜，缓缓滑坐在地上，大口大口地喘气。脸上的伤口火辣辣地疼，可他的心，却比伤口更疼。

他转过头，轻轻把手放在衣柜的门板上。

**「孩子们，」** 他的声音很轻，**「没事了。坏人走了。」**

衣柜里静悄悄的。

过了很久，衣柜门缓缓打开，念念探出头来，眼睛红红的，看见阿邪满脸的血，眼泪一下子涌了出来。

**「哥哥……你的脸……」**

阿邪愣了一下，这才意识到孩子们一直在衣柜里。他的伤势，他咬牙硬撑的剧痛，全都暴露在了孩子们面前。

他本想藏起来的。

**「没事，」** 他挤出一个笑容，哪怕扯动了伤口疼得浑身发抖，也还是笑着说，**「哥哥不疼。」**

念念扑上来抱住他，把脸埋在他的颈窝里，小声哭了起来。安安也跟在后面，紧紧抱住他的腿，小身子还在抖。

**「哥哥……」** 念念的声音带着哭腔，**「都是我们的错……要不是为了藏我们……你也不会……」**

**「傻孩子，」** 阿邪轻轻拍着念念的背，声音温柔得不像话，**「这不是你们的错。这是哥哥自己的选择。」**

**「哥哥不后悔。」**

他的眼神望向窗外的黑暗，心里默默地说：**三年前，我没能救下那对姐弟，那是我一辈子的愧疚。这一次，我绝不会再让悲剧重演。**

**就算是死，也不会。**
