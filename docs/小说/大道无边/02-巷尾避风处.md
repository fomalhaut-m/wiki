# 第二章：巷尾避风处

深渊巷道的尽头，有一片被遗忘的角落。

这里曾经是某座倒塌大殿的废墟，后来被废弃的杂物堆满——破旧的纸箱、腐烂的木板、还有一些不知从哪来的小孩子衣服。风被高墙挡住，进不来，只有微弱的磷火照亮这片狭小的空间。

阿邪贴着墙根往前走，脚步放得极轻。

他听到了哭声。

很小的哭声，压得极低，像是怕被什么人听到，又忍不住从喉咙里溢出来。他停下脚步，整个人僵在原地，心脏狂跳。

是孩子的哭声。

**是那两个灵根稚童。**

他慢慢靠近，躲在一摞纸箱后面，透过缝隙往里看——

一个约莫十二三岁的女孩，把一个更小的男孩紧紧护在怀里。女孩的肩膀一抽一抽的，小声啜泣，怀里紧紧抱着一个掉了左耳的兔子布偶。男孩大概五六岁，明明自己也在发抖，却还伸手拍着姐姐的背，奶声奶气地说："姐姐别哭，安安保护你。"

阿邪的呼吸一下子停住了。

这个画面——和三年前那场火里的一幕，太像了。那时候姐姐也是这样，把弟弟护在怀里，哭得浑身发抖，而他在人群里站着，一步都迈不出去。

他的眼眶一下子红了。

**「念念，别怕……」** 男孩小声说，**「姐姐，安安有拳头，能打坏人。」**

**「安安……」** 女孩的声音带着哭腔，**「爹娘被那些黑雾抓走了……我们再也见不到他们了……」**

**「不会的！」** 男孩握紧小拳头，**「等安安长大了，安安去找他们！安安会保护姐姐的！」**

阿邪看着这一幕，感觉有什么东西在自己胸口炸开。

三年前他没能救下的那对姐弟，是不是也是这样，互相依靠着，在火光里等待救援？可惜没有人来救他们。他也没有。

他躲在那堵墙后面，心脏像是被一只无形的手狠狠攥紧，疼得浑身发冷。

**「怎么办？」** 他在心里问自己。

带他们跑？墨渊的势力遍布整个深渊，哪怕逃到天涯海角也会被找到。

不带走？明天卯时墨渊就会来，届时这两个孩子会被直接拖进深渊，沦为开辟通道的祭品，尸骨无存。

**「三年前我怂了，」** 他的指甲深深掐进掌心，**「这一次，我绝对不能再重蹈覆辙。」**

他深吸一口气，从纸箱后面走了出来。

两个孩子的哭声戛然而止。女孩瞬间停止了哭泣，把弟弟护得更紧，眼睛睁得圆圆的，警惕地盯着他。

阿邪在离他们两步远的地方停下，缓缓蹲下身，让自己的视线和孩子平齐。他把手背到身后——这双手沾染了太多深渊的寒气，别吓到孩子。

**「我不是坏人。」** 他放轻了声音，尽量让自己的语气听起来柔和，**「我不会伤害你们。」**

女孩没有说话，只是把他盯着，眼神里满是戒备。

男孩却歪了歪头，小声对姐姐说：**「姐姐……他身上的味道，和那些黑雾不一样。」**

女孩愣了一下，看了看阿邪，又看了看弟弟。

阿邪没有动，就那样静静地蹲在那里，等着。他的眼神里没有贪婪、没有恶意，只有一种说不清道不明的——愧疚。

他欠这两个孩子的。

他欠三年前那对姐弟的。

他欠他自己的。

**「跟我走，」** 他开口，声音很轻，却很坚定，**「我带你们去一个安全的地方。」**

**「真的？」** 男孩眨眨眼，**「你不会把我们交给那些坏人？」**

**「不会。」** 阿邪看着他们，一字一句，**「我发誓。」**
