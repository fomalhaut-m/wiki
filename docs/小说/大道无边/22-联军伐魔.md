# 第二十二章：联军伐魔

阿邪突破金丹的消息，很快传遍了整个修仙界。

各方势力纷纷派人来打探，想知道这个曾经是万魔渊爪牙的人，是如何在三年内从筑基突破到金丹的。

阿邪没有隐瞒。

他把自己修炼心魔道的经历，一五一十地告诉了来访者。

**「心魔道的核心，不是斩断，而是接纳。」** 他说，**「接纳自己的恐惧，接纳自己的痛苦，接纳自己的不完美。然后——把它们变成力量。」**

这个回答让很多人若有所思。

但也有人嗤之以鼻——

**「说得好听。可真的能做到的，有几个人？」**

**「心魔道本来就是最难走的路，能走通的都是凤毛麟角。」**

**「不过，如果真能复制他的经验，也许整个修仙界的实力都会提升一个台阶……」**

阿邪没有理会这些议论。

他现在的首要任务，是对付血煞魔君。

---

一个月后，修仙界联军组建完成。

参加联军的宗门有十七个，加上散修联盟和几个隐世家族，总兵力超过三千人。

但真正的核心，只有五个人。

青云宗掌门（化神初期）。

玄天阁阁主（化神初期）。

普陀山住持（化神中期）。

剑修第一人——「剑魔」李太白（化神巅峰）。

以及——阿邪（金丹初期）。

**「一个金丹初期来参加这种级别的战斗？」** 有人私下议论，**「不是送死吗？」**

**「你懂什么？三年前他筑基的时候，就敢和墨渊对着干。」**

**「可墨渊只是化神中期，血煞魔君可是化神后期，差了一个大境界呢！」**

**「谁知道呢……也许他又有新的突破……」**

阿邪听到了这些议论，只是笑了笑。

他知道自己在真正的强者面前还有很大差距。但心魔道不是靠蛮力的道。

它是——以心映道。

只要他的心够坚定，只要他的道够纯粹——

没有什么是不可能的。
