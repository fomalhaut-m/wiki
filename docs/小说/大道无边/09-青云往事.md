# 第九章：青云往事

洞府里，周长生给阿邪倒了一杯茶。

**「说说吧，」** 周长生坐在他对面，**「这三年，你都去了哪里？干了什么？为什么不辞而别？」**

阿邪捧着茶杯，沉默了很久。

他不知道该从哪里说起。深渊？墨渊？爪牙？还是那两个孩子？

最后他选择了最简单的说法。

**「我……加入了万魔渊。」**

周长生的手顿住了。

**「什么？」**

**「师兄，你没听错。」** 阿邪的声音很平静，**「这三年来，我是墨渊的爪牙。替他抓人，替他……」**

他说不下去了。

周长生静静地听着，没有打断他。

等阿邪说完，周长生才开口。

**「为什么？」**

**「因为愧疚。」** 阿邪说，**「三年前那场火，我亲眼看着一对姐弟被烧死，没能出手相救。我恨自己，恨自己太懦弱，太自私。之后我无法面对宗门，无法面对任何人，所以选择离开。」**

**「离开之后呢？」**

**「我四处流浪，做散修，想找一条救赎的路。」** 阿邪苦笑一声，**「可散修的日子不好过，没有资源，没有靠山，处处被人欺负。有一次我差点死在妖兽爪下，是墨渊救了我。」**

**「代价呢？」**

**「替他做事。」** 阿邪说，**「他救了我，给了我力量，让我从死亡边缘活下来。作为代价，我要做他的爪牙，替他抓人。」**

**「抓去哪里？」**

**「深渊。」** 阿邪说，**「被他吞噬。永远不得超生。」**

周长生的脸色变了。

**「所以你这次回来……」**

**「第一件事，是向宗门赔罪。」** 阿邪说，**「三年前我不辞而别，给宗门带来了麻烦，我没有尽到一个弟子的责任。这声对不起，我欠了三年，今天该还了。」**

他站起身，对着周长生深深鞠了一躬。

**「师兄，替我转告长老们——阿邪不肖，让宗门蒙羞。今日来请罪，任凭宗门处置。」**

周长生沉默了很久。

然后他叹了口气。

**「小邪，你知道吗，当年你不辞而别，掌门气得差点把你从族谱上除名。外门长老更是扬言，你若敢回宗门，就亲手打断你的腿。」**

阿邪没有说话。

**「可是，」** 周长生的语气软了下来，**「三年过去了，该罚的也罚了，该气也气过了。长老们嘴上说得狠，心里其实一直在惦记着你。」**

**「外门张长老，前段时间还问我，小邪有没有消息。」**

**「掌门私下里也说过，阿邪这孩子本性不坏，只是走的路太顺了，吃了亏才明白事理。总有一天，他会回来的。」**

阿邪的眼眶一下子红了。

**「师兄……」**

**「别哭了，」** 周长生拍了拍他的肩膀，**「男子汉大丈夫，哭什么。去见见长老们吧，他们等你三年了。」**
