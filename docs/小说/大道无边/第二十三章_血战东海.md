# 第二十三章：血战东海

决战那天，东海之上，乌云压顶。

血煞魔君站在海面上，周身环绕着血红色的雾气，那雾气散发着一股浓重的血腥味，让人闻之作呕。

**「哈哈哈哈——」** 他看见联军到来，仰天大笑，**「就你们这群土鸡瓦狗，也敢来挑战本座？」**

**「今日，本座就让你们知道，什么叫——真正的魔道！」**

血雾暴涨，化作千万条血蛇，向联军扑去。

**「结阵！」** 掌门一声令下，联军迅速变换队形，组成了十七道防线。

可血蛇太强了。每一道防线在血蛇面前都支撑不过三息，就被撕开一个口子。

**「让我来！」** 剑修李太白冲出，剑光如虹，一剑斩杀了上百条血蛇。

可他的剑气虽强，面对无穷无尽的血雾，也渐渐力竭。

**「李太白，」** 血煞魔君狞笑，**「听说你是剑道第一人？本座倒要看看，你的剑，能不能挡住本座的血海！」**

血雾再次暴涨，化作一片血海，向李太白压去。

李太白的剑光在血海面前节节败退，眼看就要被吞没——

**「够了。」**

一个平静的声音响起。

阿邪从联军中走出，站在了血海和李太白之间。

**「阿邪？！」** 李太白惊愕，**「你疯了？快退回去——」**

**「李前辈，」** 阿邪回头，对他笑了笑，**「让我来。」**

阿邪没有再说话。

他只是站在那里，面对着那片铺天盖地的血海，眼神平静。

**「心魔道——」**

他轻声说。

**「映。」**

刹那间，一道金光从他体内爆发。

那金光不攻击血海，而是——映照。

映照血煞魔君的内心。

映照那片血海背后的真相。

血煞魔君的攻击骤然停住了。

**「你……」** 他的声音发颤，眼神里闪过一丝恐惧，**「你看到了什么……」**

**「我看到了。」** 阿邪的声音很轻，却清晰地传遍整个战场。

**「你曾经也是一个凡人。」**

**「你也曾经有父母、有妻儿、有想要保护的人。」**

**「可他们都死了。死在修仙者的手里。死在你面前。」**

**「从那以后，你就发誓要报复整个修仙界——让所有人都尝尝失去至亲的痛苦。」**

血煞魔君的脸色变了。

变得苍白，变得扭曲，变得狰狞。

**「住口！」**

**「你的仇恨，我理解。」** 阿邪说，**「因为我也曾经被仇恨蒙蔽过。我以为我活着就是为了赎罪，为了还债。可后来我明白了——」**

**「仇恨只会带来更多的仇恨。报复只会带来更多的报复。」**

**「你要杀的，不只是那些所谓的'正道修士'。你真正想杀的——是那个无能为力的自己。」**

**「住口！！」** 血煞魔君暴怒，血海再次翻涌，向阿邪扑去。

可这一次，那血海的力量明显弱了很多。

因为他的执念——被阿邪映照出来了。

被映照出来的执念，就像被阳光照射的阴影，会慢慢消散。

**「心魔道的第二重——」** 阿邪的声音在金光中响起。

**「映心之后，是——斩。」**

**「不是斩你，是斩你的执念。」**

**「斩你心中的仇恨。」**

**「斩你心中的绝望。」**

**「斩你心中那个——无能为力的自己。」**

金光化作一道利剑，直刺血煞魔君的眉心。

血煞魔君想躲，可他发现自己躲不开——那不是普通的攻击，而是直接作用于灵魂深处。

**「不——！」**

他发出一声嘶吼，整个人向后倒去。

血海失去了力量的支撑，瞬间溃散，化作漫天的血雾，消散在海风中。

血煞魔君躺在海面上，气息微弱。

他没有死。但他所有的执念——被阿邪斩断了。
