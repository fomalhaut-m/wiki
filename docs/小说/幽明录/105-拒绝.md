# 卷五：大道（最终卷）

## 第一百零五章：拒绝

「我拒绝。」

阿邪开口了。

「道"的眼神变了。

「为什么？」

「因为——」阿邪说，「我不能留在这里。」

「我有念念和安安要照顾。」

「我有黎娘要回去见。」

「我有——我的人生要过。」

「可是——」

「可是我愿意成为你的朋友。」

「我愿意——理解你。」

「可我不会——留在这里。」

「我会——回到我的世界。」

「继续走我的路。」

「继续——做我认为对的事。」

"道"沉默了。

过了很久，它才开口。

「你不怕我报复你吗？」

「不怕。」阿邪说，「你已经等了这么久。」

「你不会因为我的拒绝而毁灭一切。」

「因为——你已经学会了等待。」

「学会了——理解。」

「这就是——你的成长。」

"道"看着他。

它的眼神里，有失望，有遗憾——

可更多的，是一种说不清的——

释然。

「你说得对。」它说。

「我学会了等待。」

「学会了——理解。」

「那你走吧。」

「回到你的世界。」

「继续你的生活。」

「可记住——」

「如果有一天——你需要我——」

「我会在这里。」

「等待你。」
