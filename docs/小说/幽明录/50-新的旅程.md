# 卷二：真相

## 第五十章：新的旅程

就在那天傍晚。

阿邪收到了一封信。

信是从很远的地方寄来的。

寄信人的名字，他从未听说过。

可信的内容，却让他久久不能平静。

信上写的是——

「维度回廊的另一端，有一扇门。」

「门后是你从未见过的世界。」

「如果你想知道真相——」

「就来吧。」

信的落款是一个符号。

那个符号，阿邪曾经在那块铜片上见过。

是四维文字。

是——「道」的文字。

阿邪合上信。

他知道自己该怎么做了。
