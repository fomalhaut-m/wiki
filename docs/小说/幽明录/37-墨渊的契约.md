# 卷二：真相

## 第三十七章：墨渊的契约

念念的身体僵住了。

「墨渊？」

「是的。」鲛人点头，「墨渊当时还不是幽都之主。他只是深渊里一个强大的存在。」

「神殿许诺给他很多东西——权力，地位，资源。」

「唯一的条件，就是杀死主人。」

「墨渊答应了？」

「他答应了。」

念念的指甲掐进了掌心。

「为什么？」

「因为神殿告诉他，主人手里有一颗'烛龙内丹'。」

「烛龙内丹？」

「那是烛龙一族代代相传的宝物。」鲛人解释道，「拥有毁天灭地的力量。」

「神殿想要那颗内丹，可他们不敢直接动手。」

「于是他们借墨渊的手来抢。」

「然后呢？」

「墨渊杀了主人。」鲛人的声音变得沉重，「主人为了保护你们，把内丹的力量封印在了自己的龙珠里。」

「他把龙珠留给了你们。」

「然后他——」

她的声音哽咽了。

「他被墨渊吞噬了。」

念念的眼泪夺眶而出。

「那神殿呢？」她问，「神殿拿到了内丹了吗？」

「没有。」鲛人摇头，「墨渊在杀死主人的那一刻，改变了主意。」

「他吞噬了主人的灵魂，却没有把内丹交给神殿。」

「他独吞了那颗内丹的力量。」

「所以神殿和墨渊闹翻了？」

「闹翻了。」鲛人点头，「从那以后，神殿和墨渊就成了死敌。」

「这就是为什么三年前，墨渊会在幽都崛起——」

「——是因为神殿在背后推波助澜？」

「是的。」鲛人说，「神殿想用墨渊来牵制天庭。」

「可他们没想到，墨渊会越来越强大。」

「强大到连他们都控制不了。」
