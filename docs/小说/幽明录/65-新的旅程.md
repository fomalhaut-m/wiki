# 卷四：归途

## 第六十五章：新的旅程

阿邪又一次站在了十字路口。

他本以为自己可以过平静的日子。

可命运不允许。

「我要走了。」他对念念和安安说。

「去哪里？」念念的眼眶红了。

「去找一些——朋友。」

「什么样的朋友？」

「一些能帮助我们的朋友。」

安安握住他的手。

「哥哥，我们能一起去吗？」

阿邪看着他们。

他不想让他们涉险。

可他也知道——把他们单独留下，更危险。

「……好。」他说，「一起去。」

念念和安安的脸上露出了笑容。

黎娘站在一旁，看着他们。

「你们走吧。」她说，「我会在这里等你们回来。」

「黎娘——」

「别婆婆妈妈的。」黎娘摆摆手，「我七十三了，什么场面没见过。」

「去吧去吧。」

「办完事就回来。」

阿邪看着她。

「谢谢你，黎娘。」

「说了别谢。」黎娘笑了，「快走吧，别让我等太久。」

---

阿邪带着念念和安安，离开了扶摇海。

他们的目的地，是三十六洞天之一。

剑无痕说，那里藏着一个"观测者"的碎片。

那个碎片，可能是他们的希望。
