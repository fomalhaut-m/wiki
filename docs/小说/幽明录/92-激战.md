# 卷四：归途（终）

## 第九十二章：激战

阿邪、念念、安安——三个人一起向毁灭者发起了攻击。

阿邪释放出四维世界的力量，光芒如潮水般涌向毁灭者。

念念召唤出巨大的水龙，带着滔天的海浪。

安安凝聚出凶猛的火凤，带着毁灭一切的气势。

三道攻击同时击中毁灭者——

可毁灭者只是挥了挥手，就将所有攻击化解。

「就这？」毁灭者冷笑，「不过如此。」

它反手一挥，一道黑色的闪电击中了念念。

「念念！」阿邪大喊。

念念被打飞出去，重重撞在地上。

「姐姐！」安安冲过去。

「我没事——」念念挣扎着站起来，「只是——受了点伤。」

可阿邪看出来了——

念念的伤很重。

毁灭者的力量远超他们的想象。

「看来——」毁灭者说，「你们也不过如此。」

「那我就——送你们一程。」

它抬起手，准备发动致命一击。

阿邪挡在念念和安安面前。

「我不会让你伤害他们。」

「哦？」毁灭者笑了，「就凭你？」

「就凭我。」阿邪说，「还有——」

他闭上眼睛。

「还有——所有那些相信我的人。」
