# 卷四：归途（终）

## 第九十四章：胜利的代价

毁灭者被光芒吞噬。

它发出痛苦的嘶吼。

然后——

消散了。

彻底的消散。

阿邪站在原地，大口喘着气。

他赢了。

可代价也很惨重。

念念和安安都受了重伤。

他们需要很长时间才能恢复。

「哥哥——」念念虚弱地说，「我们——赢了吗？」

「赢了。」阿邪说，「暂时赢了。」

他看向天空。

天空恢复了平静。

可他知道——这只是暂时的。

「道」还在。

毁灭者只是它的一个使者。

还会有更多的敌人。

更多的危险。

可至少——

他们还有时间。

还有时间变得更强。

还有时间——

找到最终的答案。
