# 卷三：维度

## 第五十三章：门

阿邪带着念念和安安进入了建木。

建木的内部和中空。

他们沿着一条螺旋向上的通道走着，通道的墙壁是金色的，上面刻满了密密麻麻的符文。

那些符文和阿邪在铜片上看到的一样——是四维文字。

「这些字是什么意思？」念念好奇地问。

「我也不懂。」阿邪说，「可我想，等我们到了四维世界，就会懂了。」

他们走了很久。

通道越来越窄，越来越暗。

到了最后，他们几乎是在爬行。

念念和安安有些害怕，可他们没有抱怨。

他们紧紧跟着阿邪，一步也不落下。

终于——

他们看到了光。

那不是普通的光。

是——无数种颜色的光，叠加在一起形成的奇异景象。

阿邪停下脚步。

他看见了一扇门。

那扇门就悬浮在他们面前。

门是透明的，像是一块凝固的水晶。

门上没有把手，没有锁孔。

只有——

一双手的印记。

「这是——」

「这是你的门。」守树人的声音从身后传来。

阿邪回头，发现守树人不知什么时候跟了进来。

「我说过，只有你能打开它。」守树人说，「因为这扇门，是你内心的投影。」

「只有你伸出手，承认自己，它才会为你打开。」

「承认自己？」

「承认你是谁。」守树人说，「承认你的过去，你的现在，你的未来。」

「承认你所有的恐惧，所有的希望，所有的羁绊。」

「只有这样，门才会为你打开。」

阿邪看着那扇门。

门上映着他自己的影子。

还有一个模糊的影子——那两个他曾经没能救下的孩子。

「我明白了。」他说。

然后，他伸出了手。
