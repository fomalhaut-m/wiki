# 第十一章：拦截

那黑影站在他们面前，像一堵黑色的墙。

阿邪下意识地把念念和安安护在身后。

「你是什么人？」他问。

黑雾中传来一阵低沉的笑声。

「什么人？」那声音说，「阿邪，你不认识我了？」

黑雾缓缓散去，露出一张脸。

那是一张苍白的脸，眉目清秀，看起来不过二十出头，可那双眼睛里却透着一种与年龄不符的沧桑。

阿邪的瞳孔骤然收缩。

「是你。」

「是我。」那人笑了，「三百年了，阿邪。三百年不见，你还是老样子。」

「认识？」念念在旁边小声问。

「认识。」阿邪的声音很沉，「他叫无赦。是墨渊手下的四大战将之一。」

「三百年前，我在一次任务中和他交过手。」

「差点死在他手里。」

无赦听完阿邪的介绍，笑容更深了。

「不错，记忆力很好。」他说，「我记得那次，你被我打断了三根肋骨，在床上躺了整整三个月。」

「没想到你还活着。」

「我命大。」阿邪说。

「命大？」无赦摇摇头，「不是命大。是主人不想让你死。」

「主人的意思是——」

他的眼神变得森冷。

「你可以走。」

阿邪的身体僵了一下。

「什么意思？」

「字面意思。」无赦说，「主人说了，你和这件事无关，可以自行离开。」

「但是——」

他的目光越过阿邪，落在念念和安安身上。

「这两个崽子，要留下。」

阿邪的心猛地一沉。

果然。

墨渊没有那么好说话。

「不行。」他说。

「不行？」无赦挑了挑眉，「阿邪，你搞清楚状况。我不是在跟你商量。」

「我知道。」阿邪说，「但我不会把他们交给你。」

无赦看着他，眼神变得玩味起来。

「有意思。」他说，「三年了，你终于学会说'不'了。」

「早该这样。」

他抬起手，黑雾在他掌心凝聚。

「不过，主人说的条件，是有期限的。」

「你有两个时辰。」

「两个时辰之内，你不去归墟，我就当你是拒绝。」

「拒绝的下场——」

他看向念念和安安。

「你应该清楚。」

阿邪的拳头紧紧攥着。

两个时辰。

从尘秽市到归墟，最快也要三天。

他不可能在两个时辰内到达。

除非——

「我有办法。」他说。

无赦挑眉：「哦？」

「告诉我你的条件。」阿邪说，「完整版的条件。」

无赦愣了一下，然后笑了。

「你倒是学聪明了。」

他收回了凝聚的黑雾，负手而立。

「条件很简单。」

「第一，这两个崽子暂时留在尘秽市，由我的人看管。」

「第二，你必须在三个月内完成三件事。」

「什么事？」

「第一，去归墟，找到'建木残根'，带回来。」

「第二，去赤水河底，找到烛龙的龙珠，带回来。」

「第三——」

无赦的眼神变得意味深长。

「去尘秽市地下的镜殿，找到里面的东西，带回来。」

阿邪的心跳加速。

建木残根。龙珠。镜殿。

每一样都不简单。

「为什么要这些？」他问。

「不该你问的别问。」无赦说，「你只需要知道，主人要这三样东西，有他的用处。」

「三个月内完成，这两崽子就自由了。」

「完成不了——」

他没有说完。

但那意思已经很明显了。

阿邪沉默了。

他在权衡。

三个月的期限，三件几乎不可能完成的任务——墨渊这是在故意刁难他吗？还是说，他真的需要这三样东西？

他不知道。

可他知道，他没有选择。

「好。」他从牙缝里挤出这个字，「我接受。」

无赦笑了。

「爽快。」

「不过——」他的笑容收敛，「在你出发之前，我得先做一件事。」

他抬起手，一道黑光射出，没入念念的眉心。

「你干什么！」阿邪暴怒，冲上前去。

可他还没碰到无赦，就被一股无形的力量弹开。

「别紧张。」无赦说，「只是在她身上留了个印记。」

「这样，不管她跑到哪里，我都能找到她。」

「同样——」

他又射出一道黑光，没入安安的眉心。

「你！」

「放心，不是什么致命的东西。」无赦说，「只是一个定位印记而已。」

「三个月后，你完成任务，我自然会解除。」

「如果你跑了，或者三个月内没完成——」

他笑了笑，没有说完。

阿邪的拳头攥得咯咯作响。

他知道，无赦这是在拿念念和安安做人质。

这是墨渊的老手段。

可他没有办法拒绝。

因为这是他唯一的选择。

「好。」他从牙缝里挤出这个字，「三个月。」

无赦满意地点点头。

「那就这样。」

「三个月后，同一个地方，我等你。」

他转身，黑雾在他脚下升起，带着他缓缓升空。

「对了。」他忽然想起了什么，「那两个时辰，我改主意了。」

「现在就计时。」

「三个月零两个时辰。」

「超时——」

他消失在黑雾中，只有笑声在空气中回荡。

「后果自负。」
