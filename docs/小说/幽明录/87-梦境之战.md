# 卷四：归途（终）

## 第八十七章：梦境之战

梦魇向阿邪发起了攻击。

黑色的雾气从他身上涌出，化作无数尖锐的触手，向阿邪扑来。

阿邪没有躲避。

他伸出手，一道光芒从他掌心升起。

那是四维世界的气息。

是他从混沌中带回的力量。

光芒和触手碰撞在一起，发出震耳欲聋的轰鸣。

梦境开始震颤。

「你——」梦魇的声音变了，「你身上有'道'的力量？」

「不是'道'的力量。」阿邪说，「是我自己的力量。」

「是——超越'道'的可能性。」

梦魇的脸色变了。

它没想到阿邪会这么强。

它派出过很多使者，去抹除那些威胁"道"的存在。

可从来没有人——能正面抵挡它的攻击。

「你——到底是什么？」

「我是一个选择相信自己的人。」阿邪说。

他向前迈出一步，光芒更盛。

「也是一个选择保护别人的人。」

「这就是——我和你最大的不同。」
