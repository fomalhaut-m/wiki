# 卷四：归途（续）

## 第七十八章：裂隙深渊

离开天璇洞之后，阿邪他们来到了下一个目的地。

裂隙深渊。

这是尘世和幽都交界处的一道巨大裂缝。

传说这道裂缝是当年天地大劫时留下的，深不见底，连接着无数个空间。

这里是魔道中人最喜欢聚集的地方。

因为在这里，正道的法则力量最薄弱。

「小心点。」阿邪叮嘱道，「这里很危险。」

「我们只是来找东西，不要惹麻烦。」

他们沿着裂缝的边缘向下走去。

越往下走，光线越暗，空气中的灵气也越混乱。

到了最后，他们来到了一个巨大的地下空间。

空间里到处都是黑色的岩石和紫色的火焰。

还有一些奇形怪状的生物在游荡。

「那就是五元子的位置吗？」念念小声问。

「应该是。」剑无痕说，「五元子掌管火系的法则。」

「它最喜欢的地方，就是这种灼热的地方。」

阿邪看向空间的深处。

那里有一团紫色的火焰在燃烧。

那火焰很大，像一座小山。

「走。」

他们向那团火焰走去。
