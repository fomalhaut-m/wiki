# 卷五：大道（最终卷）

## 第一百章：契约

阿邪和"道"对话了很久。

他们谈论了混沌，谈论了三维世界，谈论了梦境和现实。

他们谈论了孤独，谈论了理解，谈论了——希望。

「我想和你做一个约定。」"道"说。

「什么约定？」

「你继续做你的选择。」

「继续走你的路。」

「继续——成为这个世界的守护者。」

「而我——」

「我会继续我的梦。」

「可不会再试图控制你。」

「不会再派出使者来抹除你。」

「我会给你时间。」

「给你空间。」

「给你——选择的权利。」

阿邪看着"道"。

他没想到"道"会提出这样的约定。

「为什么？」

「因为——」"道"说，「你是第一个愿意理解我的人。」

「第一个愿意成为我的朋友的人。」

「我不想失去你。」

阿邪沉默了。

他想起了观测者说的话。

想起了"道"是混沌的一部分。

想起了它为什么会分裂。

原来——

它不是想毁灭一切。

它只是——想要被理解。

「好。」阿邪说，「我接受这个约定。」

「我会继续走我的路。」

「也会——继续理解你。」

「直到有一天——」

「你能真正理解这个世界的美好。」

「理解——生命的价值。」
