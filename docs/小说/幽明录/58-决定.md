# 卷三：维度

## 第五十八章：决定

「我需要知道，」阿邪开口，「如果你说的都是真的——为什么你要告诉我这些？」

「为什么要给我选择的机会？」

观测者笑了。

「因为你是第一个。」他说，「第一个来到这里的'梦中人'。」

「之前的观察者——一元子，烛阳，还有那个在镜殿里自称'维度观测者'的意识——」

「他们都是'道'的一部分。」

「只有你，是真正从梦境中走出来的存在。」

「你是第一个。」

「也是唯一一个。」

阿邪愣住了。

「所以——你一直在这里等我？」

「是的。」观测者点头，「我等了很久很久。」

「等一个能够做出选择的人。」

「不是因为我想被拯救。」

「是因为——」

他的声音变得温柔。

「是因为，这个梦里的人——」

「他们也有活着的权利。」

「他们不是'道'的棋子，不是命运的玩偶。」

「他们是——生命。」

「是应该被尊重的存在。」

「所以——」

他看向阿邪。

「所以我需要你做出选择。」

「不是因为我想怎么样。」

「是因为——这是属于你们的权利。」

「你们有权利决定自己的命运。」

阿邪看着他。

他看到了自己。

不是镜子里映出的自己，而是——更深处的自己。

那个在无尽可能性中游荡的，真正的自己。

「我明白了。」他说。

他转过身，看向念念和安安。

「你们呢？」他问，「你们想怎么做？」

念念和安安对视一眼。

然后，念念开口了。

「哥哥，」她说，「我不懂什么混沌，什么维度。」

「我只知道一件事。」

「什么？」

「我不想让任何人消失。」

「不管是好人还是坏人。」

「只要是活着的，就有活着的价值。」

阿邪看着她。

他没想到，念念会说出这样的话。

「安安呢？」他问。

「我——」安安挠挠头，「我也一样。」

「我不想让姐姐消失。」

「不想让哥哥消失。」

「不想让黎娘婆婆消失。」

「不想让任何我们认识的人消失。」

「所以——」

「所以哥哥，」念念接过话头，「不管你做什么决定，我们都支持你。」

阿邪看着他们。

他的眼眶湿了。
