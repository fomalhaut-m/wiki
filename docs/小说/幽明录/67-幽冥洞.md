# 卷四：归途

## 第六十七章：幽冥洞

穿过裂缝之后，阿邪他们发现自己来到了一个完全不同的世界。

这里的天空是紫色的，不是白天也不是黑夜，而是一种永恒的黄昏。

地面上长满了奇异的植物——有的发着光，有的会移动，有的——似乎在看着他们。

「这里就是洞天？」念念好奇地四处张望。

「是的。」剑无痕说，「幽冥洞。」

「传说这里曾经是幽冥之神的居所。」

「后来幽冥之神消失了，这里就成了一个无主的空间。」

「可里面的灵气依然浓郁，是修炼的绝佳场所。」

阿邪没有说话。

他的注意力被别的东西吸引了。

他感觉到了——

有什么东西在呼唤他。

「这边。」他说。

他向着呼唤的方向走去。

众人跟在他身后，穿过了一片发光的森林，越过了一条银色的河流，最后——

他们来到了一座宫殿前。

宫殿很古老，墙壁上布满了青苔和藤蔓。

可宫殿的大门却是敞开的，里面透出淡淡的光芒。

「有人吗？」念念小心翼翼地问。

没有回应。

只有回声在宫殿里回荡。

阿邪走进宫殿。

宫殿内部很大，到处都是灰尘和蛛网。

显然已经很久没有人来过了。

阿邪沿着走廊向前走，最终来到了一间石室前。

石室的门也是开着的。

他走了进去。

然后——

他看见了一个人。

一个盘腿坐在石台上的老人。

老人的身体已经干枯了，皮肤紧贴在骨头上，像是一具干尸。

可老人的眼睛却是睁着的。

那双眼睛还在发光——淡淡的金色光芒。

「你来了。」

老人开口了。

那声音苍老而疲惫，却带着一种说不出的威严。

「我等你——很久了。」
