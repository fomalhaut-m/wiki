# 卷五：大道（最终卷）

## 第一百零四章：请求

「那你想让我做什么？」阿邪问。

「我有一个请求。」「道"说。

「什么请求？」

「留在这里。」「道"说，「留在我身边。」

「作为我的观测者。」

「代替我——观察三维世界。」

「这——」阿邪愣住了。

「我知道这对你来说很为难。」「道"说，「你有你想保护的人。」

「你有你想回去的地方。」

「可我——」

「我已经等待了太久太久。」

「我不想再孤独下去了。」

「我想——有一个人能真正理解我。」

「哪怕——只是陪伴也好。」

阿邪看着"道"。

那不是一张神的脸。

那是一张——充满孤独的脸。

和无数年前，他在海边遇到的某个孩子——

有着相同的表情。

「我——」
