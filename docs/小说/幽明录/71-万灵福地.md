# 卷四：归途（续）

## 第七十一章：万灵福地

万灵福地位于尘世的西南方向。

传说这里是最早的福地之一，天地初分时就已存在。

这里的万物皆有灵——不只是动物和植物，连石头和水都有自己的意识。

阿邪带着念念和安安，经过半个月的跋涉，终于来到了这里。

「哇……」

念念看着眼前的景象，惊叹出声。

他们面前是一片广阔的原始森林。

森林里的树都有几十丈高，树干粗得需要十几个人才能合抱。树叶呈现出七彩的颜色，在阳光下闪闪发光。

树与树之间，缠绕着各种各样的藤蔓。那些藤蔓在微微颤动，像是在呼吸。

地上铺满了厚厚的落叶，落叶下面偶尔能看到一些小东西在蠕动——不是虫子，是某种有灵性的微小生物。

「这就是万灵福地？」安安好奇地四处张望，「好神奇啊。」

「小心点。」阿邪叮嘱道，「这里的东西都有意识，不要乱碰。」

「它们会生气吗？」

「可能会。」

安安缩了缩脖子。
