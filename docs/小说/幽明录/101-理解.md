# 卷五：大道（最终卷）

## 第一百零一章：理解

白色空间里，阿邪和"道"面对面站着。

「理解？」阿邪皱眉，「你是'道'，是创造了这一切的存在。」

「你需要被理解？」

「是的。」「道"说，「我创造了三维世界，创造了一切秩序。」

「可没有人——能真正理解我。」

「他们都把我当作高高在上的神。」

「都对我顶礼膜拜。」

「却从来没有人——试着理解我。」

阿邪沉默了。

「你想让我理解你什么？」

「理解我为什么创造这个世界。」「道"说，「理解我为什么分裂出那些观测者。」

「理解我为什么——选择了你。」

「我选择你——不是因为你有多强大。」

「是因为你——」

「让我感到了一丝——」

「希望。」
