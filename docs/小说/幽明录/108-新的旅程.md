# 卷五：大道（最终卷）

## 第一百零八章：新的旅程

平静的日子过了一年。

一年后的某一天——

阿邪收到了一个人的来访。

那是一个年轻人，穿着一身白衣，脸上带着淡淡的笑意。

「你是——」阿邪问。

「我是七元子。」年轻人说，「'道'的第七个碎片。」

「我找了很久，才找到你。」

阿邪看着他。

「你来找我做什么？」

「来告诉你一件事。」七元子说，「'道'醒了。」

阿邪的心沉了下去。

「醒了？」

「是的。」七元子点头，「它做了一个决定。」

「什么决定？」

「它决定——结束它的梦。」

阿邪的瞳孔收缩。

「你是说——」

「三维世界会消失。」七元子说，「所有的一切，都会回归混沌。」

阿邪沉默了。

他早就知道这一天会来。

可没想到——会来得这么快。

「你来找我——是为了什么？」

「是为了告诉你——」「道"给你留了一个选择。」

「什么选择？」

「你可以带着念念和安安，跟我走。」

「去哪里？」

「去一个——不会被毁灭的地方。」

阿邪沉默了。

他看向念念和安安。

他们站在门口，脸上带着担忧的表情。

「让我想想。」他说。
