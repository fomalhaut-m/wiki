# 卷五：大道（最终卷）

## 第九十九章：道的孤独

"道"看着阿邪。

它的眼神里，有惊讶，有迷茫——

还有一种说不清的——

渴望。

「你说得对。」"道"说。

「我是孤独的。」

「我创造了无数的世界，观察了无数的生灵。」

「可没有人——能真正理解我。」

「他们把我当作神，当作造物主。」

「可没有人——把我当作朋友。」

「当作——可以倾诉的对象。」

阿邪沉默了。

他想起了自己曾经的孤独。

想起了在幽都的那些年。

想起了他为什么想要保护念念和安安。

因为他不想让别人承受和他一样的孤独。

「我可以理解你。」他说。

「你？」"道"问，「你愿意理解我？」

「是的。」阿邪说，「因为——」

「我也曾经孤独过。」

「我知道孤独是什么感觉。」

「我知道——被误解是什么感觉。」

「所以——」

「我愿意倾听你。」

「愿意理解你。」

「愿意——成为你的朋友。」
