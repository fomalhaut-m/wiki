# 卷四：归途（续）

## 第七十七章：四元子

在洞的最深处，阿邪他们找到了四元子。

四元子不是一个人，而是一滴水。

一滴悬浮在虚空中的水。

那滴水很大，有脸盆那么大，表面流动着复杂的光芒。

「这就是四元子？」念念惊讶地看着那滴水。

「是的。」剑无痕说，「四元子掌管水系的法则。」

「它是'道'分裂出的第四个意识碎片。」

「水无常形，无常势。」

「所以它的形态和你想象的不一样。」

那滴水动了动。

然后，一个声音从中传出。

那声音很柔和，像流水一样。

「你们来了。」

「我等你们很久了。」

阿邪上前一步。

「四元子前辈。」

「不必多礼。」那滴水说，「我知道你在找什么。」

「你在找对抗'道'的方法。」

「我帮不了你太多。」

「但我可以给你一样东西。」

那滴水的表面泛起涟漪。

然后，一个东西从中飘了出来。

那是一块透明的晶石。

「这是水之魂。」四元子说，「蕴含着水系法则的精华。」

「把它交给你的同伴——那个叫念念的女孩。」

「她的血脉里流淌着烛龙的力量。」

「烛龙是水火双系的神兽。」

「有了这个，她就能掌控水系的力量。」
