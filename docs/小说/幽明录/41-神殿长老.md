# 卷二：真相

## 第四十一章：神殿长老

神殿的核心是一座巨大的议事堂。

议事堂里有十二把椅子，围成一个圆圈。

这就是神殿的最高权力机构——长老会。

白渊和阿邪走进去的时候，堂内已经坐了七个人。

这七个人就是神殿的七位长老。

「白渊，你回来了。」坐在主位上的老人开口了。

他看起来很老了，满头白发，脸上布满了皱纹。

可他的眼睛却很亮，亮得像两颗星星。

「师叔。」白渊躬身行礼。

「这位是——」另一个长老看向阿邪。

「我的朋友。」白渊说，「阿邪。」

「阿邪？」那长老的眉头皱了起来，「就是那个曾经在幽都为墨渊做事的人？」

阿邪的心一沉。

他们已经调查过他了。

「是的。」白渊没有否认，「他曾经是墨渊的部下。」

「可他后来背叛了墨渊，亲手杀死了他。」

「这种事，你怎么知道的？」

「我亲眼看见的。」

「你亲眼看见的？」那长老的声音变得尖锐，「你在场？」

「我在场。」白渊说，「我和阿邪一起，在深渊里和墨渊对峙。」

「我亲眼看见他用心魔道唤醒了墨渊最后的意识。」

「墨渊最后选择了自我毁灭。」

「这就是阿邪杀死墨渊的方式。」

议事堂里一片沉默。

七位长老面面相觑，不知道该说什么。

「那他现在来这里做什么？」主位上的老人开口了。

「我请他来帮忙的。」白渊说，「查一件案子。」

「什么案子？」

「关于——」白渊停顿了一下，「关于烛龙的案子。」

全场哗然。

「烛龙？」

「三百年前被墨渊杀死的那个？」

「白渊，你提这个做什么？」

「因为我在师父的日记里发现了一些东西。」白渊说，「关于烛龙之死和神殿的关系。」

「神殿和烛龙之死有什么关系？」主位上的老人问。

「有直接关系。」白渊说，「神殿——」

「住口！」

一个长老突然站了起来。

「白渊，你知不知道你在说什么？」

「我知道。」白渊说。

「你知道还敢说？」

「我为什么不敢说？」

「因为——」那长老的脸色变得铁青，「因为这会毁了神殿！」

阿邪看着这一幕，嘴角微微扬起。

看来，神殿内部的裂痕比他想象的还要深。
