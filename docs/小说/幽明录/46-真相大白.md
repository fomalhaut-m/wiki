# 卷二：真相

## 第四十六章：真相大白

三位长老消失后，天空恢复了平静。

裂缝愈合了。

神殿恢复了原样。

可所有人都知道，有什么东西不一样了。

「'道'显现了……」

「我们被审判了……」

「这是真的……」

恐慌和议论在神殿中蔓延。

可阿邪知道，这只是开始。

「道」给了他一个机会。

一个揭露神殿所有秘密的机会。

他用了三天时间，把神殿内部所有参与当年事件的名单，全部列了出来。

然后，他把这份名单，投递给了天界。

很快，天界的裁决下来了。

神殿被勒令解散。

所有参与当年事件的人，全部被逮捕。

那些不知情的弟子，被允许离开。

而那些知情的、参与的——

他们的名字，被刻在了一块石碑上，立在归墟之滨。

那块石碑叫做「罪碑」。

碑上刻着所有参与者的名字，以及他们犯下的罪行。

任何一个路过的人，都可以去看那块碑。

都可以知道，白木神殿曾经做过什么。
