# 卷四：归途（终）

## 第九十章：新的挑战

平静的日子持续了三个月。

三个月里，念念和安安的能力越来越强。

念念能够召唤强大的水龙，安安能够凝聚毁灭性的火凤。

他们的配合也越来越默契——水火相济，攻守兼备。

阿邪也在进步。

三元子的记忆在他脑海中不断发酵，转化为他自己的力量。

他找到了一条新的修炼之路——不是传统的承劫，也不是心魔道——

而是一种融合了四维知识的新道。

一种——属于他自己的道。

可就在一切都在向好的方向发展时——

危险，再次降临。

---

那是一个黄昏。

阿邪正在院子里教念念和安安修炼。

突然，天空裂开了。

一道巨大的裂缝横贯天际，从裂缝中走出一个身影。

那身影比梦魇更强大，更可怕。

它的身上散发着毁天灭地的气息。

「又来了。」阿邪的眼睛眯了起来。

「这次——是谁？」

那身影开口了，声音如雷霆：

「吾乃——毁灭者。」

「'道'最后的使者。」

「来——抹除你们。」

阿邪站起身。

他知道——最后的决战，就要来了。
