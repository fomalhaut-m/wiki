# 卷五：大道（最终卷）

## 第九十六章：恢复与成长

念念和安安的伤养了三个月才完全恢复。

这三个月里，他们没有闲着。

念念在练习控制水之魂的力量——从最基础的吸水、喷水，到能够召唤滔天巨浪。

安安在练习控制火之魂的力量——从最基础的点火、控火，到能够凝聚毁灭一切的火焰。

他们的进步速度超出了所有人的预期。

「不愧是烛龙的血脉。」剑无痕感叹，「这种觉醒速度，放在整个修仙界都是罕见的。」

阿邪看着他们成长，心里既欣慰又担忧。

欣慰的是他们的力量越来越强。

担忧的是——

他们要面对的敌人也越来越强。

「哥哥。」念念走过来，「我们的力量已经恢复得差不多了。」

「什么时候——再出发？」

阿邪看着她。

「你想去哪里？」

「去——」念念想了想，「去面对'道'。」

「我想知道——这一切的真相到底是什么。」

阿邪沉默了。

他想起了四维世界里观测者告诉他的话。

想起了三维世界是"道"的梦。

想起了总有一天，梦境会醒来。

「好。」他说，「那我们就去。」

「去找'道'。」

「去找到——真正的答案。」
