# 卷四：归途（续）

## 第七十五章：礼物

「年轻人。」老树的声音打断了阿邪的思绪。

「嗯？」

「我能感觉到，你身上有'道'的印记。」

「可那印记——和你不完全融合。」

「你是想走自己的路，对吗？」

「是的。」

「好。」老树说，「那我送你一份礼物。」

树干上突然裂开了一道缝。

从缝里，飘出了一颗绿色的珠子。

珠子很小，只有弹珠大小，表面流动着淡淡的光芒。

「这是什么？」

「这是万灵珠。」老树说，「是神树最后的力量结晶。」

「它可以帮你——」

「理解万物的语言。」

「和它们沟通。」

「不管是动物、植物，还是石头和水。」

「你都能听到它们的声音。」

阿邪接过那颗珠子。

珠子入手温润，带着淡淡的暖意。

「谢谢前辈。」

「不必谢我。」老树说，「我只是——在等待一个能听懂的人。」

「等了很久很久。」

「现在，我终于等到了。」
