# 卷四：归途（续）

## 第七十九章：五元子

走到火焰前，阿邪他们停下了脚步。

「五元子前辈。」阿邪开口，「晚辈有事相求。」

紫色火焰剧烈跳动了一下。

然后，一个暴躁的声音从中传出：

「什么事？有屁快放！」

阿邪愣了一下。

这个五元子的性格，和他想象的完全不一样。

「我在寻找观测者的碎片，想要集合大家的力量，对抗'道'。」

「对抗'道'？」火焰的声音变得嘲讽，「就凭你？」

「你知道'道'有多强吗？」

「你知道我们这些碎片有多弱吗？」

「我们加起来，也不过是'道'的一小部分！」

「你凭什么觉得你能对抗它？」

阿邪沉默了。

他不知道该怎么回答。

「因为我不想放弃。」

他最终还是开口了。

「我不想看着所有人——都在'道'的梦里消失。」

「我不想让这个世界——只是某个疯子的幻想。」

「我想让它——成为真实的。」

「成为值得被守护的。」

火焰沉默了。

过了很久，它才再次开口。

这一次，声音里的暴躁消失了，取而代之的是一种奇异的平静。

「……很久没有人这么和我说话了。」

「上一个这么说的人——」

「是一元子。」

「他当年也说过同样的话。」

「他说——」

「'我想让这个世界变得更好。'」

「然后他就消散了。」

「为了对抗'道'，他把自己燃烧殆尽了。」

火焰剧烈跳动了一下。

「你——」

「你真的想走那条路吗？」

「那条路——没有回头的。」
