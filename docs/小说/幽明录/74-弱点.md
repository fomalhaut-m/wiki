# 卷四：归途（续）

## 第七十四章：弱点

「弱点？」阿邪的眼睛亮了起来，「'道'有弱点？」

「是的。」老树说，「每一个分裂出去的存在，都有弱点。」

「包括那些观测者的碎片。」

「包括——'道'本身。」

「它的弱点是什么？」

「孤独。」

阿邪愣了一下。

「孤独？」

「是的。」老树说，「'道'是一个分裂的意识。」

「它从混沌中分裂出来，就注定是孤独的。」

「它观察万物，却没有人能真正理解它。」

「它创造了世界，却没有人能真正走进它的内心。」

「这种孤独——是它最大的弱点。」

「因为孤独，所以它渴望被理解。」

「因为渴望被理解，所以它一直在等待——」

「等待一个能真正读懂它的人。」

阿邪沉默了。

他想起了观测者说的话。

想起了"道"为什么会分裂出那些观测者的碎片。

想起了——它为什么会给他选择的机会。

原来——

它不是高高在上的神。

它只是一个孤独的存在。

在无尽的梦境中，等待着有人能懂它。
