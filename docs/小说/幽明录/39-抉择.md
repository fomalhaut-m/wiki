# 卷二：真相

## 第三十九章：抉择

阿邪看着白渊。

「我为什么要帮你？」

「因为这不只是神殿的事。」白渊说，「这也是念念和安安的事。」

「他们的父亲是被神殿害死的。」

「他们有权利知道真相。」

「可这和我有什么关系？」

「因为你是他们的哥哥。」白渊说，「你比任何人都关心他们。」

「而我需要你的帮助。」

阿邪沉默了。

他想起了念念和安安。

想起了念念跪在烛龙棺椁前哭泣的样子。

想起了安安说「我要保护姐姐」时的眼神。

他们有权知道真相。

可——

「我只想平静地生活。」他说。

「我知道。」白渊说，「可有些事情，不是想躲就能躲的。」

「你身上有四维世界的印记。」

「你知道这意味着什么吗？」

「意味着什么？」

「意味着你会成为所有人关注的对象。」

「正道、魔道、天庭——」

「他们都会来找你。」

「都会想要利用你。」

「你不可能躲得掉的。」

阿邪沉默了。

白渊说的没错。

他不可能躲得掉。

他已经卷入了这一切。

从他决定帮助念念和安安的那一刻起，他就已经没有退路了。

「好。」他说，「我帮你。」

白渊笑了。

「我就知道你会答应。」

「别高兴太早。」阿邪说，「我只帮你查真相。」

「其他的，我不参与。」

「够了。」白渊说，「真相查出来之后的事，我会处理。」
