# 卷四：归途

## 第七十章：下一个目标

离开幽冥洞之后，阿邪和剑无痕商量下一步的计划。

「三元子给了你什么？」剑无痕问。

「记忆。」阿邪说，「'道'的记忆。」

「那些记忆里，有什么有用的信息吗？」

「有。」阿邪点头，「我知道了其他观测者碎片的位置。」

「除了二元子和三元子，还有四个碎片散落在各处。」

「四个？」

「是的。」阿邪说，「四元子，五元子，六元子，七元子。」

「他们分别在——」

他闭上眼睛，在记忆中搜索。

「七十二福地之一，'万灵福地'。」

「三十六洞天之一，'天璇洞'。」

「尘世与幽都的交界处，'裂隙深渊'。」

「还有——」

他停顿了一下。

「还有在哪里？」

「在念念和安安的心里。」

剑无痕愣住了。

「在他们的心里？」

「是的。」阿邪看向念念和安安，「三元子的记忆告诉我——」

「他们的血脉里，流淌着烛龙的力量。」

「而烛龙——是'道'最初的化身之一。」

「所以他们的心里，藏着观测者的碎片。」

念念和安安对视一眼。

「我们的心里？」安安摸了摸自己的胸口，「藏着什么？」

「藏着——另一个你。」阿邪说。

「另一个我？」

「是的。」

「另一个——我。」
