# 第十七章：赤水

离开归墟之后，阿邪直奔赤水。

赤水在归墟的西北方向，距离很远。

阿邪用了两天两夜的时间，日夜兼程，终于在第三天清晨抵达了赤水流域。

赤水是一条很大的河。

它的水面呈现出淡淡的红色，像是融化的琥珀，又像是稀释的血液。河的两岸长满了红色的树木，那些树木的叶子也是红色的，在风中沙沙作响，像是无数只小手在挥舞。

阿邪站在河边，看着那红色的水面，心里涌起一股复杂的情绪。

念念的父亲，就是赤水的河神。

也是烛龙的后裔。

三年前，墨渊吞噬了河神。念念和安安亲眼目睹了这一切。

现在，他要在这条河里，找到河神的龙珠。

他深吸一口气，纵身跃入水中。

---

赤水的水比普通的水要稠密得多。

那感觉像是在泥浆里游泳，每划动一下手臂都要费很大的力气。

可阿邪还是继续向下潜去。

他需要找到河底。

河底应该有一座宫殿——那是河神的住所。

根据古老的传说，水神都会在水底建造自己的宫殿，用珊瑚和贝壳装饰，用明珠照明。

阿邪继续下潜，水压越来越大，呼吸也越来越困难。

就在他快要撑不住的时候，他终于看见了——

一座宫殿。

那宫殿是用红色的珊瑚建成的，在水中散发着淡淡的光芒。宫殿的大门敞开着，里面一片漆黑，像是一张张开的巨口。

阿邪游到门前，犹豫了一下。

他不确定里面有没有危险。

可他知道，他没有别的选择。

他深吸一口气，游了进去。

---

宫殿里面比他想象的要大得多。

那走廊长得看不到尽头，两旁的墙壁上刻满了壁画。阿邪一边游一边看，看见那些壁画上画的是——

一条龙。

一条红色的龙，在水中蜿蜒盘旋，威严而神圣。

那应该就是烛龙了。

念念和安安的父亲。

阿邪继续向前走，走廊的尽头出现了一扇门。

门是开着的，里面透出淡淡的光芒。

他游过去，穿过门——

然后他愣住了。

门后是一个巨大的殿堂。

殿堂的正中央，有一个祭台。

祭台上放着一颗珠子。

那珠子大约有拳头大小，通体红色，表面流动着淡淡的光芒，像是一颗被凝固的火焰。

龙珠。

阿邪的心跳加速。

这就是他要找的东西。

他游上前，正要去拿——

「住手。」

一个声音从身后传来。

阿邪猛地转身。

一个身影站在殿堂的入口处。

那是一个女人。

她穿着一件红色的长裙，长发在水中飘散，面容清秀而苍白。她的眼睛是金色的，带着一种和人类完全不同的光芒。

她的下半身——

是一条鱼的尾巴。

阿邪的瞳孔骤然收缩。

这是一个鲛人。

「你是谁？」阿邪问。

「我是赤水河神的侍女。」鲛人冷冷地说，「我在这里守护龙珠，已经三百年了。」

「三百年？」

「自从主人死后。」鲛人的声音带着一丝哀伤，「我就在这里等。」

「等什么？」

「等一个能够拿走龙珠的人。」

阿邪愣了一下。

「你就是那个人吗？」

「我——」

「你身上有主人的气息。」鲛人打断他，「还有小主人和少主人的气息。」

「小主人？少主人？」

「念念和安安。」

阿邪的心猛地一跳。

「你认识他们？」

「我看着他们长大的。」鲛人的眼眶红了，「主人死后，夫人把他们送到人间，我本想跟着去保护他们，可封印太紧了，我出不去。」

「三百年了，我只能在这里等着。」

「等着他们有一天会回来。」

她看着阿邪，眼泪从眼角滑落，在水中化作一颗颗晶莹的珠子。

「你把他们带回来了吗？」

阿邪沉默了。

「……没有。」他说，「他们在一个安全的地方。」

「但是——」

「但是什么？」

「他们身上被墨渊的人下了印记。」阿邪说，「我需要龙珠来解除印记。」

鲛人沉默了。

她低下头，看着祭台上的龙珠。

「主人死的时候，留下了一句话。」她说，「他说，如果有一天，有人带着他的孩子的气息来找龙珠，就把龙珠给他。」

「他说，那个人会替他了结一切。」

阿邪愣住了。

他知道？

河神早就知道会有人来？

「主人是烛龙的后裔。」鲛人说，「烛龙能看见过去和未来。」

「主人早就看见了你。」

「看见了你带着他的孩子来这里。」

「看见了你为他的孩子做的一切。」

阿邪不知道该说什么。

原来，从一开始，他就被算计了。

可他不知道是该愤怒，还是该释然。

「拿走吧。」鲛人向后退了一步，「龙珠本来就是你的。」

「主人说过，只有你能救他们。」

「只有你能终结这一切。」

阿邪走到祭台前，伸手拿起龙珠。

珠子入手温热，像是一团被凝固的火焰。

他握紧珠子，转身看向鲛人。

「我会照顾好他们的。」他说，「我答应你。」

鲛人点点头，眼泪还在流。

「去吧。」她说，「完成主人的遗愿。」

「完成你自己的路。」
