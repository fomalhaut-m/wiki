# 卷四：归途

## 第六十六章：洞天

三十六洞天，各有各的特点。

阿邪他们要去的那个，叫做"幽冥洞"。

幽冥洞在尘世的西北边陲，藏在一座无名山脉的最深处。

传说这座山脉在天地初分时就存在了，比很多神山都要古老。

阿邪带着念念和安安，花了七天七夜，终于来到了山脚下。

「就是这里？」念念看着那座巍峨的山脉。

「是的。」剑无痕说，「幽冥洞就在山的内部。」

「入口在哪里？」

「需要找。」剑无痕说，「洞天的入口不是固定的。」

「它只对有缘人开放。」

阿邪打量着四周。

山脉很大，到处都是悬崖峭壁和密林。

要在这么大的范围里找一个入口，确实不容易。

「我来试试。」他说。

他闭上眼睛，释放出一丝四维世界的气息。

那是"道"的印记，也是他身份的证明。

如果洞天里真的有观测者的碎片——

那块碎片应该会感应到他。

---

果然。

一刻钟后，山脉的某处传来了一阵微弱的光芒。

「那边！」安安指着光芒的方向。

众人快步走过去。

在两棵古树的交界处，他们看到了一道裂缝。

裂缝很窄，只能容一人侧身通过。

「这就是入口。」剑无痕说。

阿邪深吸一口气。

「走吧。」
