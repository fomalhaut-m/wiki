# 卷三：维度

## 第五十四章：四维

阿邪的手触碰到门的那一刻，世界变了。

他感觉自己被什么东西拉扯着，拉向一个未知的地方。

念念和安安的手紧紧握着他的手。

他们一起被拉了进去。

穿过门的那一刻，阿邪看到了——

无限。

那是无限。

他看到了无数的画面，无数的可能性，无数个世界。

他看到了自己成为英雄的样子。

他看到了自己成为恶魔的样子。

他看到了念念和安安长大成人的样子。

他看到了他们死去的样子。

他看到了阿蘅姨还活着的世界。

他看到了那场火灾从未发生的世界。

他看到了——一切。

所有的可能性，同时存在，同时发生。

这就是四维世界。

没有时间的束缚。

没有因果的约束。

一切都是可能的。

「好美……」

念念的声音从他身旁传来。

阿邪看向念念。

她的眼睛里倒映着无限的世界。

她的脸上，写满了惊叹。

「这就是四维吗……」安安也看呆了。

阿邪握紧了他们的手。

「别看太久。」他说，「看太久会被吸进去。」

他拉着念念和安安，向前方走去。

他不知道他们要去哪里。

可他知道——

他必须走下去。
