# 第九章：维度之匙

维度之匙。

阿邪从未听过这个词。

「维度之匙是什么？」他问。

「简单来说，」老人把铜片推到阿邪面前，「这是一把钥匙。」

「通往第四维度的钥匙。」

阿邪愣住了。

第四维度。

他在阿蘅姨讲的故事里听过这个词。那是比天界还要高的地方，是「道」的本体所在之处。

据说，承劫修炼的终极目标，就是突破三维世界的束缚，进入第四维度，成为「道」本身。

「这……这不可能。」阿邪说，「第四维度只是传说，怎么可能真的存在——」

「传说？」老人笑了，「传说都是建立在真相之上的。只不过，随着时间的推移，真相被遗忘，只剩下了传说。」

他指了指铜片上的铭文。

「你看得懂这些字吗？」

「看不懂。」

「当然看不懂。」老人说，「这是第四维度的文字。三维世界的人，是看不懂四维文字的。」

「那你——」

「我能看懂，是因为我在四维世界里待过。」老人的眼神变得悠远，「很久很久以前，我曾经是四维世界的一员。」

阿邪彻底懵了。

四维世界的一员？

这个看起来普普通通的老人，曾经是四维世界的一员？

「你……你到底是什么人？」阿邪问，声音有些发颤。

「我说了，我是一元子。」老人说，「'一元'，指的是宇宙的起源。四维世界的道，就是'一'。而我，是从'一'里面生出来的东西。」

「或者说——」

「我是'一'的一个念头。」

阿邪听得云里雾里。

什么四维世界，什么道的念头，这些概念对他来说太过遥远，遥远得像是天方夜谭。

「你不需要现在就明白。」老人似乎看出了他的困惑，「有些事情，需要慢慢领悟。」

「我只能告诉你，这把钥匙选择你，是因为你身上有它需要的东西。」

「什么东西？」

「这个，你现在也不需要知道。」老人说，「你只需要知道，从今天起，你和四维世界之间，有了一条通道。」

「这条通道在哪里？」

「就在你心里。」

阿邪的心猛地一跳。

你终于来了。

他忽然想起了那个声音。那个在铜片中出现的火焰里的身影。

那声音说：「你终于来了。我等你很久了。」

原来，那不是幻觉。

原来，真的有什么东西在等他。

「我……我要做什么？」阿邪问，声音有些干涩。

「你什么都不用做。」老人说，「你只需要活着。好好活着。」

「等你准备好了，那扇门自然会为你打开。」

他从石凳上站起来，走到亭子边缘，背对着阿邪。

「好了，该说的我都说了。」

「现在，你该走了。」

「走了？」阿邪愣了一下，「去哪里？」

「去你想去的地方。」老人说，「归墟，还是幽都，都可以。」

「我会在这里等你。等你下一次来的时候，我还有很多事情要告诉你。」

他顿了顿。

「还有，那两个孩子。」

阿邪的身体僵了一下。

「你怎么知道——」

「我知道很多事情。」老人没有回头，「那两个孩子，是烛龙的后裔。他们身上流着上古神族的血。」

「他们的命运，和你紧紧绑在一起。」

「好好照顾他们。」

「他们会是你最重要的伙伴。」

阿邪张了张嘴，想说什么。

可还没等他开口，老人的身影就开始变淡，像是被什么东西逐渐擦去。

「等等——」

「下次见。」

老人的声音在空气中回荡，越来越远，越来越淡。

直到彻底消失。

阿邪独自站在亭子里，看着空无一人的石凳，脑子里一片空白。

过了很久，他才回过神来。

他低下头，看着手里的铜片。

铜片还是原来的样子，静静地躺在掌心，没有任何异常。

可他知道，一切都不一样了。

他深吸一口气，转身离开亭子，向裂缝的出口走去。

他要回到念念和安安身边。

他还有很多事情要做。
