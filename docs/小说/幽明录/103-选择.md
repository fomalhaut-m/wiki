# 卷五：大道（最终卷）

## 第一百零三章：选择

「那你为什么选择我？」阿邪问。

「因为你和之前所有的观测者都不一样。」「道"说，「他们都是被我创造的存在。」

「而你——」

「你是从三维世界的梦里走出来的。」

「你不属于我。」

「你是独立的。」

「你身上有我自己都没有的东西——」

「可能性。」

「不属于我的可能性。」

「这种可能性——让我感到——希望。」

阿邪沉默了。

他想起了自己一路走来的经历。

想起了从幽都逃出来的那个夜晚。

想起了在废墟中遇到念念和安安的时刻。

想起了他做过的每一个选择。

「我不是什么特殊的人。」他最终开口，「我只是——做了我认为对的事。」

「我知道。」「道"说，「正因为如此——你才特殊。」

「你没有宏大的目标。」

「你只是想保护你想保护的人。」

「你只是想——做一个好人。」

「这种纯粹的愿望——」

「是我从未见过的。」
