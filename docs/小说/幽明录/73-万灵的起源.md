# 卷四：归途（续）

## 第七十三章：万灵的起源

老树的声音在森林中回荡。

「很久很久以前——久到连我都记不清是什么时候——」

「这片土地上只有一棵神树。」

「那棵神树，是'道'种下的第一棵树。」

「神树的根连接着混沌，枝叶伸向四维。」

「它吸收混沌的力量，滋养这片土地。」

「慢慢地，这片土地上开始有了生命。」

「先是微生物，然后是植物，然后是动物，最后——是人。」

「所有这些生命，都带着神树的灵气。」

「所以这片土地上的万物，都有自己的灵性。」

「包括我。」

老树的声音变得悠远。

「我就是那棵神树的残骸。」

「当年'道'分裂的时候，神树也受到了波及。」

「它的一部分化作了无数的生命，遍布整个三维世界。」

「另一部分——就是我——则留在了这里。」

「成为了万灵福地的守护者。」

阿邪静静地听着。

「前辈想告诉我什么？」

老树沉默了一会儿。

「我想告诉你——」

「'道'并不是无敌的。」

「它的力量来自混沌，可它并不理解混沌。」

「它只是混沌的一部分，却妄想超越混沌。」

「这是它的局限性。」

「也是——它的弱点。」
