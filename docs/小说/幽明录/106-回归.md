# 卷五：大道（最终卷）

## 第一百零六章：回归

阿邪从四维世界回到了扶摇海。

念念和安安跟在他身后，脸上带着平静的微笑。

「哥哥，」念念问，「你见到'道'了吗？」

「见到了。」

「它说了什么？」

「它说——它理解我了。」阿邪说，「它不会再派人来阻止我们。」

「它给了我们选择的权利。」

安安的眼睛亮了。

「那我们自由了？」

「可以这么说。」阿邪笑了，「我们自由了。」

「我们可以——过自己想过的生活了。」

黎娘从屋子里走出来，看见他们回来，脸上露出了笑容。

「你们回来了。」

「回来了，黎娘。」阿邪说，「我们——回家了。」

黎娘的眼眶湿润了。

她等了这么久，终于等到了他们回来。

「好孩子。」她说，「回来就好。」

「回来就好。」
