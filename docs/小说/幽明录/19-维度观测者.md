# 第十九章：维度观测者

阿邪盯着那个身影，心跳加速。

维度观测者。

这是他从未听过的名词。

「维度观测者是什么？」他问。

「是第四维度的存在。」那身影说，「或者说，是第四维度在三维世界的投影。」

「四维不是空间吗？怎么会有——」

「四维不是空间。」那身影打断他，「四维是时间。」

「时间？」

「对。时间。」那身影说，「三维世界的人以为时间是线性的，从过去到现在到未来。可实际上，时间是四维的。」

「在四维世界里，过去、现在、未来同时存在。」

「就像一条河——你以为你站在河岸上，看着水流从上游流到下游。可实际上，整条河同时存在着。你只是没法同时看到所有的水。」

阿邪听不太懂。

这对他来说太过超前了。

「那我为什么会被卷入这一切？」他问，「一元子为什么选择我？」

「因为你特殊。」那身影说。

「特殊？」

「你有成为四维存在的潜质。」

「什么潜质？」

那身影沉默了。

然后他向阿邪走来，光芒从他身上剥落，露出真容。

那是一张——

阿邪愣住了。

那张脸。

那张脸和他一模一样。

「我是你的未来。」那身影说，「或者说，我是四维世界里的你。」

「四维世界里的我？」

「对。」那身影点头，「在这个三维世界里，你被困在时间里，只能从'现在'走向'未来'。可在四维世界里，你的所有可能性同时存在。」

「你可能成为英雄，也可能成为恶魔。可能幸福，也可能悲惨。」

「所有的可能性都在那里，同时发生。」

「而我——」

他指了指自己。

「我是所有可能性的总和。」

「是四维世界里完整的你。」

阿邪的脑子要炸了。

四维、时间、可能性、完整的自己——

这些概念像是一团乱麻，在他脑子里纠缠不清。

「我现在不需要做决定。」那身影说，「先完成眼前的事情。」

「无赦让你来镜殿找的东西，我知道在哪里。」

「你知道？」

「我是这里的主人。」那身影说，「镜殿里的一切，我都了如指掌。」

「他要找的是——」

他抬起手，一道光芒射出，落在地面上，形成了一幅图案。

那图案是一个圆环，环环相扣，像是一条咬住自己尾巴的蛇。

「这是'循环之钥'。」那身影说，「是打开维度回廊的钥匙。」

「维度回廊？」

「是连接三维世界和四维世界的通道。」那身影说，「有了它，你就不需要在四维世界里冒险。」

「你可以通过回廊，直接看到四维世界的景象。」

「不需要进去？」

「不需要。」那身影点头，「进去有风险，但用回廊观测，是安全的。」

「你就能够得到你需要的信息。」

阿邪的心动了。

这正是他需要的。

「那东西在哪里？」

「就在镜殿的最深处。」那身影说，「沿着这条路走，你会看到一扇门。」

「门后面就是循环之钥。」

「拿走吧。」

阿邪看着那身影。

「你为什么要帮我？」

「因为你是另一个我。」那身影笑了，「我帮你，就是帮我自己。」

他的身影开始变淡。

「记住——」

「有些事情，只有你自己能决定。」

「不要相信任何人给你的答案。」

「包括我。」

「找到属于你自己的路。」

他的身影彻底消失了。

只留下阿邪一个人，站在镜殿中央。

---

阿邪沿着那身影指的方向走去。

走了大约一盏茶的功夫，他看见了一扇门。

门是银白色的，和周围的墙壁一样，表面刻满了密密麻麻的符文。

他在门前停下，深吸一口气，然后推开了门。

门后是一个小小的房间。

房间的正中央，有一个水晶台。

水晶台上放着一个环形的物件。

那环形物件大约有巴掌大小，像是一个圆环被拉长后又扭曲成了特殊的形状。它的表面流动着淡淡的光芒，像是某种液体在内部缓缓流动。

循环之钥。

他找到了。

他把它收入怀中，转身离开。
