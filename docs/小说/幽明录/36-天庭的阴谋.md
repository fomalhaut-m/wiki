# 卷二：真相

## 第三十六章：天庭的阴谋

鲛人的声音在贝壳房里回荡。

念念和安安屏住呼吸，一动不动地听着。

「天庭为什么要杀父亲？」念念问。

「因为主人违抗了命令。」鲛人说，「天庭的规矩很严格。神族不得与凡人结合。」

「这是天道法则，任何人都不能违背。」

「父亲违背了，所以要死？」

「是的。」鲛人点头，「但天庭不能亲自动手。他们担心会落下话柄。」

「于是他们把任务交给了白木神殿。」

「神殿不是正道的吗？」安安忍不住问。

「正道？」鲛人冷笑，「这世上哪有什么正邪之分。」

「神殿需要天庭的支持来维持地位，天庭需要神殿来处理那些见不得光的事情。」

「他们互相利用，各取所需。」

「所以神殿就答应了？」

「神殿答应了。」鲛人的声音变得冰冷，「他们派出了一队精英，去刺杀主人。」

「可主人是烛龙的后裔，实力强大。」

「刺杀失败了。」

「神殿的人死伤惨重。」

「于是他们换了策略。」

「什么策略？」

鲛人沉默了。

过了很久，她才开口。

「他们找到了墨渊。」
