# 卷四：归途（终）

## 第八十四章：念念与安安的成长

接下来的日子，念念和安安在阿邪的指导下学习掌控自己的力量。

念念学会了操控水——从最基础的吸水、喷水，到后来能够凝聚水盾、召唤水龙。

安安学会了操控火——从最基础的点火、控火，到后来能够凝聚火墙、召唤火凤。

他们的进步速度超出了阿邪的预期。

「不愧是烛龙的血脉。」剑无痕感叹，「这种觉醒速度，放在整个修仙界都是罕见的。」

「可他们的力量还不稳定。」阿邪说，「需要时间磨练。」

「你打算怎么办？」

「继续训练。」阿邪说，「等他们的力量稳定了，我们再去面对'道'。」
