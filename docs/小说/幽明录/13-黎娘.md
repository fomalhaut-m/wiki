# 第十三章：黎娘

黎娘的铺子在尘秽市的东街尽头。

那是一条狭窄的小巷，巷子两旁堆满了各种药材和奇怪的器具。空气里弥漫着一股混杂的药味，有的香，有的臭，混合在一起，形成一种说不出的诡异气息。

阿邪走进巷子深处，看见了一扇半掩的木门。

门上没有匾额，只有一种奇特的草药味道从里面飘出来。

他敲了敲门。

「进来。」

一个女人的声音从里面传出，沙哑而慵懒。

阿邪推门进去。

里面是一个宽敞的作坊，四周摆满了各种瓶瓶罐罐，有的冒着泡，有的散发着淡淡的光芒。正对着门的工作台后面，坐着一个女人。

那女人穿着一件黑色的长袍，头发用一根木簪随意挽着，脸上蒙着一层薄纱，只露出一双眼睛。

那是一双极其明亮的眼睛，像两颗浸在月光里的黑曜石，带着一种慵懒而危险的光芒。

「哟，这不是幽都的小红人吗？」黎娘开口，声音里带着一丝调侃，「什么风把你吹来了？」

「有事相求。」阿邪说。

「求？」黎娘笑了，「我可是个生意人，不做亏本买卖。」

「你想要什么？」

「先说来听听。」

阿邪深吸一口气。

「我想请你帮我照顾两个孩子。三个月。」

「两个孩子？」黎娘挑眉，「就是那两个烛龙的后裔？」

阿邪的心一沉。

她已经知道了。

「消息倒是灵通。」他说。

「我在尘秽市开铺子，消息不灵通，早就死了八百回了。」黎娘说，「说吧，为什么要我照顾他们？」

「因为我没有别的选择。」阿邪说，「墨渊的人在监视他们，我不能带着他们到处跑。」

「所以你找到了我？」

「是。」

「为什么是我？」

「因为你不在乎墨渊。」阿邪说，「你不属于任何势力，只关心自己的利益。」

「你觉得墨渊会因为两个崽子，来找我麻烦？」

「短时间内不会。」阿邪说，「你在尘秽市有根基，墨渊不会轻易动你。」

「而且——」他顿了顿，「我可以帮你做一件事。」

「什么事？」

「九幽露。」

黎娘的眼睛亮了一下。

九幽露，是一种极其罕见的药材，据说只生长在深渊最深处。这种药材是炼制「续命丹」的关键材料，而续命丹能够延长修士的寿元。

「你能弄到九幽露？」黎娘问。

「我不能。」阿邪说，「但我可以帮你找到线索。」

「我要去归墟，那里有通往深渊的裂缝。如果有机会，我会帮你留意。」

黎娘沉默了一会儿。

然后她笑了。

「有意思。」她说，「成交。」

「不过——」

她的眼神变得认真起来。

「丑话说在前头，我只负责照顾他们三个月。三个月后，你自己来接人。」

「如果三个月内出了任何问题，我不负责。」

「没问题。」

「还有——」

黎娘从桌上拿起一个小瓶子，递给阿邪。

「这是我炼的'匿息丹'，能够暂时掩盖气息。你那两个崽子身上的印记，服用这个之后，可以压制一段时间。」

「不是解除？」

「如果是墨渊亲自下的印记，我也解不了。」黎娘说，「这个只能暂时压制。」

「能压多久？」

「看情况。」黎娘说，「正常情况下，三到五天。」

「如果要激烈活动，可能只有一两天。」

阿邪接过瓶子，收入怀中。

「谢了。」

「谢？」黎娘笑了，「这只是交易。」

「快去吧，三个月可不长。」
