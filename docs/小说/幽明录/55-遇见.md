# 卷三：维度

## 第五十五章：遇见

在四维世界里走了不知道多久——也许是几秒钟，也许是几年——阿邪终于看到了一个身影。

那身影站在无限的可能性之中，像是一块礁石屹立在汪洋大海中。

「你终于来了。」

那身影转过来。

阿邪愣住了。

那张脸——

和他一模一样。

「我们见过。」那身影笑了，「在镜殿里。」

「你是——维度观测者？」

「那只是我的名字之一。」那身影说，「你可以叫我'观测者'。」

「或者——叫我另一个你。」

念念和安安躲在他身后，警惕地看着那身影。

「他是谁？」念念小声问。

「是四维世界里的我。」阿邪说。

「另一个哥哥？」安安瞪大了眼睛。

「可以这么说。」

观测者笑了。

「你带来了你的同伴。」他说，「很好。」

「你带他们来，是正确的选择。」

「因为——」

他的眼神变得认真起来。

「你要找的答案，就在这里。」

「就在四维世界里。」
