# 卷五：大道（最终卷）

## 第九十七章：再入四维

阿邪带着念念和安安，再次来到了归墟。

建木还矗立在那里，枝叶繁茂，仿佛从未改变。

可阿邪知道——他已经不是三年前的那个他了。

「守树人。」他开口，「我需要再去四维世界。」

建木的深处传来一阵沙沙声。

然后，守树人的声音响起：

「我知道你会来。」

「等你很久了。」

阿邪走进建木，顺着根系向下走。

念念和安安紧紧跟在他身后。

走了不知道多久——

他们再次看到了那扇门。

那扇通往来世的门。

「你们准备好了吗？」阿邪问。

「准备好了。」念念和安安异口同声。

阿邪伸出手，推开了门——

光芒涌入。

世界变了。

他们来到了四维世界。

在这里，时间没有意义。

空间没有界限。

一切的可能性，同时存在。

阿邪看向远方。

他知道"道"在那里等着他。

「走吧。」他说，「去见'道'。」

「去找到——真正的答案。」
