# 卷三：维度

## 第五十二章：维度回廊

去四维世界不是一件简单的事。

阿邪需要先回到归墟，找到守树人。

守树人住在建木下，已经等了他很久。

「你来了。」守树人说。

「我来了。」阿邪说，「我收到了那封信。」

「我知道。」守树人点头，「那是我写的。」

阿邪愣了一下。

「你写的？」

「是的。」守树人说，「我一直在等你做出这个决定。」

「为什么要等我？」

「因为这是你的旅程。」守树人说，「不是我能替你走的。」

「我只能给你指引，给你方向。」

「可路——要你自己走。」

阿邪沉默了。

他想起了很多事。

想起了三年前他第一次见到守树人的情景。

想起了他得到建木残根的时刻。

想起了"道"在他身上留下的印记。

「维度回廊在哪里？」他问。

「就在这里。」守树人指向身后的建木，「就在建木的最深处。」

「建木的最深处？」

「是的。」守树人点头，「建木是连接三维和四维的桥梁。」

「它的根系延伸到四维世界。」

「你只需要顺着它的根系走，就能到达维度回廊。」

「然后呢？」

「然后——」守树人停顿了一下，「你会看到一扇门。」

「那扇门，是你自己的门。」

「只有你能打开它。」
