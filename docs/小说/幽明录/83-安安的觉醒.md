# 卷四：归途（终）

## 第八十三章：安安的觉醒

看着姐姐的能力觉醒，安安也迫不及待了。

「哥哥，我的呢？」

阿邪笑着把火之魂交给他。

「别着急。」

「来，像姐姐一样，握住它。」

安安接过那颗紫色的晶石。

晶石和安安的手掌差不多大，他两只手才能握住。

「闭上眼睛。」阿邪说，「感受它。」

安安照做了。

紫色的晶石开始发热，越来越热——

然后，它融入了安安的体内。

一道紫色的光芒从安安身上爆发。

和念念的蓝色不同——

这道光芒是灼热的，狂暴的，带着毁灭一切的气势。

「小心！」剑无痕喊道。

阿邪一挥手，在安安周围布下了一道防护。

火焰没有爆发出来，而是被控制住了。

它只存在于安安的体内，燃烧着，咆哮着——

然后渐渐平息。

安安睁开眼睛。

他的眼睛也变了——变成了深邃的紫色，像两颗燃烧的星辰。

「我感觉——」他低头看着自己的手，「我好热。」

「像是有什么东西在我体内燃烧。」

「那是火之魂的力量。」阿邪说，「和姐姐一样，你觉醒了。」

「烛龙的火系力量，在你体内苏醒了。」

安安握紧拳头。

他感觉到了——

前所未有的力量。
