# 卷五：大道（最终卷）

## 第一百一十章：道

阿邪做出了他的决定。

他不会走。

他会留在这个世界。

和念念、安安一起。

和所有他认识的人一起。

面对——即将到来的毁灭。

「你疯了吗？」七元子问，「你知道留下意味着什么吗？」

「知道。」阿邪说，「意味着——可能会死。」

「可那又怎样？」

「我已经活过了很多年。」

「我有了念念和安安。」

「有了黎娘，有了剑无痕，有了所有那些——我想保护的人。」

「这辈子——已经值了。」

七元子看着他，眼神复杂。

「你真的——不怕死吗？」

「不是不怕。」阿邪说，「是——」

「比起死——我更怕失去他们。」

「失去——我选择的道。」

他看向远方。

天空还是那么蓝。

海水还是那么清。

这个世界——还是那么美。

「这就是我的道。」他说。

「不是毁灭。」

「不是逃避。」

「是——守护。」

「守护我想守护的一切。」

「这就是——大道无形。」
