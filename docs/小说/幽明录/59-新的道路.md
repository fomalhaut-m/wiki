# 卷三：维度

## 第五十九章：新的道路

阿邪做了一个决定。

不是观测者提出的任何一个选择。

而是——一个新的选择。

「我选择继续做梦。」他说。

观测者微微皱眉。

「可同时——」他继续说，「我会找到另一条路。」

「一条不需要任何人消失的路。」

「一条所有人都能好好活下去的路。」

「你确定？」观测者问，「这条路没有人走过。」

「我知道。」阿邪说，「可正因为没有人走过——」

「才更有价值去探索。」

他看向念念和安安。

「我答应过他们，要保护他们。」

「我会做到的。」

「用我自己的方式。」

观测者沉默了很久。

然后，他笑了。

那笑容里带着惊讶，带着欣赏，带着——希望。

「果然没看错你。」

「你比我想象的更特别。」

「不是因为你有多强大。」

「是因为——」

「你永远不放弃。」

「永远相信还有另一条路。」

他向后退了一步，身影开始变淡。

「去吧。」他说，「回到你的世界。」

「去走你的路。」

「我会在这里看着你。」

「如果有一天——」

「你需要帮助——」

「我会尽我所能。」

阿邪看着他。

「谢谢你。」

「谢我什么？」

「谢谢你告诉我这些。」

观测者笑了。

「谢你自己吧。」他说，「是你自己选择相信的。」

「不是我。」
