# 卷五：大道（最终卷）

## 第一百零七章：平静的日子

从那以后，阿邪和念念、安安在扶摇海过上了平静的日子。

他们日出而作，日落而息。

念念学会了做饭，安安学会了打渔。

阿邪有时候会去山里采药，有时候会在海边发呆。

黎娘住在隔壁，每天都会过来串门。

剑无痕偶尔也会来看他们，给他们讲外面的世界发生了什么。

日子一天天过去。

平淡，却充实。

「哥哥。」念念问，「你幸福吗？」

「幸福。」阿邪说，「非常幸福。」

「为什么？」

「因为——」阿邪看向远方，「我有了想保护的人。」

「有了——属于自己的家。」

「这就够了。」
