# 卷四：归途（终）

## 第九十一章：毁灭者

毁灭者站在天际，俯视着阿邪他们。

它的身影巨大，遮蔽了半边天空。

它身上散发出的气息，让整个扶摇海都在颤抖。

「你们就是'道'选中的人？」毁灭者的声音如雷霆，「不过是一群蝼蚁。」

「蝼蚁？」阿邪冷笑，「我们曾经击败过梦魇。」

「梦魇？」毁灭者笑了，「那不过是一个小角色。」

「而我——」

「是'道'最强的使者。」

「是毁灭的化身。」

它抬起手，一道黑色的光球在掌心凝聚。

「让我——送你们上路。」

阿邪的眼神变得凝重。

他知道毁灭者不是梦魇能比的。

这是真正的威胁。

「念念，安安——」他说，「准备好了吗？」

「准备好了！」念念和安安异口同声。

阿邪深吸一口气。

「那就——」

「一起上！」
