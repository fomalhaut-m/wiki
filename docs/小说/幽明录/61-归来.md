# 卷四：归途

## 第六十一章：归来

回到扶摇海的时候，已经是傍晚。

夕阳的余晖洒在海面上，金光粼粼。

阿邪带着念念和安安，沿着海边的小路向山上走去。

远处的小房子还亮着灯。

「哥哥姐姐回来了！」

黎娘站在院子里挥手。

「你们可算回来了！」

她快步走过来，上下打量着念念和安安。

「没事吧？有没有受伤？」

「没有没有，」念念笑着说，「我们都很好。」

「黎娘婆婆，你怎么会在这里？」安安好奇地问。

「我来照顾房子啊。」黎娘说，「你们一走就是好几天，我担心房子没人管，就搬过来住了。」

「那婆婆你的药铺呢？」

「关了。」黎娘说，「我那些存货早就卖完了，也懒得再进货了。」

「反正我也用不上什么钱了。」

阿邪看着她，心里有些感动。

「谢谢你，黎娘。」

「谢什么？」黎娘摆摆手，「我也闲着没事，顺便而已。」

她的目光落在阿邪身上，眼神里带着一丝探究。

「怎么样？有收获吗？」

阿邪想了想。

「有。」他说，「很大的收获。」

「什么收获？」

「收获了很多答案。」阿邪说，「也收获了很多问题。」

黎娘挑了挑眉。

「听起来很深奥。」

「不深奥。」阿邪笑了，「只是——」

「只是需要时间慢慢想。」
