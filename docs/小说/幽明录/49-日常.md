# 卷二：真相

## 第四十九章：日常

某一天。

阿邪坐在海边，看着念念和安安在沙滩上玩耍。

念念在捡贝壳，安安在追螃蟹。

他们的笑声，随着海风飘过来。

阿邪的嘴角微微扬起。

这就是他想要的生活。

平平淡淡。

简简单单。

没有什么惊天动地的大事。

只有和喜欢的人们，待在一起。

「在想什么？」

黎娘不知道什么时候走了过来。

「在想，这样的日子很好。」

「是啊。」黎娘在他旁边坐下，「这样的日子很好。」

「可不会持续太久的。」

「为什么？」

「因为——」黎娘看向海面，「因为你是特殊的存在。」

「而特殊的存在，永远不会得到安宁。」

「他们会被各种各样的人盯上。」

「正道，魔道，天庭——」

「所有人都会来找你。」

「你怎么可能躲得掉呢？」

阿邪沉默了。

他知道黎娘说的是对的。

他已经不是三年前那个在幽都里浑浑噩噩度日的人了。

他现在有太多的印记——

四维世界的印记。

"道"的印记。

这些印记，会给他带来无穷无尽的麻烦。

可——

他看向念念和安安。

也许——

这就是他选择的代价吧。
