# 卷二：真相

## 第三十八章：阿邪的决定

阿邪站在贝壳房外，听完了鲛人的讲述。

他的拳头紧紧攥着。

神殿。

原来从头到尾，都是神殿在幕后操纵一切。

念念和安安父母的死，墨渊的崛起，白木神殿和墨渊之间的恩怨——

所有的一切，都是神殿的阴谋。

他转身离开。

他需要去见白渊。

---

白渊住在扶摇海的客栈里。

阿邪找到他的时候，他正在喝茶。

「你来得比我想象的快。」白渊说。

「我有事要问你。」

「问吧。」

「神殿和墨渊的契约，你知道多少？」

白渊的手顿了一下。

「你查到了。」

「我查到了。」阿邪说，「我想知道更多。」

「关于什么？」

「关于——」阿邪的眼神变得锐利，「你师父的日记。」

白渊沉默了。

「你师父知道这件事，对吗？」

「他知道。」白渊说，「他一直知道。」

「那他为什么不公开？」

「因为——」白渊苦笑，「因为神殿是他的家。」

「他不想背叛自己的家。」

「哪怕那个家是肮脏的？」

「哪怕那个家是肮脏的。」白渊点头，「这就是神殿的人。」

「他们嘴上说着正义，心里却藏着无数的秘密。」

「我师父是一个典型。」

阿邪沉默了。

「所以你想揭露这一切。」

「是的。」白渊说，「神殿需要一次彻底的清洗。」

「把那些腐烂的东西，全部挖出来。」

「烧掉。」

「你愿意帮我吗？」
