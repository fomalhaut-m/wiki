# 卷四：归途

## 第六十八章：观测者的遗志

阿邪看着那个老人。

他知道，这个老人不是普通的修士。

他是"观测者"。

是"道"分裂出来的意识碎片之一。

「你是——」

「我是三元子。」老人说，「'道'的第三个碎片。」

「在我之前，有一个一元子和二元子。」

「一元子已经在很久以前消散了。」

「二元子——我猜你已经见过了。」

「二元子？」阿邪想了想，「你是说——那个在镜殿里的？」

「是的。」三元子点头，「那是我二哥的意识碎片。」

「他比我更早找到继承者。」

「他选择了你。」

阿邪沉默了。

他想起了镜殿里那个自称"维度观测者"的存在。

那个给了他维度之匙的人。

那个告诉他四维世界真相的人。

「那你是——」

「我是来给你最后一份礼物的。」三元子说。

「什么礼物？」

三元子抬起手。

一道光芒从他指尖射出，没入阿邪的眉心。

阿邪感觉自己体内多了什么东西。

那不是力量，不是知识——

而是——记忆。

无数的画面在他脑海中闪过。

他看到了"道"诞生之初的样子。

他看到了混沌的起源。

他看到了三维世界的创造。

他看到了——

一切的开始。

「这是我的记忆。」三元子的声音在他脑海中回响，「是'道'的记忆。」

「我把它交给你。」

「希望你能用它找到——」

「那条属于你自己的路。」

光芒渐渐消散。

阿邪睁开眼睛。

三元子的身体已经开始崩解了。

他的皮肤化作了灰尘，他的骨头化作了沙砾。

「谢谢你——」阿邪说。

「不必谢我。」三元子的声音越来越弱，「要谢就谢你自己。」

「是——你的选择——给了我们希望。」

「我们这些碎片——等了很久——」

「等一个能走出新路的人。」

「你——就是那个人。」

「不要——让我们失望——」

话音落下。

三元子的身体彻底崩解，化作了一堆尘土。

石室里只剩下阿邪，和他脑海中新添的那些记忆。
