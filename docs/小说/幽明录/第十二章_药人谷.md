# 第十二章：药人谷

黑雾彻底散去后，阿邪站在原地，半天没有说话。

念念怯生生地拉了拉他的袖子。

「哥哥……」

阿邪回过神，低头看着她。

她的小脸上写满了恐惧，可她还是强撑着问：「那个人……在我们身上留下了什么？」

「一个印记。」阿邪说，「用来追踪你们的位置。」

「能去掉吗？」

阿邪沉默了。

他不知道。

以他的能力，恐怕很难解除墨渊手下的秘术。

可他没有把这件事告诉念念。

「我来想办法。」他说，「先找个地方休息。」

他拉着念念和安安，继续向客栈走去。

一路上，他的脑子在飞速运转。

三个月。

三件任务。

建木残根，龙珠，镜殿。

每一样都够他忙活一辈子了。

可他只有三个月。

而且，他还要带着念念和安安。

不能把他们丢下不管。

不能让他们再陷入危险。

他需要一个安全的地方，把他们暂时安置下来。

他想了一圈，最后想到了一个人。

黎娘。

药人谷的黎娘。

他们在来尘秽市的路上，曾经遇到过她。那时候她正在尘秽市收购药材，听说阿邪是从幽都来的，还主动和他搭话。

她是个亦正亦邪的人物，不属于任何势力，只关心自己的研究和利益。

如果能说服她暂时收留念念和安安——

不，太冒险了。

黎娘不是善人。她帮阿邪，一定会有条件。

可他现在没有别的选择。

他叹了口气，站起身来。

「念念。」

念念抬起头。

「我要出去一趟。」他说，「你们在这里等我。」

「去找谁？」

「一个……可能帮我们的人。」
