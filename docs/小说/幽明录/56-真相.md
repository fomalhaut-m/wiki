# 卷三：维度

## 第五十六章：真相

阿邪跟着观测者走了很久。

或者说，走了很多个可能性。

在四维世界里，方向是没有意义的。

唯一有意义的，是"选择"。

每一个选择，都会导向一个新的可能性。

观测者带着他们，穿过了无数个可能性，最终——

他们来到了一个地方。

那是一个空旷的空间。

什么都没有。

只有——

一颗球。

一颗悬浮在半空中的球。

球是透明的，里面流动着淡淡的光芒。

「这是什么？」阿邪问。

「这是'混沌'。」观测者说。

阿邪的心猛地一跳。

「混沌？」

「是的。」观测者点头，「不是噬的混沌——是真正的混沌。」

「是一切可能性的总和。」

「是四维世界的本源。」

阿邪看向那颗球。

球里的光芒在流动，像是一条永不停歇的河流。

「你带我来这里，是要告诉我什么？」

「告诉你这个世界的真相。」观测者说。

「什么真相？」

观测者沉默了。

过了很久，他才开口。

「你知道'道'是怎么来的吗？」

「不知道。」

「'道'——」观测者说，「'道'就是混沌的一部分。」

「在很久很久以前，混沌中诞生了一个意识。」

「那个意识，想要理解自己。」

「于是它开始观察。」

「观察了不知道多少年。」

「它观察了所有的可能性，所有的世界，所有的存在。」

「然后——它疯了。」

阿邪的瞳孔收缩。

「疯了？」

「是的。」观测者说，「它疯得太彻底，连自己是什么都忘了。」

「它忘记了自己是混沌的一部分。」

「它开始以为，自己是超越混沌的存在。」

「是创造了混沌的上帝。」

「是'道'。」

「于是——它开始按照自己的想象，重新塑造世界。」

「这就是三维世界的由来。」

阿邪感觉自己的脑子要爆炸了。

「你是说——」

「我是说，」观测者打断他，「三维世界是'道'的一个梦。」

「一个疯子的梦。」

「这个梦里，一切都有秩序，一切都有因果。」

「可梦终究是梦。」

「总有一天，'道'会醒来。」

「到时候——」

他的声音变得沉重。

「三维世界会消失。」

「所有的一切，都会回归混沌。」
