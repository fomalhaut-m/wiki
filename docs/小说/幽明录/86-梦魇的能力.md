# 卷四：归途（终）

## 第八十六章：梦魇的能力

阿邪在梦中和梦魇对峙着。

这是他第一次和"道"的使者正面交锋。

「你的能力是操控梦境？」阿邪问。

「是的。」梦魇说，「我可以进入任何人的梦境。」

「在他们最脆弱的时候——」

「摧毁他们。」

阿邪看着梦魇。

他知道梦魇说的不是虚言。

梦境是人心最脆弱的地方。

在那里，所有的恐惧都会被放大，所有的防线都会崩塌。

普通人根本无力抵抗梦魇的入侵。

可他不是普通人。

他有"道"的印记。

他见过四维世界的真相。

「你觉得你能打败我？」阿邪问。

梦魇笑了。

那笑容里带着嘲讽。

「打败你？」

「不需要打败。」

「我只需要——抹除你。」

「在梦里抹除你。」

「然后——」

「在现实中，你也会消失。」

「就像——从未存在过一样。」
