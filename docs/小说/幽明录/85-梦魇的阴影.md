# 卷四：归途（终）

## 第八十五章：梦魇的阴影

就在念念和安安训练的时候，危险也在逼近。

那天夜里，阿邪做了一个梦。

梦里，他回到了幽都。

幽都还是那么黑暗，那么阴森。

他走在弱水河畔，看着那些熟悉又陌生的景色。

然后他听到了脚步声。

「阿邪。」

一个声音从他身后传来。

阿邪猛地转身。

一个身穿黑袍的人站在他面前。

那人的脸被阴影遮住，看不清长相。

「你是谁？」阿邪警惕地问。

「我是梦魇。」那人说，「'道'派来的使者。」

「'道'知道你选择了自己的路。」

「它不能接受。」

「所以——」

「它派我来，抹除你。」

阿邪的瞳孔收缩。

「抹除我？」

「是的。」梦魇说，「抹除你的存在。」

「让你——从未出现过。」
