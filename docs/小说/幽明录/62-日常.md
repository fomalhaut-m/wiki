# 卷四：归途

## 第六十二章：日常

那天晚上，大家围坐在院子里吃晚饭。

饭是念念做的——鱼汤泡饭，简单却温暖。

饭桌上，大家有说有笑。

「对了，」黎娘突然想起什么，「你们走的那几天，村里来了个陌生人。」

「陌生人？」阿邪放下筷子，「什么人？」

「不知道。」黎娘说，「是个年轻人，穿得很奇怪，不像是本地人。」

「他在村里转了几天，然后就走了。」

「走之前还问我们，认不认识一个叫阿邪的人。」

阿邪的眉头皱了起来。

「他问我的名字？」

「是的。」黎娘说，「我当时没理他，就说不知道。」

「他长什么样子？」

「二十来岁，挺俊的。」黎娘想了想，「哦对了，他腰上挂着一把剑。」

「剑？」

「嗯。那剑很特别，剑鞘是黑色的，上面刻着一些奇怪的纹路。」

阿邪和白渊对视一眼。

「是神殿的人吗？」

「不像。」白渊摇头，「神殿的剑修我认识，没见过这个人。」

「那会是谁？」

阿邪沉默了。

他有一种直觉——那个陌生人，是冲着他来的。
