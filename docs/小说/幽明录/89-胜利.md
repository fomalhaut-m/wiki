# 卷四：归途（终）

## 第八十九章：胜利

梦魇被消灭后，阿邪从梦中醒来。

窗外，天已经亮了。

念念和安安也醒了。

「我们赢了？」安安揉着眼睛。

「赢了。」阿邪说，「暂时赢了。」

他看向窗外。

天边泛起了鱼肚白，新的一天开始了。

可他知道——这只是暂时的平静。

"道"不会就此罢休。

它会派出更多的使者。

更多的威胁。

更多的危险。

可他不怕。

因为他不再是一个人了。

他有了念念和安安。

有了剑无痕。

有了所有那些愿意和他并肩作战的人。

「接下来——」他说，「我们要变得更强。」

「强到能够保护所有人。」

「强到能够——」

他看向远方。

「强到能够改变这个世界的命运。」
