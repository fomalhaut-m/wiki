# 卷三：维度

## 第五十一章：信

阿邪把那封信给念念和安安看了。

「这是什么意思？」念念问。

「我也不确定。」阿邪说，「但这封信来自四维世界。」

「四维世界？」安安歪着头，「那是什么地方？」

「是很远很远的地方。」阿邪解释道，「远到我们这辈子都不一定能到的地方。」

「比天庭还远吗？」

「比天庭还远。」

念念沉默了一会儿。

「哥哥，你打算去吗？」

阿邪看着她。

「你觉得呢？」

念念想了想。

「我觉得——」

「如果哥哥想去，我们就跟着去。」

「不管去哪里，我们都要在一起。」

阿邪的眼眶微微热了。

「你们不怕吗？」

「怕。」念念老实地说，「可更怕和哥哥分开。」

「我和姐姐一样的想法。」安安举手。

阿邪看着他们，心里涌起一股暖流。

他曾经以为，自己注定要孤独一辈子。

可现在，他有了念念和安安。

他有了想要保护的人。

这就够了。

「好。」他说，「那我们就去。」
