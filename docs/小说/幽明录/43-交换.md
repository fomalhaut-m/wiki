# 卷二：真相

## 第四十三章：交换

阿邪和白渊来到议事堂外的一个僻静角落。

「你要做什么？」白渊皱眉，「我正要揭露真相——」

「你揭露不了的。」阿邪打断他。

「为什么？」

「因为你说的那些长老里，至少有三个是知情的。」

白渊的脸色变了。

「你确定？」

「确定。」阿邪说，「我在墨渊身边待了三年。他的情报网不是摆设。」

「他知道神殿内部的很多事。」

「包括当年的契约细节。」

「你的意思是——」

「我的意思是，你师父的日记，扳不倒神殿。」

白渊沉默了。

「那你说怎么办？」

「我有另一个办法。」阿邪说，「但需要你的配合。」

「什么办法？」

「让'道'介入。」

白渊愣住了。

「道？」

「是的。」阿邪说，「我见过'道'。它给了我力量，也给了我一些信息。」

「它告诉我，这个世界需要一个真正的审判。」

「不是神殿内部的审判。」

「是——天道的审判。」

白渊的瞳孔收缩。

「你疯了。」

「也许吧。」阿邪说，「可你有更好的办法吗？」

白渊沉默了。

良久。

「说说你的计划。」
