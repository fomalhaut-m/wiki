# 卷三：维度

## 第六十章：归来

阿邪带着念念和安安，走出了四维世界。

当他们重新站在建木里的时候，外面已经过去了一整天。

可对他们来说，感觉像是过了很久很久。

他们经历了太多。

看到了太多。

也——想明白了太多。

「哥哥。」念念开口。

「嗯？」

「我们接下来做什么？」

阿邪想了想。

「回家。」他说。

「回扶摇海。」

「继续我们以前的生活。」

「可是——」安安疑惑，「那些事情——混沌什么的——」

「我们以后再想。」阿邪说，「现在，我只想回家。」

他拉着念念和安安的手，走出了建木。

外面阳光正好。

建木的枝叶在风中摇曳，发出沙沙的声响。

远处，有人在唱歌。

是渔民在出海。

阿邪看着这一切，心里充满了平静。

这就是他的选择。

不为拯救世界。

不为成为英雄。

只是为了——

和喜欢的人们，待在一起。

过平常的日子。

这就是他的道。

他的——大道无形。

---

*（卷三：维度，完）*
