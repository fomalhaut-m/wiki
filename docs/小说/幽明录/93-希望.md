# 卷四：归途（终）

## 第九十三章：希望

阿邪闭上眼睛。

他感受到了——

来自四面八方的力量。

那是剑无痕的力量。

那是观测者的力量。

那是三元子、四元子、五元子的力量。

那是——所有相信他的人的力量。

他们没有消失。

他们在他体内。

在他的信念里。

在他选择的道路中。

「我明白了。」阿邪睁开眼睛。

他的眼睛里闪烁着光芒。

那是——希望的光芒。

「毁灭者。」他说，「你以为你代表毁灭。」

「可你不知道——」

「希望，比毁灭更强大。」

他向前迈出一步。

一步——

他的身上爆发出了耀眼的光芒。

那光芒不是四维世界的力量。

也不是"道"的力量。

而是——人类的力量。

是信念的力量。

是爱的力量。

是——希望的力量。

「这就是——我的道。」

「我的——大道无形。」

光芒冲向毁灭者——

这一次——

毁灭者没能挡住。
