# 第十六章：建木

阿邪犹豫了一下。

他不知道这个声音是谁，也不知道把手伸出去会发生什么。

可他知道，他已经没有退路了。

他深吸一口气，把手掌放在石门上。

下一刻，一道光从门里射出，没入他的眉心。

阿邪的身体猛地一僵。

他感觉到了什么——那是一种极其奇怪的感觉，像是有什么东西在扫描他的身体，在他意识深处翻找着什么。

过了不知道多久，那感觉才消失。

「有意思。」

那个声音再次响起。

「你身上有维度之匙的印记。」

「又有烛龙的血脉气息。」

「还有——一元子的气息。」

「你到底是什么人？」

阿邪张了张嘴，想说什么。

可还没等他开口，石门就缓缓打开了。

门后是一片巨大的空间。

那空间大得看不到边际，天花板高得看不见，只有一片金黄色的光芒从四面八方照下来，让整个空间看起来像是一个巨大的琥珀。

空间的地面是一片白色的玉石，晶莹剔透，像是凝固了的水滴。玉石的表面光滑如镜，倒映着上方那金黄色的光芒，形成了一种奇异的对称效果。

而在空间的正中央——

有一棵树。

那树很高，高得看不到树冠，只能看见粗壮的树干。树干的颜色是金色的，表面布满了密密麻麻的纹路，像是某种古老的符文。

树干上挂着一些东西——阿邪眯起眼睛，仔细辨认。

那是一些……果实？

不，不是果实。

是光球。

一颗颗拳头大小的光球，挂在树干上，散发着柔和的光芒。那些光球有很多种颜色——红的、蓝的、紫的、透明的——像是某种宝石，又像是某种灵魂。

阿邪看着这一幕，心里涌起一股说不出的震撼。

这就是归墟？

这就是万水归集之地？

「那是建木。」

那个声音再次响起。

「真正的建木。」

「传说中支撑天地的那棵神木。」

阿邪回过头。

不知什么时候，他的身后站着一个身影。

那是一个年轻人，穿着一件白色的长袍，长发披散在肩头，面容俊美得不像凡人。他的眼睛是金色的，像是两颗燃烧的太阳，带着一种神圣而不可侵犯的气息。

「你是——」

「我是守树人。」年轻人说，「归墟的守树人。」

「我在这里等了很久了。」

「等你很久了？」

「对。等你这样一个人。」

年轻人看向阿邪，金色的眼睛里闪过一丝异样的光芒。

「你知道你身上背负着什么吗？」

阿邪沉默了。

他从一开始就被卷入了一场他完全不了解的棋局。铜片、一元子、维度之匙、四维世界——所有这些线索纠缠在一起，形成了一张他看不清全貌的网。

而他，只是这张网上的一个结点。

「我需要知道。」他说，「我需要知道这一切的答案。」

「答案？」守树人笑了，「答案就在你心里。」

「维度之匙会告诉你一切。」

「当你准备好的时候。」

「什么是准备好？」

「当你不再逃避的时候。」

守树人转过身，再次看向那棵建木。

「还有别的事吗？」

「我——」阿邪张了张嘴，「我要去赤水。找烛龙的龙珠。」

守树人回过头，看着他。

「烛龙的龙珠在赤水河底。」他说，「那里被墨渊封印了，以你现在的修为，进不去。」

「进不去？」

「封印很强。不是你能打破的。」

阿邪的心沉了下去。

如果龙珠拿不到——

「不过——」守树人话锋一转，「我可以帮你。」

「帮我？」

「我的本体在建木里，我出不了归墟。」守树人说，「但我可以借你一样东西。」

他抬起手，在空中画了一个符文。

那符文亮起，然后化作一道金光，飞入阿邪的眉心。

阿邪感觉到什么东西融入了他的意识——不是知识，不是记忆，而是一种……能力？

「这是建木之叶的力量。」守树人说，「能够破开一切封印的力量。」

「但只能用一次。」

「一次？」

「一次。」守树人点头，「而且只能用在你最需要的时候。」

「如果你现在就用，可能进得去赤水河底，但出来的时候就没有保障了。」

阿邪沉默了。

他需要这个力量。

可他不知道什么时候才是「最需要的时候」。

「好好考虑。」守树人说，「这个选择，只有你能做。」

他的身影开始变淡。

「等等——」阿邪想追问什么。

「有些事情，是命中注定的。」守树人的声音越来越远，「你改变不了开始，但你可以改变结局。」

「一元子说得对——」

「你来的时候，一切就开始了。」

「至于结局——」

他的身影彻底消失。

「掌握在你自己的手里。」
