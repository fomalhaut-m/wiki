# 卷四：归途

## 第六十三章：陌生人

第二天一早，阿邪在村口遇见了那个陌生人。

正如黎娘描述的——二十来岁，挺俊的，腰间挂着一把黑色的剑。

那陌生人看见阿邪，眼睛亮了起来。

「你就是阿邪？」

「是我。」阿邪说，「你是谁？」

「我叫剑无痕。」陌生人说，「来自——很远的地方。」

「你找我做什么？」

剑无痕没有直接回答。

他只是盯着阿邪，眼神里带着一种说不清的情绪。

「你就是那个杀死墨渊的人？」

「是。」

「你还见过'道'？」

「是。」

「你身上有四维世界的印记？」

阿邪的眉头皱得更紧了。

「你到底是谁？」

剑无痕笑了。

那笑容里带着一丝苦涩。

「我是另一个你。」他说。

「什么？」

「另一个你。」剑无痕重复道，「另一个四维世界里的你。」

「我们在四维世界里见过。」

阿邪愣住了。

他想起了观测者说的话——四维世界里，有无数个可能性。

有无数个"阿邪"。

「你是——四维世界的我？」

「不完全是。」剑无痕说，「我是四维世界的'观测者'。」

「是'道'分裂出来的那部分意识。」

「可同时——」

他停顿了一下。

「同时，我也是你。」

「是无数个可能性中，最接近你的那个。」
