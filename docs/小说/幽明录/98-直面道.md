# 卷五：大道（最终卷）

## 第九十八章：直面道

在四维世界的深处，阿邪见到了"道"。

"道"不是一个人。

它是一个——意识。

一个由无数可能性组成的意识。

「你来了。」"道"说。

「我来了。」阿邪说。

「你又来了。」

「是的。」

「为什么？」

「因为我想找到答案。」阿邪说，「我想知道——这一切的意义是什么。」

「你想让'道'停止它的梦？」

「不。」阿邪说，「我想让'道'——理解它的梦。」

「理解？」

「是的。」阿邪说，「你创造了这个世界。」

「可你不知道这个世界里的人在想什么。」

「你观察万物，却没有人能理解你。」

「这种孤独——」

「我知道。」

"道"沉默了。

阿邪继续说：

「你不是无敌的。」

「你有弱点。」

「你的弱点——是孤独。」

「因为孤独，你渴望被理解。」

「因为渴望被理解，所以你创造了这个梦。」

「可梦境终究是梦境。」

「总有一天，你会醒来。」

「到时候——」

「一切都会消失。」
