# 卷三：维度

## 第五十七章：选择

阿邪站在那里，脑子里一片空白。

三维世界是"道"的梦。

总有一天，梦境会醒来。

到时候，所有的一切都会消失。

「为什么告诉我这些？」他问。

「因为你有选择的权利。」观测者说。

「什么选择？」

「继续做梦的选择。」观测者说，「或者——」

「或者什么？」

「或者让'道'醒来。」

阿邪愣住了。

「让'道'醒来？」

「是的。」观测者点头，「如果你什么都不做，'道'迟早会自己醒来。」

「那是一个漫长的过程——可能是一万年，可能是十万年。」

「可终有一天，它会醒来。」

「到时候，三维世界会消失。」

「可如果你愿意——」

观测者的眼神变得认真。

「你可以选择提前唤醒它。」

「让它现在就醒来。」

「然后——」

「然后怎样？」

「然后一切都会消失。」观测者说，「可同时——」

「同时，一切都会自由。」

「回归混沌。」

「成为无限可能性中的一部分。」

「不再是某个疯子的梦境。」

阿邪沉默了。

他需要时间。

他需要思考。

「还有第三个选择吗？」

「有。」观测者说。

「什么选择？」

「继续做梦。」观测者说，「什么都不改变。」

「让'道'继续它的梦。」

「让三维世界继续运转。」

「让一切都保持原样。」

「直到那一天到来。」

阿邪看向念念和安安。

念念和安安站在他身后，眼神里有恐惧，有迷茫——

可更多的，是信任。

他们相信他。

他们相信他能做出正确的选择。

阿邪深吸一口气。
