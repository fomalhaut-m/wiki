# 第十五章：归墟

下山的路比上山要容易得多。

阿邪沿着山坡一路小跑，很快就来到了山脚下的平原。

平原上种满了水稻和麦子，一片金黄色的海洋在月光下闪闪发光。远处的村庄零星散布，偶尔传来几声犬吠和鸡鸣，充满了人间烟火气息。

这里和幽都完全是两个世界。

阿邪深吸一口气，继续向东走去。

他要去的地方在归墟的深处。

根据老头给他的那张地图，归墟的入口在东海之滨的某处岩石下面。从外面看，那只是一块普通的礁石，可一旦潮水退去，礁石下面就会露出一道裂缝。

那道裂缝，就是通往归墟深处的入口。

阿邪沿着海岸线走了大约两个时辰，终于找到了那块礁石。

礁石很大，足有三四丈高，表面布满了藤壶和海藻，看起来和周围的礁石没什么两样。

阿邪等到潮水退去，然后绕到礁石的另一侧——

他看见了。

礁石下面，确实有一道裂缝。

那裂缝只有一人多高，宽度刚好容一人侧身通过。裂缝的边缘是光滑的，像是被什么东西反复摩擦过，透着一股古老的气息。

他深吸一口气，侧身走了进去。

---

裂缝里面的世界，和他想象的完全不同。

他以为他会走进一条黑暗潮湿的通道，却没想到，裂缝里面竟然有光。

那光是从地面散发出来的，柔和而温暖，像是某种矿石发出的荧光。地面很干燥，铺着一层细密的白色沙子，踩上去发出轻微的沙沙声。

他沿着裂缝向前走，两旁的岩壁上开始出现一些图案。

那些图案是用某种红色的颜料画上去的，画的是一些奇怪的生物——有的像鱼，有的像龙，还有一些完全看不出是什么的东西。

阿邪仔细辨认着那些图案。

他认出了其中几个——

烛龙。應龙。夔牛。

这些都是《山海经》中记载的上古神兽。

看来，这个地方比他自己知道的还要古老。

他继续向前走，走了大约一炷香的功夫，前方出现了一道门。

那门是石头做的，高约两丈，宽约一丈，表面刻满了密密麻麻的符文。那些符文和铜片上的文字一样，阿邪一个都看不懂。

门的两侧各站着一个人形石像。

石像的面容模糊，看不出具体的五官，只能看出大致的轮廓——像是两个穿着铠甲的武士。

阿邪在门前停下脚步。

他要怎么进去？

他试着推了推门，石门纹丝不动。

他又试着把手放在门上的符文上，想看看会不会有什么反应——

就在他的手指触碰到符文的瞬间，门上的符文突然亮了起来。

那光是从门的中心向外扩散的，一圈一圈，像是水波一样。

然后，那些符文开始移动了。

它们从门上剥落下来，在空气中重组，形成了一行行的文字。

那些文字——

阿邪认出来了。

和铜片上的铭文一样。

这是四维文字。

他看不懂。

可就在他准备收回手的时候，一个声音在他脑海中响起。

「你能来到这里，说明你有缘。」

「伸出手，让我看看你。」

「看看你配不配进入归墟。」
