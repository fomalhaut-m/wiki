# 卷二：真相

## 第四十五章：审判

那一天，白木神殿上空出现了异象。

天裂了。

不是普通的裂，是从天的最深处向外撕裂的裂。

裂缝里透出的光，不是普通的光——

那是法则本身的光芒。

是"道"的光。

神殿里所有的人都看到了这一幕。

他们纷纷跑出屋外，仰头看着天空的裂缝。

「这是——」

「天裂？」

「怎么可能？天怎么会裂？」

「是末日的征兆吗？」

阿邪站在神殿的最高处。

他手里握着维度之匙。

那匙正在发光。

光芒从他手里升腾而起，直冲天际。

然后——

一道声音从天空中传来。

「吾乃'道'。」

「今有罪者，暴露于光。」

「神殿之人，三百年前，与魔道契约，杀害无辜之神。」

「此罪，吾已知晓。」

「罪者，当受审判。」

话音刚落。

神殿的十二把长老椅中，有三把突然碎裂。

三位长老惨叫着倒地。

他们的身体在发光，在燃烧，在消散。

「不——！」

「饶命——！」

可没有人能救他们。

他们犯下的罪，被"道"的光照得无所遁形。

他们为当年的契约所做的每一件事，都被"道"看在眼里。

所以——

他们被抹除了。

不是死。

是被从历史中抹除。

他们的存在，被彻底消除了。
