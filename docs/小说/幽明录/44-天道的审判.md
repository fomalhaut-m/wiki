# 卷二：真相

## 第四十四章：天道的审判

阿邪的计划很简单。

他要用"道"给的力量，打开一条通往天界的通道。

然后，他会把神殿的所有罪行，投射到天界去。

让天界来审判神殿。

「这不可能。」白渊说，「天界和神殿之间有屏障。任何人都无法穿越。」

「我不是任何人。」阿邪说，「我有维度之匙。」

「维度之匙能打开四维世界的通道。」

「而天界，就在四维世界里。」

白渊的眼睛亮了。

「你是说——」

「我是说，我有办法。」阿邪说，「但我需要你的帮助。」

「什么帮助？」

「我需要你去稳住那些长老。」

「别让他们在关键时刻破坏我的行动。」

白渊点头。

「还有呢？」

「还有——」阿邪停顿了一下，「我需要你保护念念和安安。」

「如果我的行动失败了，他们会成为目标。」

「我会保护他们的。」白渊说，「我保证。」

阿邪看着他的眼睛。

他看到了真诚。

「好。」他说，「那就开始吧。」
