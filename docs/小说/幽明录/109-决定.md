# 卷五：大道（最终卷）

## 第一百零九章：决定

阿邪想了很久。

他想起了他一路走来的经历。

想起了他最初的选择。

想起了他为什么选择保护念念和安安。

「我有一个问题。」他问七元子。

「什么问题？」

「如果我走了——其他人怎么办？」

「其他人？」

「扶摇海的渔民，归墟的守树人，万灵福地的老树——」

「所有那些——，生活在这个世界里的人。」

「他们会怎么样？」

七元子沉默了。

「他们会——消失。」

「和这个世界一起——回归混沌。」

阿邪看着他。

「我不能接受这个答案。」

「那你想怎么办？」

「我想——」阿邪说，「找到另一个办法。」

「一个不需要任何人消失的办法。」

「你还能找到吗？」

「我不知道。」阿邪说，「可我必须试试。」

「因为——」

「这就是我选择的道。」

「我的——大道无形。」
