# 卷四：归途

## 第六十四章：身份

阿邪把剑无痕带回了家。

念念和安安看见家里多了个陌生人，有些紧张。

「哥哥，他是谁？」

「是一个……远方的亲戚。」阿邪说。

他不想让孩子们知道太多。

剑无痕倒是很自然地和念念安安打招呼。

「你们就是念念和安安吧？」他笑着说，「我听说过你们。」

「听说过我们？」念念警惕地看着他，「你从哪里听说的？」

「从——很多地方。」剑无痕说，「在四维世界里，我知道很多事情。」

念念更紧张了。

阿邪拍了拍她的肩膀。

「没事。」他说，「他不是坏人。」

---

那天晚上，阿邪和剑无痕在院子里单独谈话。

「你来找我，是为了什么？」

「为了警告你。」剑无痕说。

「警告我？」

「是的。」剑无痕的表情变得严肃，「你做的选择——不走观测者给的三条路，而是选择自己的第四条路——」

「那个选择，已经被'道'察觉了。」

阿邪的心沉了下去。

「'道'察觉到了？」

「是的。」剑无痕点头，「'道'的梦境开始不稳定了。」

「因为你给了它一个新的可能性。」

「一个它没有想过的可能性。」

「这让它感到——威胁。」

阿邪沉默了。

他早就猜到，这条路不会好走。

可他没想到，代价会来得这么快。

「那现在怎么办？」

「现在——」剑无痕说，「你需要找到其他的'观测者'。」

「其他的观测者？」

「是的。」剑无痕说，「'道'分裂出了很多意识碎片。」

「那些碎片散落在三维世界的各个角落。」

「有的在洞天里，有的在福地里，有的——」

「有的在人的心里。」

「找到他们，」

「让他们帮助你。」

「只有这样，你才能对抗'道'的威胁。」
