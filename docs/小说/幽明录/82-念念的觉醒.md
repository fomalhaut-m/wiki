# 卷四：归途（终）

## 第八十二章：念念的觉醒

光芒持续了大约一刻钟，然后渐渐消散。

念念睁开眼睛。

她的眼睛变了——不再是普通的黑色瞳孔，而是变成了深邃的蓝色，像是两汪深不见底的潭水。

「姐姐！」安安凑过去，「你的眼睛——」

「怎么了？」念念摸了摸自己的脸。

「变成了蓝色！」

「蓝色？」念念愣了一下。

她走到水缸前，低头看向水面。

水面映出了她的倒影——包括那双蓝色的眼睛。

「这是——」

「水之魂的力量。」阿邪说，「你的血脉觉醒了。」

「水系的力量在你体内流淌。」

「从今以后，你可以操控水——」

「像呼吸一样自然。」

念念看着水缸里的自己，心里涌起一股奇异的感觉。

她能感觉到——

有什么东西在她体内流动。

像水一样，温柔而强大。

「我能——试试吗？」

「试试吧。」

念念深吸一口气，伸出手。

下一刻——

水缸里的水突然涌动，化作一道水柱，从缸中升起，在空中盘旋。

「哇——！」安安惊叹。

念念自己也惊呆了。

她只是想了想——水就动了。

这就是——水系的力量吗？
