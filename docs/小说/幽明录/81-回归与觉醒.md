# 卷四：归途（终）

## 第八十一章：回归与觉醒

回到扶摇海后，阿邪把水之魂和火之魂分别交给了念念和安安。

「这是什么？」念念接过那颗流动着蓝色光芒的晶石。

「水之魂。」阿邪说，「蕴含着水系法则的精华。」

「四元子让我把它交给你。」

念念愣了一下。

「交给我？」

「是的。」阿邪点头，「你体内有烛龙的血脉。烛龙是水火双系的神兽。」

「有了这个，你就能完全掌控水系的力量。」

念念看着手里的晶石，眼神复杂。

「可我——」

「你可以的。」阿邪说，「我相信你。」

念念沉默了一会儿，然后点点头。

「好。」

她把晶石握在手心，闭上眼睛。

晶石缓缓融入了她的身体。

刹那间，一道蓝色的光芒从她体内爆发，照亮了整个屋子。

「姐姐！」安安惊呼。

「没事。」阿邪说，「她在觉醒。」
