# 卷四：归途（续）

## 第八十章：火之礼

阿邪看着那团火焰。

他感觉到了里面的挣扎和痛苦。

五元子——这个掌管火系法则的存在——曾经失去过同伴。

它害怕再次失去。

害怕再次面对那个孤独的结局。

「我知道。」阿邪说，「我知道那条路不好走。」

「可我还是想试试。」

「就算失败了——」

「至少我努力过。」

「至少我——没有放弃。」

火焰再次沉默。

这一次，沉默持续了很久。

然后——

火焰熄灭了。

不是真的熄灭，而是收缩了。

收缩成了一颗小小的紫色晶石。

那晶石飘到阿邪面前。

「拿去。」五元子的声音从晶石中传出，「这是火之魂。」

「和四元子的水之魂是一对。」

「把火之魂交给那个叫安安的小子。」

「他的血脉里也有烛龙的力量。」

「烛龙是水火双系的——」

「有了这个，他就能掌控火系的力量。」

阿邪接过晶石。

「谢谢前辈。」

「别谢我。」五元子的声音变得低沉，「谢你自己。」

「谢你自己——没有放弃。」
