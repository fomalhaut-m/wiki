# 卷二：真相

## 第四十二章：分裂

议事堂内的气氛越来越紧张。

白渊和几位长老对峙着，谁也不让谁。

「够了。」主位上的老人终于开口了，「都安静。」

他的声音不大，却带着不容置疑的威严。

所有人都安静下来。

「白渊，」老人说，「你有证据吗？」

「白渊，」老人说，「你有证据吗？」

「有。」白渊说，「我师父的日记。」

「日记里记载了什么？」

「记载了神殿和墨渊的契约。」

「还记载了——」

他停顿了一下。

「还记载了，当年的任务，是谁下达的。」

全场死一般的寂静。

「是谁？」老人问。

白渊深吸一口气。

「是——」

「等等。」

另一个声音打断了他。

所有人都看向声音的来源。

是阿邪。

「阿邪？」白渊皱眉，「你要做什么？」

「我想说一件事。」阿邪说。

所有人的目光都集中在他身上。

「我想说——」阿邪扫视了一圈在场的所有长老，「我知道神殿和墨渊的契约。」

「因为三年前，我在墨渊身边待过很长时间。」

「我听说过一些事情。」

「什么事情？」老人问。

阿邪没有直接回答。

他只是看向白渊。

「让我和他单独谈谈。」
