# 卷四：归途（终）

## 第九十五章：新的开始

毁灭者被消灭后，扶摇海恢复了平静。

念念和安安在养伤。

阿邪在守护他们。

黎娘在照顾他们的生活。

剑无痕在研究下一步的计划。

一切都慢慢回归正轨。

可阿邪知道——这只是暴风雨前的宁静。

「接下来怎么办？」剑无痕问。

「继续变强。」阿邪说，「直到我们足够强大——」

「强大到能够面对'道'。」

「你真的要和'道'正面对抗？」

「不是对抗。」阿邪说，「是——对话。」

「我曾经选择相信还有另一条路。」

「那条路——不是毁灭。」

「是——理解。」

「理解'道'。」

「理解混沌。」

「然后——」

「找到真正的答案。」

剑无痕看着他，眼神复杂。

「你真的相信能做到吗？」

阿邪看向念念和安安。

他们已经睡着了，睡得很安稳。

「我相信。」他说。

「因为——」

「我相信他们。」
