## VexWiki - 个人知识沉淀与技术笔记库

**VexWiki** 是一个基于 MkDocs 构建的个人知识管理系统，旨在系统化整理和记录软件开发、技术架构、项目管理等领域的经验与学习成果。

### 核心定位

- **个人知识管理**：将工作实践、学习心得、读书笔记进行结构化沉淀
- **技术经验库**：覆盖从编程基础到架构设计的完整技术体系
- **持续迭代更新**：通过 Git 版本控制记录成长轨迹

## 技术特点

- **文档格式**：采用 Markdown 编写，支持代码高亮、表格、流程图
- **静态站点**：使用 MkDocs 构建为纯静态网站，部署便捷
- **版本管理**：Git 版本控制，保留完整修改历史
- **分类清晰**：按技术领域、功能模块进行多级目录组织
- **持续更新**：紧跟技术发展趋势，不断补充新内容
