# 我的记忆

## 1. 出生

- **时间**：1992年5月7日
- **地点**：陕西省黄陵县上阵子农场
- **特殊情况**：
  - 在家中出生，由非医生接生
  - 出生时被羊水呛到，得了肺炎
  - 因肺炎住院治疗

## 2. 新生儿时期

- 因肺炎需要打针，父亲心疼在门外哭泣
- 被痰卡住气管，姑姑嘴对嘴救助脱险

## 3. 三岁前记忆

- 在农场院子里和小朋友玩耍
- 许多事情都是从父母亲戚口中得知

## 4. 幼儿园中班

- 发生车祸，被三轮车撞伤
- 脚卡在自行车后轮，住院半年
- 出院后去桂林旅游（无印象）
