# 自传

## 童年时期

出生于陕西省黄陵县，童年在农场度过。幼儿园时期曾遭遇车祸，住院半年。

小学在南塘小学就读，后经历多次转学，在西四路纺织子校度过一段重要时光。这时期结识了许多朋友，包括后来的伴郎师洁鑫。

## 求学经历

初中在前进路中学8班就读，班级人数达108人。初中时期暗恋过同学，但学业上开始跟不上进度。

后进入西职院学习，在那里认识了重要朋友李洪圆。

## 工作经历

早期从事电工相关工作，曾在多家公司任职：
- 西安开天科技有限公司（普通工人）
- 人人乐（电工）
- 西安仁浩自动化控制有限公司（装配电工）
- 西安蓝辉机电
- 陕西诺立自动控制工程有限公司（工作时间较长）
- 陕西浩远通信
- 东润科技有限公司
- 反射狐（获得认可，工作开心）

## 技术转型

后来决定学习Java编程，转型为软件开发工程师。这段经历中结识了中之一、赵鑫、吴昊等新朋友。

## 成长感悟

从电工到高级工程师的转型是一个艰辛但充实的旅程。每一个阶段的经历都塑造了现在的自己。
