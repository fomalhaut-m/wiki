# 分布式框架

## Dubbo（RPC 框架）

## 核心概念

Apache Dubbo 是高性能、轻量级的 RPC 分布式服务框架：

| 角色 | 说明 |
|------|------|
| Provider | 服务提供者 |
| Consumer | 服务消费者 |
| Registry | 注册中心（Zookeeper / Nacos） |
| Monitor | 监控中心 |
| Container | 服务运行容器 |

## 工作流程

```
Consumer → Registry → 获取 Provider 列表 → 直连调用
                                    ↓
                              Monitor 统计
```

## 快速使用

**服务接口：**
```java
public interface HelloService {
    String sayHello(String name);
}
```

**Provider（服务提供方）：**
```java
// application.yml
dubbo:
  application:
    name: dubbo-provider
  registry:
    address: zookeeper://127.0.0.1:2181
  protocol:
    name: dubbo
    port: 20880

@Service
public class HelloServiceImpl implements HelloService {
    @Override
    public String sayHello(String name) {
        return "Hello " + name;
    }
}
```

**Consumer（服务消费方）：**
```java
// 引用远程服务
@Reference
private HelloService helloService;

public void test() {
    String result = helloService.sayHello("Dubbo");
}
```

## Zookeeper（分布式协调）

## 概述

Zookeeper 是分布式一致性服务框架，用于**配置管理、命名服务、分布式锁、集群管理**。

## 数据模型

树形结构，每个节点叫 **ZNode**：
```
/
├── /app1
│   ├── /config
│   └── /users
├── /dubbo
│   └── /com.example.HelloService
└── /locks
```

## 常用命令

```bash
# 启动
zkServer.sh start

# 连接
zkCli.sh -server 127.0.0.1:2181

# 查看
ls /dubbo
get /dubbo/com.example.HelloService

# 创建
create /myapp "config"
create -e /myapp/ephemeral "temp"  # 临时节点

# 更新
set /myapp "newconfig"

# 删除
delete /myapp
```

## Java 客户端（Curator）

```java
CuratorFramework client = CuratorFrameworkFactory.newClient(
    "127.0.0.1:2181", new ExponentialBackoffRetry(1000, 3));
client.start();

// 创建节点
client.create().forPath("/mylock", "lock".getBytes());

// 获取数据
byte[] data = client.getData().forPath("/mylock");

// 分布式锁
InterProcessMutex lock = new InterProcessMutex(client, "/locks/mylock");
lock.acquire(10, TimeUnit.SECONDS);
try {
    // 临界区操作
} finally {
    lock.release();
}

client.close();
```

## Zookeeper 在 Dubbo 中的作用

Dubbo 用 Zookeeper 做**注册中心**：
- Provider 启动时把自己的 IP/端口/services 写入 `/dubbo/com.example.HelloService/providers`
- Consumer 订阅 `/dubbo/.../providers`，获取 Provider 列表
- Provider 下线时，Zookeeper 会通知 Consumer 更新列表
