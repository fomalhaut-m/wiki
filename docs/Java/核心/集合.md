# 集合框架

## 顶层接口结构

```
Iterable<T>
  └── Collection<E>
        ├── List<E>         有序、可重复
        ├── Set<E>          无序、去重
        └── Queue<E>        队列（FIFO）
              └── Deque<E>  双端队列
```

## List（有序、可重复）

| 实现类 | 底层 | 特点 |
|--------|------|------|
| `ArrayList` | 数组 | 查询快，增删慢，线程不安全 |
| `LinkedList` | 双向链表 | 增删快，查询慢 |
| `Vector` | 数组 | 线程安全，已淘汰 |
| `CopyOnWriteArrayList` | 数组 | 读多写多，线程安全 |

### 常用方法

```java
list.add(1, elem)       // 指定位置插入
list.get(index)          // 获取
list.remove(index)       // 按索引删
list.remove(elem)        // 按对象删
list.set(index, elem)    // 修改
list.contains(elem)      // 是否包含
list.subList(from, to)   // 切片
```

## Set（无序、去重）

| 实现类 | 底层 | 特点 |
|--------|------|------|
| `HashSet` | 哈希表 | 性能最优，不保证顺序 |
| `LinkedHashSet` | 哈希表 + 链表 | 保持插入顺序 |
| `TreeSet` | 红黑树 | 自然顺序/自定义排序 |

### TreeSet 排序

```java
// 自然顺序（元素实现 Comparable）
TreeSet<Integer> ts = new TreeSet<>();

// 自定义排序
TreeSet<User> ts = new TreeSet<>(Comparator.comparing(User::getAge));
```

## Map（键值对，Key 无序不可重复）

| 实现类 | 底层 | 特点 |
|--------|------|------|
| `HashMap` | 哈希表 | 高性能，线程不安全 |
| `LinkedHashMap` | 哈希表 + 链表 | 保持插入顺序 |
| `TreeMap` | 红黑树 | Key 自然排序 |
| `Hashtable` | 哈希表 | 线程安全，已淘汰 |
| `ConcurrentHashMap` | 分段锁（1.7）/ CAS（1.8+） | 高并发安全 |

### 常用方法

```java
map.put(key, value)       // 插入/覆盖
map.get(key)              // 获取
map.remove(key)           // 删除
map.containsKey(key)      // 是否包含 Key
map.containsValue(value)  // 是否包含 Value
map.keySet()              // 获取所有 Key
map.values()              // 获取所有 Value
map.entrySet()            // 获取键值对集合
map.getOrDefault(key, defaultValue)  // 安全取值
```

### HashMap 底层原理（JDK 1.8+）

```
数组（Node[]） + 链表（拉链法） + 红黑树（树化阈值 8）
```

**put 流程：**
1. 计算 Key 的 hash 值，确定数组下标
2. 该位置为空，直接插入
3. 有值 → equals 判断是否相同 → 相同覆盖，不同成链表
4. 链表长度 ≥ 8 → 树化为红黑树

**扩容：** 容量达到 `capacity * loadFactor`（默认 0.75）时，扩容 2 倍。

## Queue（队列，FIFO）

| 方法 | 描述 |
|------|------|
| `add/offer` | 添加到队尾（满时抛异常/返回 false） |
| `remove/poll` | 移除队首（空时抛异常/返回 null） |
| `element/peek` | 查看队首（不删除） |

```java
Queue<String> q = new LinkedList<>();
q.offer("A");    // 入队
q.poll();        // 出队
q.peek();        // 查看
```

**阻塞队列：**

| 实现 | 特点 |
|------|------|
| `ArrayBlockingQueue` | 有界，数组实现 |
| `LinkedBlockingQueue` | 无界（Integer.MAX_VALUE），链表实现 |
| `PriorityBlockingQueue` | 支持优先级排序 |
| `SynchronousQueue` | 容量为 0，必须同步交付 |

## Collections 工具类

```java
Collections.sort(list)                    // 排序
Collections.reverse(list)               // 反转
Collections.shuffle(list)                // 洗牌
Collections.max(list) / min(list)         // 最值
Collections.frequency(list, elem)        // 元素出现次数
Collections.swap(list, i, j)             // 交换
Collections.synchronizedList(list)       // 线程安全包装
Collections.emptyList() / emptyMap()     // 返回空不可变集合
```

## 并发集合（java.util.concurrent）

| 集合 | 说明 |
|------|------|
| `CopyOnWriteArrayList` | 写时复制，读不加锁，适用于读多写少 |
| `ConcurrentHashMap` | 分段锁（1.7）/ CAS+ synchronized（1.8+），替代 Hashtable |
| `ConcurrentLinkedQueue` | 无界非阻塞队列 |
| `BlockingQueue` | 阻塞队列，支持 take/put 自动阻塞 |
| `ConcurrentSkipListMap/Set` | 跳表实现，有序并发安全 |
