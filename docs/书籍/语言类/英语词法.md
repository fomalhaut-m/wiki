# 英语词法扫盲

> 来源：英语学习笔记 | 标签：语言 / 英语

---

## 词类总览

| 词类 | 说明 | 示例 |
|------|------|------|
| **名词** | 表示人/事物/地点/概念 | book, China, happiness |
| **代词** | 代替名词 | I, you, he, it |
| **动词** | 表示动作或状态 | run, be, think |
| **形容词** | 修饰名词 | beautiful, big |
| **副词** | 修饰动词/形容词/副词 | quickly, very |

---

## 名词

### 分类
- **可数名词**：单数 → 复数（book → books）
- **不可数名词**：无法计数（water, information）

### 复数变化规则
| 规则 | 示例 |
|------|------|
| 加 s | book → books |
| 去掉 sses | class → classes |
| 辅音 + y → ies | city → cities |
| f/fe → ves | leaf → leaves |

---

## 代词

| 类型 | 示例 |
|------|------|
| 人称代词 | I, you, he, she, it, we, they |
| 物主代词 | my, your, his, her, its, our, their |
| 指示代词 | this, that, these, those |
| 疑问代词 | who, whom, whose, which, what |

---

## 动词

| 类型 | 示例 |
|------|------|
| 及物动词 | I eat an apple.（接宾语） |
| 不及物动词 | I sleep well.（不接宾语） |
| be 动词 | am, is, are, was, were |
| 助动词 | do, does, did, have, has |

---

## 形容词

- 放在名词前：a **beautiful** girl
- 系表结构：The food **is** delicious.
- 比较级/最高级：big → bigger → biggest

---

## 副词

- 修饰动词：He runs **quickly**.
- 修饰形容词：The movie is **very** interesting.
- 修饰副词：She speaks **extremely** clearly.
