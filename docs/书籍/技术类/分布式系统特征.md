# 分布式系统特征

> 来源：《分布式系统》读书笔记 | 标签：技术 / 分布式

---

## 定义

分布式系统 = 组件分布在连网计算机上 + 通过传递消息通信和协调的系统

---

## 三大特征

| 特征 | 说明 |
|------|------|
| **组件并发** | 资源竞争与协作，共享资源能力随硬件增加而提高 |
| **缺乏全局时钟** | 同步准确性有限，无正确时间的全局概念 |
| **组件故障独立性** | 单个组件故障，其他组件仍可运行 |

---

## 七大挑战

| 挑战 | 说明 |
|------|------|
| **异构性** | 网络/硬件/OS/编程语言/软件实现的差异 |
| **开放性** | 新服务能被增加并供多种客户使用 |
| **安全性** | 机密性、完整性、可用性 |
| **可伸缩性** | 用户/资源激增时仍保持有效 |
| **故障处理** | 检测→掩盖→容错→恢复→冗余 |
| **并发性** | 多客户并发操作需同步 |
| **透明性** | 访问/位置/并发/故障/性能/伸缩透明 |

---

## 八种透明性

| 透明性 | 说明 |
|--------|------|
| **访问透明** | 用相同操作访问本地和远程资源 |
| **位置透明** | 不需要知道资源的物理/网络位置 |
| **并发透明** | 多进程并发使用共享资源互不干扰 |
| **故障透明** | 屏蔽错误，用户仍能完成任务 |
| **性能透明** | 负载变化时可重新配置提高性能 |
| **伸缩透明** | 系统扩展不改变结构或算法 |
