# Blender 与 Ae 基础

本文档介绍两款主流 3D 和视频特效软件的基础知识。

## Blender

Blender 是一款**全能型 3D 创作平台**，集 3D 建模、雕刻、动画、渲染、模拟、视频编辑、VFX、游戏开发于一体。

### 核心优势

| 优势 | 说明 |
|------|------|
| **3D 内容生成** | 尤其擅长有机模型（角色、生物）和复杂场景的搭建 |
| **渲染引擎** | Cycles（照片级渲染）和 Eevee（实时预览） |
| **动画系统** | 关键帧、骨骼绑定、动作捕捉 |
| **物理模拟** | 流体、布料、粒子、爆炸、烟雾等效果 |

### 适用场景

- 游戏角色建模
- 动画制作
- 产品可视化
- 建筑渲染

## Adobe After Effects

Adobe After Effects 是专业**动态图形与 VFX 软件**，专注于动态图形设计、视频特效合成、运动跟踪等。

### 核心优势

| 优势 | 说明 |
|------|------|
| **后期合成** | 适合片头片尾、广告特效、UI 动效 |
| **Adobe 集成** | 与 Photoshop、Premiere Pro 无缝协作 |
| **特效插件** | Trapcode、Form 等丰富插件生态 |

### 适用场景

- 电影特效
- 广告制作
- UI 动效
- 视频包装

## 配置推荐

| 组件 | 最低要求 | 推荐配置 |
|------|---------|----------|
| CPU | 4核 | 8核及以上 |
| 显卡 | 显存 4GB | 显存 8GB 及以上 |
| 内存 | 8GB | 16GB+ |
| 硬盘 | 256GB 固态 | 512GB 固态 + 1TB 机械 |
| 显示器 | 1080P | 2K 或 4K |
| 鼠标 | 标准鼠标 | 带中键滚轮 |

### Blender 特殊要求

- **键盘**：建议使用全尺寸键盘，便于使用快捷键
- **鼠标**：必须带中键滚轮，是核心操作的必备

## 协同工作

通过 **BlenderAE 插件**，可以在 Blender 和 After Effects 之间实现数据的导入导出：

- Blender 负责 3D 建模和渲染
- Ae 负责后期合成和特效

## 学习路径

1. **Blender 入门** - 基础操作、界面布局、简单建模
2. **Blender 进阶** - 材质、灯光、渲染
3. **Ae 入门** - 基础动画、特效合成
4. **实战项目** - 综合运用两款软件完成作品
