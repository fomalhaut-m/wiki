## 核心知识点

### 逻辑结构
- **集合**：数据元素同属一个集合
- **线性结构**：线性表、栈、队列（一对一关系）
- **树形结构**：二叉树、平衡树、红黑树（一对多关系）
- **图状结构**：网状结构（多对多关系）

### 存储结构
- **顺序存储**：连续内存空间
- **链式存储**：指针链接
- **索引存储**：索引表
- **散列存储**：哈希存储

## 常用数据结构

| 类型 | 代表结构 |
|------|---------|
| 线性 | 数组、链表、栈、队列 |
| 树形 | 二叉树、B树、B+树、红黑树 |
| 图形 | 有向图、无向图 |
| 散列 | 哈希表 |
