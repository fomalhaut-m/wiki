# Vue 3 学习笔记

## 简介

Vue 3 是 Vue.js 的下一代版本，带来了更好的性能、TypeScript 支持和 Composition API。

## 新特性

| 特性 | 说明 |
|------|------|
| **Composition API** | 组合式API |
| **Teleport** | 传送门组件 |
| **Fragments** | 多根节点支持 |
| **Suspense** | 异步组件加载 |
| **更好的TypeScript支持** | 完整的类型推导 |
