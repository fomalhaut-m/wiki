# ECMAScript 6

## 简介

ES6是JavaScript的第六版标准，引入了许多新语法和特性。

## 核心新特性

| 特性 | 说明 |
|------|------|
| **let/const** | 块级作用域 |
| **箭头函数** | 简化的函数语法 |
| **模板字符串** | 多行字符串和插值 |
| **解构赋值** | 对象和数组解构 |
| **Promise** | 异步编程 |
| **Class** | 类语法糖 |
| **模块化** | import/export |

## 学习路径

```
变量声明 → 函数增强 → 异步处理 → 模块化
```
