## 模式分类

| 类型 | 说明 |
|------|------|
| **创建型** | 单例、工厂、建造者、原型 |
| **结构型** | 适配器、装饰器、代理、组合、外观、桥接 |
| **行为型** | 策略、观察者、命令、模板方法、迭代器、责任链、状态、备忘录、解释器 |
