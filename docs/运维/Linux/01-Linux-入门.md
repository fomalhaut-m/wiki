# Linux 入门

## 概述

Linux 是一款开源、稳定、安全的操作系统，处理高并发能力强，企业级应用广泛部署在 Linux 服务器上。

**基本信息：**
- 创始人：林纳斯（Linus Torvalds）
- 吉祥物：Tux（企鹅）
- 核心理念：一切皆文件

## Linux vs Windows

| 对比 | Linux | Windows |
|------|-------|---------|
| 费用 | 免费开源 | 收费 |
| 安全性 | 高（权限模型） | 一般 |
| 稳定性 | 高（适合服务器） | 桌面更优 |
| 软件生态 | 服务器软件丰富 | 桌面软件丰富 |
| 命令行 | 强大，Shell 原生 | PowerShell / CMD |

## 发行版本

| 版本 | 特点 |
|------|------|
| **CentOS / RHEL** | 企业级，稳定，兼容 Red Hat |
| **Ubuntu** | 社区活跃，桌面/服务器两用 |
| **Debian** | 最纯净，社区驱动 |
| **SUSE** | 企业级，付费支持 |

## 安装环境

**常用组合：** Windows + VM（VMware / VirtualBox）+ CentOS

| 网络模式 | 说明 |
|----------|------|
| **NAT** | 推荐，IP 不冲突，Linux 可访问外网 |
| 桥接 | 方便，可能造成 IP 不够用 |

## 常用远程工具

| 工具 | 用途 |
|------|------|
| **Xshell** | 远程终端控制 |
| **Xftp** | 文件上传下载 |

## vi / vim 编辑器

vim 是 vi 的增强版，语法高亮、代码补全、编译跳转等功能更丰富。

### 三种模式

| 模式 | 进入方式 | 用途 |
|------|---------|------|
| **正常模式** | 打开文件默认进入 | 移动光标、删除、复制、粘贴 |
| **插入模式** | 按 `i` 或 `a` | 编辑文本 |
| **命令行模式** | 按 `:` | 读取、存盘、替换、退出 |

### 常用快捷键

| 命令 | 说明 |
|------|------|
| `gg` | 跳到首行 |
| `G` | 跳到尾行 |
| `dd` | 删除当前行 |
| `yy` | 复制当前行 |
| `p` | 粘贴 |
| `/关键词` | 搜索 |
| `u` | 撤销 |
| `:set nu` | 显示行号 |

## 运行级别

| 级别 | 说明 |
|------|------|
| 0 | 关机 |
| 1 | 单用户（维护模式） |
| 3 | 多用户有网络（命令行） |
| 5 | 图形界面 |
| 6 | 重启 |

```bash
# 切换运行级别
init [0|1|2|3|4|5|6]

# 查看当前级别
runlevel
```

## 目录结构

```
/           根目录
├── /bin    常用命令
├── /boot   启动文件
├── /dev    设备文件（一切皆文件）
├── /etc    配置文件
├── /home   用户主目录
├── /lib    系统动态连接库
├── /opt    额外安装软件
├── /proc   内存映射（虚拟文件系统）
├── /root   超级管理员目录
├── /sbin   超级管理员命令
├── /usr    用户程序和文件
├── /var    经常修改的文件（日志）
└── /tmp    临时文件
```
