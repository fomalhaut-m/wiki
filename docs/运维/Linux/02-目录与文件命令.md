# 目录与文件命令

## 目录操作

| 命令 | 说明 | 示例 |
|------|------|------|
| `pwd` | 显示当前目录绝对路径 | `pwd` |
| `ls [-a/-l]` | 列出目录内容 | `ls -la` |
| `cd [路径]` | 切换目录 | `cd ~` 回到家目录；`cd ..` 上级 |
| `mkdir [-p]` | 创建目录 | `mkdir -p a/b/c` 创建多级 |
| `rmdir` | 删除空目录 | `rmdir dirname` |

## 文件操作

| 命令 | 说明 | 示例 |
|------|------|------|
| `touch 文件名` | 创建空文件 | `touch a.txt b.txt` |
| `cp [-r] 源 目标` | 拷贝文件/目录 | `cp -r dir1 dir2/` |
| `rm [-rf]` | 删除文件/目录 | `rm -rf dirname` |
| `mv 旧名 新名` | 重命名或移动 | `mv a.txt b.txt` |
| `ln -s 源 链接名` | 创建软链接 | `ln -s /data mydata` |

## 文件查看

| 命令 | 说明 | 特点 |
|------|------|------|
| `cat [-n]` | 一次性显示文件内容 | 适合小文件，-n 显示行号 |
| `more` | 分页显示 | 按空格下一页，`q` 退出 |
| `less` | 分页显示（更强大） | 上下键翻页，支持搜索 |
| `head [-n N]` | 显示前 N 行 | `head -n 5 file` |
| `tail [-n N]` | 显示后 N 行 | `tail -n 5 file` |
| `tail -f` | 实时追踪文件更新 | 常用于看日志 |

## 输出重定向

```bash
# 覆盖写入
命令 > 文件

# 追加写入
命令 >> 文件

# 将命令1的输出作为命令2的输入
命令1 | 命令2
```

## 查找

### find 查找文件

```bash
# 按文件名
find / -name "*.txt"

# 按用户
find / -user username

# 按大小
find / -size +100M
```

### grep 文本搜索

```bash
# 在文件中搜索关键词
grep "关键词" 文件名

# 递归搜索
grep -r "关键词" 目录/

# 显示行号
grep -rn "关键词" 目录/

# 忽略大小写
grep -i "关键词" 文件
```

## 压缩与解压

| 格式 | 压缩 | 解压 |
|------|------|------|
| `.tar` | `tar -cvf a.tar 文件/` | `tar -xvf a.tar` |
| `.gz` | `gzip 文件` | `gunzip 文件.gz` |
| `.tar.gz` | `tar -zcvf a.tar.gz 文件/` | `tar -zxvf a.tar.gz` |
| `.zip` | `zip -r a.zip 目录/` | `unzip a.zip` |

## 权限管理

### 权限模型

每个文件有三类权限：**所有者（u）/ 所属组（g）/ 其他（o）**

| 权限 | 符号 | 数字 |
|------|------|------|
| 读 | r | 4 |
| 写 | w | 2 |
| 执行 | x | 1 |

```bash
# 修改权限
chmod 755 文件        # 数字方式
chmod u+x 文件        # 符号方式

# 修改所有者
chown 用户名:组名 文件
```

## 帮助命令

```bash
man 命令         # 查看命令手册
help 命令        # 内部命令帮助
man -k 关键词    # 搜索关键词相关命令
```
