# Linux 系统命令

## 用户与用户组

```bash
# 用户管理
useradd 用户名          # 创建用户
userdel 用户名          # 删除用户
passwd 用户名           # 设置密码
su - 用户名             # 切换用户

# 用户组
groupadd 组名           # 创建组
groups 用户名           # 查看用户所属组
```

## 进程管理

| 命令 | 说明 |
|------|------|
| `ps -ef` | 查看所有进程 |
| `ps aux` | 查看进程（更详细） |
| `top` | 动态查看进程（类似任务管理器） |
| `pstree` | 进程树展示 |
| `kill PID` | 杀死进程（温和） |
| `kill -9 PID` | 强制杀死进程 |

```bash
# 按 CPU / 内存排序
top
# 按 P 排序CPU，按 M 排序内存

# 后台运行
nohup 命令 &
```

## 服务管理（systemd）

```bash
# 服务操作（CentOS 7+）
systemctl start 服务名
systemctl stop 服务名
systemctl restart 服务名
systemctl status 服务名
systemctl enable 服务名    # 开机启动
systemctl disable 服务名   # 取消开机启动
```

## 网络管理

| 命令 | 说明 |
|------|------|
| `ifconfig` / `ip addr` | 查看 IP 地址 |
| `netstat -tulnp` | 查看监听端口 |
| `netstat -anp` | 查看所有网络连接 |
| `ping IP/域名` | 测试连通性 |
| `curl URL` | 发起 HTTP 请求 |
| `wget URL` | 下载文件 |
| `traceroute 域名` | 路由追踪 |
| `nslookup 域名` | DNS 查询 |

```bash
# 端口占用
lsof -i:8080
netstat -tulnp | grep 8080
```

## 磁盘与文件系统

```bash
df -h              # 查看磁盘使用率
du -sh 目录/       # 查看目录大小
lsblk              # 查看块设备
mount /dev/sdb1 /mnt  # 挂载
umount /mnt        # 卸载
```

## 内存与 CPU

```bash
free -h            # 查看内存使用
cat /proc/cpuinfo  # 查看 CPU 信息
nproc              # CPU 核心数
uptime             # 运行时长 + 负载
```

## 软件包管理

### CentOS / RHEL（yum / dnf）

```bash
yum install 包名       # 安装
yum remove 包名        # 卸载
yum update 包名        # 更新
yum list installed     # 已安装列表
yum search 关键词      # 搜索
```

### Ubuntu / Debian（apt）

```bash
apt install 包名       # 安装
apt remove 包名        # 卸载
apt update             # 更新源
apt upgrade            # 升级
```

## 定时任务 cron

```bash
# 编辑 crontab
crontab -e

# 格式：分 时 日 月 周 命令
# 示例：每天凌晨3点执行
0 3 * * * /root/backup.sh

# 查看定时任务
crontab -l
```

## 日志查看

```bash
# 系统日志
tail -f /var/log/messages       # CentOS
tail -f /var/log/syslog         # Ubuntu

# 历史命令
history                          # 查看历史命令
!n                               # 执行第 n 条命令
```

## SCP 文件传输

```bash
# 从本地上传到远程
scp 文件 root@IP:/目标路径

# 从远程下载到本地
scp root@IP:/远程文件 本地路径

# 递归传输目录
scp -r 目录 root@IP:/目标路径
```
