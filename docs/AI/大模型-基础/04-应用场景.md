# 大模型应用场景

## 按模态分类

### 纯文本应用

| 场景 | 说明 |
|------|------|
| 智能客服 | 多轮对话，自动问答 |
| 内容生成 | 文章写作、代码生成、翻译 |
| 知识助手 | 个人知识库问答 |
| 文本分析 | 摘要、分类、情感分析 |

### 工具增强应用

| 场景 | 说明 |
|------|------|
| **Agent** | 大模型 + 工具调用 + 规划执行 |
| **RAG** | 大模型 + 知识库检索 |
| **MCP** | 大模型 + 外部协议工具 |

### 多模态应用

| 场景 | 说明 |
|------|------|
| 文生图 | Stable Diffusion、Midjourney |
| 图文理解 | 文档识别、OCR |
| 视频理解 | 视频摘要、内容分析 |
| 语音助手 | ASR + LLM + TTS |

## 常见产品形态

```
单一模型调用
    ↓
模型 + Memory（记忆）= 连续对话
    ↓
模型 + Tools（工具）= Agent
    ↓
模型 + Knowledge（知识库）= RAG
    ↓
模型 + MCP（外部服务）= 生态集成
```

## 企业落地常见架构

### 1. 纯 API 调用（最简单）
```
应用 → LLM API → 返回结果
```

### 2. RAG 架构（知识问答）
```
用户问题 → 检索 → 上下文拼接 → LLM → 回答
           ↑
        向量数据库
```

### 3. Agent 架构（自主执行）
```
用户目标 → 规划 → 工具调用 → 结果 → 判断是否完成 → 继续或结束
```

### 4. 多 Agent 协作（复杂任务）
```
Agent A（规划）→ Agent B（搜索）→ Agent C（执行）→ 汇总结果
```
