# 大模型训练三阶段

大模型的训练整体分为 3 个阶段：

```
预训练 → SFT（监督微调）→ RLHF（无监督微调 + 人类反馈）
```

## 1. 预训练（Pretraining）

**目标：** 让模型学习语言的通用表示。

模型在超大规模文本上做自监督学习，学习：
- 词嵌入（Word Embedding）
- 上下文依赖关系
- 句法结构
- 语义表示

> 类似于从小学到高中，打下知识基础。

## 2. SFT 监督微调

**SFT = Supervised Fine-Tuning**

在预训练模型的基础上，用**有标注的监督数据**做微调，使模型适应特定任务。

```python
# 典型 SFT 数据格式
{
    "instruction": "把下面的句子翻译成英文",
    "input": "今天天气真好",
    "output": "The weather is nice today."
}
```

> 类似于从高中毕业进入大学，学习专业技能。

## 3. RLHF 人类反馈强化学习

**RLHF = Reward Learning + Human Feedback**

让模型根据人类反馈信号持续优化：

1. **Reward Learning**：训练一个奖励模型（RM），学习人类偏好
2. **PPO 强化学习**：用 RM 的反馈信号微调主模型

```
用户反馈（夸/批评）→ 奖励模型 → 强化学习 → 模型更新
```

> 类似于大学毕业后进入职场，在实际反馈中不断成长。

## 三阶段对比

| 阶段 | 学习方式 | 数据来源 | 目标 |
|------|---------|---------|------|
| 预训练 | 自监督 | 互联网大规模文本 | 学通用语言能力 |
| SFT | 监督学习 | 人工标注 | 学特定任务能力 |
| RLHF | 强化学习 | 人类反馈 | 对齐人类偏好 |
