# 大模型的特点与分类

## 大模型的核心特点

| 特点 | 说明 |
|------|------|
| **规模大** | 参数数量庞大（十亿到万亿级） |
| **适应性强** | 通过微调可适配多种下游任务 |
| **数据广泛** | 在海量多样化文本上训练 |
| **资源需求高** | 需要大量 GPU 计算资源 |

## 分类

### 1. 大语言模型 LLM

专注于文本理解和生成：

| 模型 | 代表 |
|------|------|
| GPT 系列 | OpenAI |
| Claude | Anthropic |
| DeepSeek | 深度求索 |
| MiniMax | 稀宇科技 |
| GLM | 智谱 AI |

### 2. 多模态模型

能同时处理多种模态信息：

| 模态 | 能力 |
|------|------|
| **视觉** | 图像理解、目标检测、文生图 |
| **语音** | 语音识别（ASR）、语音合成（TTS） |
| **视频** | 视频理解、视频生成 |
| **文档** | PDF 解析、表格理解 |

### 代表性多模态模型

| 模型 | 能力 |
|------|------|
| GPT-4V | 图文理解 |
| GPT-4o | 文本 + 语音 + 视觉 |
| Gemini | 原生多模态 |
| Claude 3 | 图像理解 |
| 豆包 | 字节跳动多模态 |

## 常见应用形态

```
纯文本 LLM
    ↓ + 工具调用
Agent（智能体）
    ↓ + 知识库
RAG（检索增强生成）
    ↓ + 多模态
多模态应用
```
