# AI Agent 入门

> 来源：个人笔记整理 | 标签：AI / Agent

---

## AI Agent 是什么

让AI能够自主决策、执行任务、工具调用的智能体。

---

## 主要模块

| 模块 | 说明 |
|------|------|
| **规划** | 任务拆解、目标分解 |
| **记忆** | 短期+长期记忆 |
| **工具** | 搜索、代码执行、API调用 |
| **行动** | 执行决策、输出结果 |

---

## 技术方向

- LangChain
- AutoGPT
- CrewAI
- Dify / Coze

---

## 策略分析

- Prompt Engineering
- RAG（检索增强生成）
- Agentic Workflow

---

## 单Agent vs 多Agent

| 类型 | 说明 |
|------|------|
| **单Agent** | 单一智能体，自主决策 |
| **多Agent** | 多个智能体协作，分工 |

---

## 个性化应用

- 定制专属Agent
- 知识库对接
- 工作流自动化
