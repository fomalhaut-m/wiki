# 结构型设计模式

> 来源：GOF设计模式 | 标签：设计模式 / Java

---

## 适配器模式

**定义**：两个不兼容接口之间的桥梁。

```java
interface Target { void request(); }
class Adaptee { void specificRequest() { /* ... */ } }
class Adapter implements Target {
    Adaptee adaptee;
    void request() { adaptee.specificRequest(); }
}
```

**应用**：读卡器（内存卡→笔记本）、变压器（110V→220V）

---

## 装饰器模式

**定义**：动态给对象添加功能，不改变结构。

```java
interface Shape { void draw(); }
class Circle implements Shape { public void draw() { /* 画圆 */ } }
class Decorator implements Shape {
    Shape shape;
    Decorator(Shape s) { this.shape = s; }
    public void draw() { shape.draw(); /* 额外功能 */ }
}
```

**应用**：孙悟空72变、外套添加颜色

⚠️ 装饰器 vs 代理：装饰器增强功能，代理加以控制

---

## 代理模式

**定义**：为其他对象提供替身控制访问。

```java
interface Image { void display(); }
class RealImage implements Image { public void display() { /* 加载 */ } }
class ProxyImage implements Image {
    RealImage real;
    public void display() {
        if (real == null) real = new RealImage();
        real.display();
    }
}
```

**应用**：快捷方式、火车票代售点、Spring AOP

⚠️ 代理 vs 适配器：代理不能改变接口，适配器改变接口

## 外观模式

**定义**：隐藏系统复杂性，提供统一高层接口。

```java
class Computer {
    CPU cpu; Memory mem; Disk disk;
    void start() { cpu.start(); mem.init(); disk.read(); }
}
```

**应用**：医院接待员、封装复杂子系统

---

## 桥接模式

**定义**：抽象化和实现化解耦，各自独立变化。

```java
interface Color { String getColor(); }
class Red implements Color { public String getColor() { return "red"; } }

abstract class Shape { Color color; void setColor(Color c) { this.color = c; } }
class Circle extends Shape { void draw() { color.getColor(); } }
```

**应用**：不同形状+不同颜色组合

---

## 组合模式

**定义**：将对象组合成树形结构，部分-整体层级。

```java
abstract class Component { abstract void operation(); }
class Leaf extends Component { void operation() { /* 叶子 */ } }
class Composite extends Component {
    List<Component> children = new ArrayList<>();
    void operation() { children.forEach(Component::operation); }
    void add(Component c) { children.add(c); }
}
```

**应用**：文件系统、菜单树

---

## 享元模式

**定义**：共享大量细粒度对象，减少内存。

```java
class TreeFactory {
    static Map<String, TreeType> cache = new HashMap<>();
    static TreeType getTreeType(String name, Color c) {
        String key = name + c;
        if (!cache.containsKey(key)) cache.put(key, new TreeType(name, c));
        return cache.get(key);
    }
}
```

**应用**：大量相似对象（森林中的树）
