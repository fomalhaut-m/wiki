# PMP 项目管理认证

> 来源：PMP备考指南 | 标签：项目管理 / PMP认证

---

## 考试简介

- **时间**：每年3、6、9、12月四次
- **题量**：200道选择题
- **条件**：学士学位36个月经验 / 无学位60个月经验

---

## 考试内容

### 5大过程组

| 过程组 | 说明 |
|--------|------|
| **启动** | 确定项目章程、识别相关方 |
| **规划** | 制定项目管理计划 |
| **实施** | 执行计划、完成工作 |
| **监控** | 跟踪、审查、调整 |
| **收尾** | 正式验收、总结经验 |

### 10大知识领域

| 领域 | 核心内容 |
|------|----------|
| **整合管理** | 项目章程、变更控制、结束项目 |
| **范围管理** | WBS、确认范围、控制范围 |
| **进度管理** | 关键路径、里程碑 |
| **成本管理** | 挣值管理、预算 |
| **质量管理** | 质量成本、控制质量 |
| **资源管理** | 建设团队、处理冲突 |
| **沟通管理** | 规划沟通、管理沟通 |
| **风险管理** | 识别风险、风险应对 |
| **采购管理** | 合同类型、采购流程 |
| **相关方管理** | 识别相关方、管理相关方 |

## 答题思维（美式）

1. **诚实内部公开** — 不隐瞒问题
2. **发现问题 → 分析问题 → 解决问题**
3. **流程制度大于天**
4. **预防为主** — 流程/制度/机制
5. **积极主动** — 不等不靠不要不逃
6. **阳光法则** — 勇于面对问题，对事不对人
7. **契约精神，实事求是**

---

## 万能公式

**不等不靠不要不逃** — 积极主动面对和解决问题

**救命锦囊：**
1. 查找依据（找相关管理计划）
2. 分析原因
3. 分析影响（对变更）
4. 分析偏差（对进度、成本）
5. 制定并实施方案
